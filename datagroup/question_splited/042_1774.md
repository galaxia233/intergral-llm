【1774】 $\int \frac{\ln x\mathrm{d}x}{x\sqrt{1 + \ln x}}.$

解设 $1 + \ln x = t$ ，则 $\frac{\ln x\mathrm{d}x}{x} = (1 + \ln x - 1)\mathrm{d}(1 + \ln x) = (t - 1)\mathrm{d}t,$

故得 $\int_{x\sqrt{1 + \ln x}}^{\ln x\mathrm{d}x} = \int t^{-\frac{1}{2}}(t - 1)\mathrm{d}t = \int (t^{\frac{1}{2}} - t^{-\frac{1}{2}})\mathrm{d}t = \frac{2}{3} t^{\frac{3}{2}} - 2t^{\frac{1}{2}} + C = \frac{2}{3} (\ln x - 2)\sqrt{1 + \ln x} +C.$
