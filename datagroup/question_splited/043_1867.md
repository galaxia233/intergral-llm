гҖҗ1867гҖ‘ $\int \frac{x\mathrm{d}x}{(x + 1)(x + 2)(x + 3)}.$

и§Ји®ҫ $\frac{x}{(x + 1)(x + 2)(x + 3)} = \frac{A}{x + 1} +\frac{B}{x + 2} +\frac{C}{x + 3}$ пјҢйҖҡеҲҶеҗҺеә”жңү $x\equiv A(x + 2)(x + 3) + B(x + 1)(x + 3) + C(x + 1)(x + 2)$ .еңЁиҝҷжҒ’зӯүејҸдёӯпјҢ

д»Ө $x = -1$ пјҢеҫ— $-1 = 2A$ пјҢ $A = -\frac{1}{2}$ пјӣд»Ө $x = -2$ пјҢеҫ— $-2 = -B$ пјҢ $B = 2$

д»Ө $x = -3$ пјҢеҫ— $-3 = 2C$ пјҢ $C = -\frac{3}{2}$

дәҺжҳҜпјҢ $\int \frac{x\mathrm{d}x}{(x + 1)(x + 2)(x + 3)} = \int \left(\frac{-\frac{1}{2}}{x + 1} +\frac{2}{x + 2} +\frac{-\frac{3}{2}}{x + 3}\right)\mathrm{d}x$

$$
= - \frac {1}{2} \ln | x + 1 | + 2 \ln | x + 2 | - \frac {3}{2} \ln | x + 3 | + C = \frac {1}{2} \ln \left| \frac {(x + 2) ^ {4}}{(x + 1) (x + 3) ^ {3}} \right| + C.
$$
