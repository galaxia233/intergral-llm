【2025】 $\int \frac{\mathrm{d}x}{2\sin x - \cos x + 5}$

解设 $t = \tan \frac{x}{2}$ 则 $\sin x = \frac{2t}{1 + t^2},\cos x = \frac{1 - t^2}{1 + t^2},\mathrm{d}x = \frac{2\mathrm{d}t}{1 + t^2}.$ 代人得

$$
\int \frac {d x}{2 \sin x - \cos x + 5} = \int \frac {d t}{3 t ^ {2} + 2 t + 2} = \frac {1}{\sqrt {5}} \arctan \left(\frac {3 t + 1}{\sqrt {5}}\right) + C = \frac {1}{\sqrt {5}} \arctan \left(\frac {3 \tan \frac {x}{2} + 1}{\sqrt {5}}\right) + C.
$$
