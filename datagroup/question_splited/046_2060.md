【2060】 $\int \frac{\sin x\mathrm{d}x}{\cos x\sqrt{1 + \sin^2x}}.$

解 $\int \frac{\sin x \mathrm{d}x}{\cos x \sqrt{1 + \sin^2 x}} = \int \frac{-\mathrm{d}(\cos x)}{\cos x \sqrt{2 - \cos^2 x}} = -\int \frac{\mathrm{d}(\cos x)}{\cos^2 x \sqrt{2 \sec^2 x - 1}} = \int \frac{\mathrm{d}(\sec x)}{\sqrt{2 \sec^2 x - 1}}$

$$
= \frac {1}{\sqrt {2}} \ln \left| \sqrt {2} \sec x + \sqrt {2 \sec^ {2} x - 1} \right| + C = \frac {1}{\sqrt {2}} \ln \frac {\sqrt {2} + \sqrt {1 + \sin^ {2} x}}{| \cos x |} + C.
$$
