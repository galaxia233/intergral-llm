【4272】 $\mathrm{dz} = \frac{y\mathrm{d}x - x\mathrm{d}y}{3x^2 - 2xy + 3y^2}$

解 $z = \int_{0}^{x}\frac{y\mathrm{d}x}{3x^2 - 2xy + 3y^2} +\int_{1}^{y}0\mathrm{d}y + C = \frac{y}{3}\int_{0}^{x}\frac{\mathrm{d}x}{\left(x - \frac{1}{3}y\right)^2 + \frac{8y^2}{9}} +C$

$$
= \frac {y}{3} \cdot \frac {3}{2 \sqrt {2} y} \arctan \frac {3 \left(x - \frac {y}{3}\right)}{2 \sqrt {2} y} \Bigg | _ {0} ^ {x} + C = \frac {1}{2 \sqrt {2}} \arctan \frac {3 x - y}{2 \sqrt {2} y} + C _ {1}.
$$
