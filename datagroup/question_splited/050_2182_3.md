【2182】把所给区间分为 $n$ 个相等的子区间，求下列函数 $f(x)$ 在相应区间上的下积分和 $\underline{S}_n$ 及上积分和 $\overline{S}_n$
(3） $f(x) = 2^{x}$ $(0\leqslant x\leqslant 10)$

解（3） $h = \frac{10}{n}, \quad m_i = 2^h, \quad M_i = 2^{(i + 1)h} \quad (i = 0, 1, 2, \dots, n - 1).$

于是， $\underline{S_n} = \sum_{i = 0}^{n - 1}h2^{\prime \prime} = \frac{h(2^{n\prime} - 1)}{2^n - 1} = \frac{10230}{n(2^{\frac{10}{n}} - 1)};$ $\overline{S_n} = \sum_{i = 0}^{n - 1}h2^{(i + 1)h} = \frac{h2^h(2^{n\prime} - 1)}{2^n - 1} = \frac{10230\cdot 2^{\frac{10}{n}}}{n(2^{\frac{10}{n}} - 1)}.$
