41.63 Is it possible to extend $f(x, y) = \frac{x + y}{x^2 + y^2}$ to the origin so that the resulting function is continuous?

Let $y = mx$ . Then $\left|\frac{x + y}{x^2 + y^2}\right| = \left|\frac{x + mx}{x^2 + m^2x^2}\right| = \frac{1 + m}{|x|(1 + m^2)} \to +\infty$ as $(x, y) \to (0, 0)$ . Hence, continuous extension is impossible.
