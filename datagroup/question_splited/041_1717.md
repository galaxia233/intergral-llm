【1717】 $\int \frac{\cos x\mathrm{d}x}{\sqrt{2 + \cos 2x}}.$

提示 注意 $\frac{\cos x \mathrm{d}x}{\sqrt{2 + \cos 2x}} = \frac{\mathrm{d}(\sin x)}{\sqrt{3 - 2 \sin^2 x}}.$

解 $\int \frac{\cos x \mathrm{d}x}{\sqrt{2 + \cos 2x}} = \int \frac{\mathrm{d}(\sin x)}{\sqrt{3 - 2\sin^2 x}} = \frac{1}{\sqrt{2}} \arcsin \left( \sqrt{\frac{2}{3}} \sin x \right) + C.$
