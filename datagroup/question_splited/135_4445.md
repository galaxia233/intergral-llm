【4445】求向量 $a = y_i + z_j + x_k$ 通过以平面
$$
x = 0, y = 0, z = 0, x + y + z = a \quad (a > 0)
$$
为界的角锥的全表面的通量.利用奥斯特罗格拉茨基公式验证结果
解 解法1：
由于 $\mathrm{div}a = 0$ ，故所求的通量为 $\iint_{S} a_{n} \mathrm{d}S = \iiint_{V} \mathrm{div}a \mathrm{d}V = 0.$
