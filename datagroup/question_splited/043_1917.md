【1917】 $\int \frac{x^2 + 1}{x^4 + x^2 + 1}\mathrm{d}x.$

解 由于 $\frac{x^2 + 1}{x^4 + x^2 + 1} = \frac{x^2 + 1}{(x^2 + 1)^2 - x^2} = \frac{x^2 + 1}{(x^2 - x + 1)(x^2 + x + 1)} = \frac{1}{2}\left(\frac{1}{x^2 - x + 1} +\frac{1}{x^2 + x + 1}\right),$

所以， $\int \frac{x^2 + 1}{x^4 + x^2 + 1}\mathrm{d}x = \frac{1}{2}\int \frac{\mathrm{d}x}{x^2 - x + 1} +\frac{1}{2}\int \frac{\mathrm{d}x}{x^2 + x + 1}$

$$
\begin{array}{l} = \frac {1}{2} \int \frac {\mathrm {d} \left(x - \frac {1}{2}\right)}{\left(x - \frac {1}{2}\right) ^ {2} + \frac {3}{4}} + \frac {1}{2} \int \frac {\mathrm {d} \left(x + \frac {1}{2}\right)}{\left(x + \frac {1}{2}\right) ^ {2} + \frac {3}{4}} \\ = \frac {1}{\sqrt {3}} \arctan \frac {2 x - 1}{\sqrt {3}} + \frac {1}{\sqrt {3}} \arctan \frac {2 x + 1}{\sqrt {3}} + C _ {1} = \frac {1}{\sqrt {3}} \arctan \frac {x ^ {2} - 1}{x \sqrt {3}} + C. \\ \end{array}
