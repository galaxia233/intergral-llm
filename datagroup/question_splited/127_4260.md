【4260】 $\int_{(0,1)}^{(2,3)} (x + y) \mathrm{d}x + (x - y) \mathrm{d}y.$

解 显然，我们有

$$
(x + y) \mathrm {d} x + (x - y) \mathrm {d} y = (y \mathrm {d} x + x \mathrm {d} y) + (x \mathrm {d} x - y \mathrm {d} y) = \mathrm {d} (x y) + \mathrm {d} \left(\frac {x ^ {2} - y ^ {2}}{2}\right) = \mathrm {d} \left(x y + \frac {x ^ {2} - y ^ {2}}{2}\right),
$$

即是全微分.于是， $\int_{(0,1)}^{(2,3)}(x + y)\mathrm{d}x + (x - y)\mathrm{d}y = \int_{(0,1)}^{(2,3)}\mathrm{d}\left(xy + \frac{x^2 - y^2}{2}\right) = \left(xy + \frac{x^2 - y^2}{2}\right)\Bigg|_{(0,1)}^{(2,3)} = 4.$
