41.89 Find a set of spherical coordinates for the point whose rectangular coordinates are $(0, -1, \sqrt{3})$ .

$\rho^2 = 0^2 + (-1)^2 + (\sqrt{3})^2 = 4.$ Hence, $\rho = 2.$ $\tan \phi = \sqrt{0^2 + (-1)^2} / \sqrt{3} = 1 / \sqrt{3}.$ Therefore, $\phi = \pi / 6.$ $\tan \theta = -1 / 0 = -\infty.$ Hence, $\theta = 3\pi / 2.$ So, a set of spherical coordinates is $(2, \pi / 6, 3\pi / 2)$ .
