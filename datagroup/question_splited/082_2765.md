【2765】设函数 $f(x)$ 在区间 $(a,b)$ 内有连续的导数 $f^{\prime}(x)$ ，且

$$
f _ {n} (x) = n \left[ f \left(x + \frac {1}{n}\right) - f (x) \right].
$$

证明：在闭区间 $\alpha \leqslant x \leqslant \beta$ 上（其中 $a < \alpha < \beta < b$ ）， $f_n(x) \rightarrow f'(x)$ 。

证考虑 $[\alpha', \beta']$ ，其中 $a < \alpha' < \alpha < \beta < \beta' < b.$ 由于 $f_n(x)$ （ $n$ 充分大）在 $[\alpha', \beta']$ 上有连续导数，由微分学中值公式，得

$$
f _ {n} (x) = n \left[ f \left(x + \frac {1}{n}\right) - f (x) \right] = n f ^ {\prime} \left(x + \frac {\theta}{n}\right) \cdot \frac {1}{n} = f ^ {\prime} \left(x + \frac {\theta}{n}\right) \quad (0 <   \theta <   1).
$$

又因 $f^{\prime}(x)$ 在 $[\alpha^{\prime},\beta^{\prime}]$ 上连续，所以 $f^{\prime}(x)$ 在 $[\alpha^{\prime},\beta^{\prime}]$ 上一致连续，即对任给的 $\epsilon >0$ ，存在 $\delta = \delta (\varepsilon) > 0$ ，使对于 $[\alpha^{\prime},\beta^{\prime}]$ 上的任意点 $x^{\prime}$ 及 $x''$ ，只要当 $|x^{\prime} - x^{\prime \prime}| <   \delta$ 时，就有 $|f^{\prime}(x^{\prime}) - f^{\prime}(x^{\prime \prime})| <   \varepsilon .$ 今取 $N = [\frac{1}{\delta}] + 1 = N(\varepsilon),$ 则当 $n > N$ 时，有 $\frac{1}{n} <  \frac{1}{N} <  \delta .$ 于是，对 $[\alpha ,\beta ]$ 上的一切 $x$ 值，只要 $N$ 足够大，就可保证 $x$ 与 $x + \frac{\theta}{n}$ 均属于 $[\alpha^{\prime},\beta^{\prime}]$ .于是，对于 $[\alpha ,\beta ]$ 上的一切 $x$ 值，均有

$$
\left| f _ {n} (x) - f ^ {\prime} (x) \right| = \left| f ^ { \prime } \left(x + \frac {\theta}{n}\right) - f ^ { \prime } (x) \right| <   \varepsilon .
$$

因此， $f_{*}(x)$ 在 $[\alpha ,\beta ]$ 上一致收敛于 $f^{\prime}(x)$
