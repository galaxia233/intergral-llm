【1901】 计算积分 $\int \frac{\mathrm{d}x}{x^4 + 2x^3 + 3x^2 + 2x + 1}$

解 $Q = x^{4} + 2x^{3} + 3x^{2} + 2x + 1 = (x^{2} + x + 1)^{2},\quad Q_{1} = Q_{2} = x^{2} + x + 1.$

设 $\frac{1}{x^4 + 2x^3 + 3x^2 + 2x + 1} = \left(\frac{Ax + B}{x^2 + x + 1}\right)' + \frac{Cx + D}{x^2 + x + 1}$ ，从而，

$$
1 \equiv A (x ^ {2} + x + 1) - (2 x + 1) (A x + B) + (C x + D) (x ^ {2} + x + 1).
$$

比较等式两端 $x$ 的同次幂系数，解出 $A = \frac{2}{3}$ ， $B = \frac{1}{3}$ ， $C = 0$ ， $D = \frac{2}{3}$ .于是，

$$
\begin{array}{l} \int \frac {\mathrm {d} x}{x ^ {4} + 2 x ^ {3} + 3 x ^ {2} + 2 x + 1} = \frac {2 x + 1}{3 \left(x ^ {2} + x + 1\right)} + \frac {2}{3} \int \frac {\mathrm {d} x}{x ^ {2} + x + 1} = \frac {2 x + 1}{3 \left(x ^ {2} + x + 1\right)} + \frac {2}{3} \int \frac {\mathrm {d} \left(x + \frac {1}{2}\right)}{\left(x + \frac {1}{2}\right) ^ {2} + \frac {3}{4}} \\ = \frac {2 x + 1}{3 (x ^ {2} + x + 1)} + \frac {4}{3 \sqrt {3}} \arctan \left(\frac {2 x + 1}{\sqrt {3}}\right) + C. \\ \end{array}
$$
