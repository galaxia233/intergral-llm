20.24 $\int_0^{\pi /2}\sqrt{\sin x + 1}\cos xdx.$

$\int \sqrt{\sin x + 1}\cos x dx = \int (\sin x + 1)^{1 / 2}\cos x dx = \frac{2}{3} (\sin x + 1)^{3 / 2}$ [by Problem 19.1]. Hence, $\int_0^{\pi /2}\sqrt{\sin x + 1}\cos x dx = \frac{2}{3} (\sin x + 1)^{3 / 2}\bigg]_0^{\pi /2} = \frac{2}{3}\left\{[\sin (\pi /2) + 1]^{3 / 2} - (\sin 0 + 1)^{3 / 2}\right\} = \frac{2}{3} (2\sqrt{2} -1).$
