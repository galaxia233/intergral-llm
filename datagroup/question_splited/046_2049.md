【2049】 $\int \frac{2\sin x + \cos x}{3\sin x + 4\cos x - 2}\mathrm{d}x.$

解 本题也是2046题的特例，这里

$$
a _ {1} = 2, b _ {1} = 1, c _ {1} = 0, a = 3, b = 4, c = - 2; A = \frac {2}{5}, B = - \frac {1}{5}, C = \frac {4}{5}.
$$

代人得 $\int \frac{2\sin x + \cos x}{3\sin x + 4\cos x - 2}\mathrm{d}x = \frac{2}{5} x - \frac{1}{5}\ln |3\sin x + 4\cos x - 2| + \frac{4}{5}\int \frac{\mathrm{d}x}{3\sin x + 4\cos x - 2}$

$$
= \frac {2}{5} x - \frac {1}{5} \ln | 3 \sin x + 4 \cos x - 2 | + \frac {4}{5 \sqrt {2 1}} \ln \left| \frac {\sqrt {7} + \sqrt {3} (2 \tan \frac {x}{2} - 1)}{\sqrt {7} - \sqrt {3} (2 \tan \frac {x}{2} - 1)} \right| ^ {*}) + C.
$$

\*) 设 $t = \tan \frac{x}{2}$ , 积分即得所求式子.
