34.30 Where is the curve $x = \ln t, y = e^{t}$ concave upward?

$\begin{array}{l} dx / dt = 1 / t, \quad dy / dt = e^t. \\ \text{So, } dy / dx = te^t, \quad \frac{d^2y}{dx^2} = \frac{\frac{d}{dt}(te^t)}{1 / t} = e^t t(t + 1). \end{array}$ Thus, $d^2 y / dx^2 > 0$ if and only if $t(t + 1) > 0$ . Since $t > 0$ (in order for $x = \ln t$ to be defined), the curve is concave upward everywhere.
