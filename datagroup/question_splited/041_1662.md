【1662】 $\int \frac{\mathrm{d}x}{2 - 3x^2}$

解 $\int \frac{\mathrm{d}x}{2 - 3x^2} = \frac{1}{2}\int \frac{\mathrm{d}x}{1 - \left(\sqrt{\frac{3}{2}}x\right)^2} = \frac{1}{2}\cdot \sqrt{\frac{2}{3}}\cdot \frac{1}{2}\ln \left|\frac{1 + \sqrt{\frac{3}{2}}x}{1 - \sqrt{\frac{3}{2}}x}\right| + C = \frac{1}{2\sqrt{6}}\ln \left|\frac{\sqrt{2} + x\sqrt{3}}{\sqrt{2} - x\sqrt{3}}\right| + C.$
