【2047】 $\int \frac{\sin x + 2\cos x - 3}{\sin x - 2\cos x + 3}\mathrm{d}x.$

解此为2046题的特例，这里

$$
a _ {1} = 1, b _ {1} = 2, c _ {1} = - 3, a = 1, b = - 2, c = 3;
$$

$$
A = \frac {a a _ {1} + b b _ {1}}{a ^ {2} + b ^ {2}} = \frac {1 - 4}{1 + 4} = - \frac {3}{5}, \quad B = \frac {a b _ {1} - a _ {1} b}{a ^ {2} + b ^ {2}} = \frac {2 + 2}{1 + 4} = \frac {4}{5},
$$

$$
C = \frac {a \left(a c _ {1} - a _ {1} c\right) + b \left(b c _ {1} - b _ {1} c\right)}{a ^ {2} + b ^ {2}} = \frac {(- 3 - 3) + (- 2) (6 - 6)}{1 + 4} = - \frac {6}{5}.
$$

代入得 $\int \frac{\sin x + 2\cos x - 3}{\sin x - 2\cos x + 3}\mathrm{d}x = -\frac{3}{5} x + \frac{4}{5}\ln |\sin x - 2\cos x + 3| - \frac{6}{5}\int \frac{\mathrm{d}x}{\sin x - 2\cos x + 3}$

$$
= - \frac {3}{5} x + \frac {4}{5} \ln | \sin x - 2 \cos x + 3 | - \frac {6}{5} \arctan \frac {1 + 5 \tan \frac {x}{2} \cdot)}{2} + C.
$$

\*) 设 $t = \tan \frac{x}{2}$ , 积分即得所求式子.
