【2058】 求 $\int \frac{\mathrm{d}x}{(\sin x + 2\cos x)^3}$

解 此为2057题的特例，这里 $a = 1$ ， $b = 2$ ， $n = 3$ ； $A = \frac{2}{10}$ ， $B = -\frac{1}{10}$ ， $C = \frac{1}{10}$

代入得 $\int \frac{\mathrm{d}x}{(\sin x + 2\cos x)^3} = \frac{2\sin x - \cos x}{10(\sin x + 2\cos x)^2} +\frac{1}{10}\int \frac{\mathrm{d}x}{\sin x + 2\cos x}$

$$
\begin{array}{l} = \frac {2 \sin x - \cos x}{1 0 (\sin x + 2 \cos x) ^ {2}} + \frac {1}{1 0 \sqrt {5}} \int \frac {\mathrm {d} x}{\sin (x + \alpha)} \\ = \frac {2 \sin x - \cos x}{1 0 (\sin x + 2 \cos x) ^ {2}} + \frac {1}{1 0 \sqrt {5}} \ln \left| \tan \left(\frac {x}{2} + \frac {\alpha}{2}\right) \right| + C. \\ \end{array}
$$

其中 $\cos \alpha = \frac{1}{\sqrt{5}}, \sin \alpha = \frac{2}{\sqrt{5}}, \alpha = \arctan 2.$
