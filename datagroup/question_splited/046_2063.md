【2063】 $\int \frac{\mathrm{d}x}{(1 + \varepsilon\cos x)^2}$ （ $0 <   \varepsilon <  1)$

解 此为2059题之特例，这里

$$
a = 1, \quad b = \varepsilon , \quad n = 2, \quad A = - \frac {\varepsilon}{1 - \varepsilon^ {2}}, \quad B = \frac {1}{1 - \varepsilon^ {2}}, \quad C = 0.
$$

代入得 $\int \frac{\mathrm{d}x}{(1 + \varepsilon\cos x)^2} = -\frac{\varepsilon\sin x}{(1 - \varepsilon^2)(1 + \varepsilon\cos x)} +\frac{1}{1 - \varepsilon^2}\int \frac{1}{1 + \varepsilon\cos x}\mathrm{d}x$

$$
= - \frac {\varepsilon \sin x}{\left(1 - \varepsilon^ {2}\right) \left(1 + \varepsilon \cos x\right)} + \frac {2}{\left(1 - \varepsilon^ {2}\right) ^ {\frac {3}{2}}} \arctan \left(\sqrt {\frac {1 - \varepsilon}{1 + \varepsilon}} \tan \frac {x}{2}\right) ^ {*}) + C.
$$

\*) 利用2028题(1)的结果.
