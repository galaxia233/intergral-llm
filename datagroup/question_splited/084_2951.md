【2951】在区间 $(-\frac{\pi}{2},\frac{\pi}{2})$ 内展开 $f(x) = x\cos x.$

解 因为 $f(x)$ 为奇函数，从而， $a_0 = a_n = 0$ ，且

$$
\begin{array}{l} b _ {n} = \frac {4}{\pi} \int_ {0} ^ {\frac {\pi}{2}} x \cos x \sin 2 n x d x = \frac {2}{\pi} \int_ {0} ^ {\frac {\pi}{2}} x [ \sin (2 n + 1) x + \sin (2 n - 1) x ] d x \\ = \frac {2}{\pi} \left[ - \frac {x \cos (2 n + 1) x}{2 n + 1} + \frac {\sin (2 n + 1) x}{(2 n + 1) ^ {2}} - \frac {x \cos (2 n - 1) x}{2 n - 1} + \frac {\sin (2 n - 1) x}{(2 n - 1) ^ {2}} \right] \Bigg | _ {0} ^ {\frac {\pi}{2}} \\ = \frac {(- 1) ^ {n + 1} 1 6}{\pi} \frac {n}{(4 n ^ {2} - 1) ^ {2}}, \\ \end{array}
$$

故按展开定理， $f(x)$ 可展开为 $\frac{16}{\pi}\sum_{n = 1}^{\infty}\frac{(-1)^{n + 1}n}{(4n^2 - 1)^2}\sin 2nx = x\cos x$ $(-\frac{\pi}{2} < x < \frac{\pi}{2})$
