【2718】 $\sum_{n = 1}^{\infty}\frac{n}{n + 1}\left(\frac{x}{2x + 1}\right)^{n}.$

提示 仿 2717 题的解法.

解 由于 $\lim_{n\to \infty}\frac{\frac{n}{n + 1}}{\frac{n + 1}{n + 2}} = 1$ ，故仅当 $\left|\frac{x}{2x + 1}\right| < 1$ 即 $x^{2} < 4x^{2} + 4x + 1$ 或

$$
(3 x + 1) (x + 1) > 0 \tag {1}
$$

时，级数绝对收敛。解不等式（1），得

$$
x > - \frac {1}{3} \quad \text {或} \quad x <   - 1,
$$

即为所求的绝对收敛域。当 $x = -\frac{1}{3}$ 或 $x = -1$ 时，原级数通项不趋于零，故发散。
