【2044】 $\int \frac{\mathrm{d}x}{3 + 5\tan x}$

解 $\int \frac{\mathrm{d}x}{3 + 5\tan x} = \int \frac{\cos x}{5\sin x + 3\cos x}\mathrm{d}x.$ 此为2042题的特例，这里

$$
a _ {1} = 0, \quad b _ {1} = 1, \quad a = 5, \quad b = 3; A = \frac {3}{3 4}, \quad B = \frac {5}{3 4}.
$$

代入得 $\int \frac{\mathrm{d}x}{3 + 5\tan x} = \frac{3}{34} x + \frac{5}{34}\ln |5\sin x + 3\cos x| + C.$
