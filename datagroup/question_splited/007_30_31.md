30.31 $\int \frac{dx}{\sqrt{\sqrt{x} + 1}}.$   
Let $\sqrt{x} + 1 = z^2, -\frac{1}{2\sqrt{x}} dx = 2z dz, -\frac{1}{2(z^2 - 1)} dx = 2z dz, dx = -4z(z^2 - 1)dz.$ Then $\int \frac{dx}{\sqrt{\sqrt{x} + 1}} = -4\int \frac{z(z^2 - 1)dz}{z} = -4\int (z^2 - 1)dz = -4(\frac{1}{3}z^3 - z) + C = -\frac{4}{3}z(z^2 - 3) + C = -\frac{4}{3}\sqrt{\sqrt{x} + 1} (\sqrt{x} - 2) + C.$
