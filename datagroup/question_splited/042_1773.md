【1773】 $\int \frac{\sin^2x}{\cos^6x}\mathrm{d}x.$

提示 令 $\tan x = t$

解设 $\tan x = t$ ，则 $\frac{1}{\cos^4x}\mathrm{d}x = (1 + t^2)\mathrm{d}t,$

故得 $\int \frac{\sin^2 x}{\cos^6 x} \mathrm{d}x = \int (t^4 + t^2) \mathrm{d}t = \frac{1}{5} t^5 + \frac{1}{3} t^3 + C = \frac{1}{5} \tan^5 x + \frac{1}{3} \tan^3 x + C.$
