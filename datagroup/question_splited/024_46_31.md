46.31 Solve $\frac{dy}{dx} - y = \sin x$ .

Use an integrating factor. $\rho = \exp \left[\int (-1) dx\right] = e^{-x}, \int \rho q(x) dx = \int e^{-x} \sin x dx = -\frac{1}{2} e^{-x} (\sin x + \cos x).$ (The last result is found by integration by parts.) Hence, $y = e^{x}\left[-\frac{1}{2} e^{-x} (\sin x + \cos x) + C\right] = -\frac{1}{2} (\sin x + \cos x) + Ce^{x}.$
