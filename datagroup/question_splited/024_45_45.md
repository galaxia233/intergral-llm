45.45 Prove that any gradient field $\nabla f$ in a region is conservative. More precisely, if $\mathcal{C}$ is any curve in the region from a point $P$ to a point $Q$ , then $\int_{\mathcal{C}} \nabla f \cdot \mathbf{T} ds = f(Q) - f(P)$ .

If $\mathcal{C}$ is given by $(x(t),y(t),z(t))$ for $a\leq t\leq b,\int_{\mathbb{C}}\nabla f\cdot \mathbf{T}ds = \int_{\mathbb{C}}f_xdx + f_ydy + f_zdz = \int_a^b\left(f_x\frac{dx}{dt} +f_y\frac{dy}{dt} +f_z\frac{dz}{dt}\right)\mathrm{d}t = \int_a^b\frac{d}{dt}\left[f(x(t),y(t),z(t))\right]dt = f(x(t),y(t),z(t))\bigg]_a^b = f(Q) - f(P).$ (Note: The converse also holds if any two points in the region can be connected by a continuous curve lying in the region.)
