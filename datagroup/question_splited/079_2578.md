【2578】 $\frac{1000}{1!} + \frac{1000^2}{2!} + \frac{1000^3}{3!} + \dots + \frac{1000^n}{n!} + \dots.$

解 由于

$$
\lim  _ {n \rightarrow \infty} \frac {a _ {n + 1}}{a _ {n}} = \lim  _ {n \rightarrow \infty} \frac {\frac {1 0 0 0 ^ {n + 1}}{(n + 1) !}}{\frac {1 0 0 0 ^ {n}}{n !}} = \lim  _ {n \rightarrow \infty} \frac {1 0 0 0}{n + 1} = 0 <   1,
$$

故级数 $\sum_{n=1}^{\infty} \frac{1000^n}{n!}$ 收敛.
