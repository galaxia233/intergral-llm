【1973】 $\int \frac{\mathrm{d}x}{\sqrt{2} + \sqrt{1 - x} + \sqrt{1 + x}}.$

解 $\int \frac{\mathrm{d}x}{\sqrt{2} + \sqrt{1 - x} + \sqrt{1 + x}} = \int \frac{-\sqrt{2} + \sqrt{1 - x} + \sqrt{1 + x}}{(\sqrt{2} + \sqrt{1 - x} + \sqrt{1 + x})(-\sqrt{2} + \sqrt{1 - x} + \sqrt{1 + x})}\mathrm{d}x$

$$
\begin{array}{l} = \int \frac {- \sqrt {2} + \sqrt {1 - x} + \sqrt {1 + x}}{2 \sqrt {1 - x ^ {2}}} d x = - \frac {1}{\sqrt {2}} \int \frac {d x}{\sqrt {1 - x ^ {2}}} + \frac {1}{2} \int \frac {d x}{\sqrt {1 + x}} + \frac {1}{2} \int \frac {d x}{\sqrt {1 - x}} \\ = - \frac {1}{\sqrt {2}} \arcsin x + \sqrt {1 + x} - \sqrt {1 - x} + C. \\ \end{array}
