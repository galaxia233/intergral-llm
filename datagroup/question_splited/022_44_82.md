44.81 Find the center of mass of a plate in the form of the upper half of the cardioid $r = 2(1 + \cos \theta)$ if the density is numerically equal to the distance from the pole.

See Fig. 44-42. The mass \( M = \iint_{\mathcal{R}} r \, dA = \int_0^\pi \int_0^{2(1 + \cos \theta)} r \cdot r \, dr \, d\theta = \int_0^\pi \frac{1}{3} r^3 \Bigg]_0^{2(1 + \cos \theta)} d\theta = \frac{8}{3} \int_0^\pi (1 + \cos \theta)^3 d\theta = \frac{8}{3} \int_0^\pi \left[1 + 3\cos \theta + 3\left(\frac{1 + \cos 2\theta}{2}\right) + (\cos \theta - \sin^2 \theta \cos \theta)\right] d\theta = \frac{8}{3} \left(\frac{5}{2}\theta + 4\sin \theta + \frac{3}{4}\sin 2\theta - \frac{1}{3}\sin^3\theta\right) \Bigg]_0^\pi = \frac{8}{3} \left(\frac{5}{2}\theta + 4\sin \theta + \frac{3}{4}\sin 2\theta - \frac{1}{3}\sin^3\theta\right) = \frac{8}{3} \left(\frac{5}{2}\theta + 4\sin \theta + \frac{3}{4}\sin 2\theta - \frac{1}{3}\sin^3\theta\right) = \frac{8}{3} \left(\frac{5}{2}\theta + 4\sin \theta + \frac{3}{4}\sin 2\theta - \frac{1}{3}r^2 dr\right) = \frac{8}{3} \left(\frac{5}{2}\theta + 4\sin \theta + \frac{3}{4}\sin 2\theta - \frac{1}{3}r^2 dr\right) = \frac{8}{3} \left(\frac{5}{2}\theta + 4\sin \theta + \frac{3}{4}\sin 2\Theta - \frac{17}{5}r^2 dr\right) = \frac{8}{3} \left(\frac{5}{2}\theta + 4\sin \theta + \frac{3}{4}\sin 2\Theta - \frac{17}{5}r^2 dr\right) = \frac{8}{3} \left(\frac{5}{2}\theta + 4\sin \theta + \frac{3}{4}\sin 2\Theta - \frac{17}{5}r^2 dr\right) = \frac{8}{3} \left(\frac{5}{2}\theta + 4\sin \theta + \frac{3}{4}\sin 2\Theta - \frac{17}{5}r^2 dr\right) = \frac{8}{3} \left(\frac{5}{2}\theta + 4\sin \theta + \frac{3}{4}\sin 2\Theta - \frac{17}{5}r^2 dr\right) = \frac{8}{3} \left(\frac{5}{2}\theta + 4\sin \theta + \frac{17}{5}r^2 dr\right) = \frac{8}{3} \left(\frac{5}{2}\theta + 4\sin \theta + \frac{17}{5}r^2 dr\right) = \frac{8}{3} \left(\frac{5}{2}\theta + 4\sin \theta + \frac{17}{5}r^2 dr\right) = \left(\frac{7}{2}\theta + 4\sin 2\theta - \frac{17}{5}r^2 dr\right) = 4(7/2)\pi = 14\pi.\quad Hence,\quad x = M_y = \frac{14\pi}{20\pi/3} = \frac{21}{10}. So the center of mass is \( \left(\frac{21}{10},\frac{96}{25\pi}\right). \)
