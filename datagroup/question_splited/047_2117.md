【2117】 $\int \mathrm{ch}^4 x\mathrm{d}x.$

解 $\int \mathrm{ch}^4 x\mathrm{d}x = \int \left(\frac{1 + \mathrm{ch}2x}{2}\right)^2\mathrm{d}x = \int \left(\frac{1}{4} +\frac{1}{2}\mathrm{ch}2x + \frac{1}{4}\mathrm{ch}^2 2x\right)\mathrm{d}x$

$$
= \frac {1}{4} x + \frac {1}{4} \mathrm {s h } 2 x + \frac {1}{8} \left(x + \frac {1}{4} \mathrm {s h } 4 x\right) ^ {*}) + C = \frac {3}{8} x + \frac {1}{4} \mathrm {s h } 2 x + \frac {1}{3 2} \mathrm {s h } 4 x + C.
$$

\*) 利用 1762 题的结果.
