【2296】 $\int_0^\pi \cos^{n - 1}x\sin (n + 1)x\mathrm{d}x.$

提示 对积分 $\int_{0}^{\pi} \cos^{n+1} x \sin (n+1) x \mathrm{d}x$ 使用两次分部积分.

解 考虑积分 $I = \int_{0}^{\pi} \cos^{n+1} x \sin (n+1) x \, \mathrm{d}x,$

并对它作两次分部积分，可得 $I = I - \frac{n}{n + 1}\int_{0}^{\pi}\cos^{n - 1}x\sin (n + 1)x\mathrm{d}x.$

于是， $\int_0^\pi \cos^{n - 1}x\sin (n + 1)x\mathrm{d}x = 0.$

本题也可不用分部积分法。事实上，$\cos^{n-1} x \sin (n+1) x$ 是以 $\pi$ 为周期的函数，又是奇函数，于是，

$$
\int_ {0} ^ {\pi} \cos^ {n - 1} x \sin (n + 1) x d x = \int_ {- \frac {\pi}{2}} ^ {\frac {\pi}{2}} \cos^ {n - 1} x \sin (n + 1) x d x = 0.
$$
