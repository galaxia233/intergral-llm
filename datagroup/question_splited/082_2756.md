【2756】(1) $f_{n}(x) = \arctan nx;0 <   x <   + \infty ;$

解（1）当 $0 < x < +\infty$ 时， $\lim_{n\to \infty}\arctan nx = \frac{\pi}{2} = f(x)$ 取 $\epsilon_0$ 使 $0 < \epsilon_0 < \frac{\pi}{4}$ ，不论 $n$ 多么大，只要取 $x = \frac{1}{n}$ ，就有

$$
\left| f _ {n} \left(\frac {1}{n}\right) - f \left(\frac {1}{n}\right) \right| = \left| \arctan 1 - \frac {\pi}{2} \right| = \frac {\pi}{4} > \varepsilon_ {0}.
$$

因此， $f_{n}(x)$ 在 $(0, + \infty)$ 上收敛而不一致收敛。
