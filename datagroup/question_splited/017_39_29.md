39.29 Estimate $\int_0^1\sqrt{x}\sin xdx$ to within two-decimal-place accuracy.

$\sin x = x - x^{3} / 3! + x^{5} / 5! - x^{7} / 7! + \dots$ Hence, $\sqrt{x}\sin x = x^{3 / 2} - x^{7 / 2} / 3! + x^{11 / 2} / 5! - x^{15 / 2} / 7! + \dots,$ and, therefore, $\int_0^1\sqrt{x}\sin xdx = \frac{2}{5} -\frac{1}{3!}\frac{2}{9} +\frac{1}{5!}\frac{2}{13} -\frac{1}{7!}\frac{2}{17} +\dots.$ This is an alternating series, and, since $\frac{1}{5!}\frac{2}{13} < 0.005,$ we may use the first two terms $\frac{2}{5} -\frac{1}{54} = \frac{103}{270}\approx 0.38.$
