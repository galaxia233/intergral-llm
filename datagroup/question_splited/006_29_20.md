29.20 $\int \frac{x^2}{\sqrt{4 - x^2}} dx.$

Since $\sqrt{4 - x^2}$ is present, we let $x = 2\sin \theta$ , $dx = 2\cos \theta d\theta$ . Then (Fig. 29-2), $\sqrt{4 - x^2} = 2\cos \theta$ . So $\int \frac{x^2}{\sqrt{4 - x^2}} dx = \int \frac{4\sin^2\theta}{2\cos\theta} 2\cos \theta d\theta = 4\int \sin^2\theta d\theta = 4\int \frac{1 - \cos 2\theta}{2} d\theta = 2\left(\theta - \frac{\sin 2\theta}{2}\right) + C = 2(\theta - \sin \theta \cos \theta) + C = 2\left(\sin^{-1}\frac{x}{2} - \frac{x}{2} \cdot \frac{\sqrt{4 - x^2}}{2}\right) + C = 2\sin^{-1}\frac{x}{2} - \frac{x\sqrt{4 - x^2}}{2} + C.$
