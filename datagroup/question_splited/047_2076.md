【2076】 $\int x\mathrm{e}^{x}\sin x\mathrm{d}x.$

解 $\int x\mathrm{e}^{x}\sin x\mathrm{d}x = \int x\sin x\mathrm{d}(\mathrm{e}^{x}) = x\mathrm{e}^{x}\sin x - \int \mathrm{e}^{x}(\sin x + x\cos x)\mathrm{d}x$

$$
\begin{array}{l} = x \mathrm {e} ^ {x} \sin x - \int (\sin x + x \cos x) \mathrm {d} \left(\mathrm {e} ^ {x}\right) = \mathrm {e} ^ {x} (x \sin x - \sin x - x \cos x) + \int \mathrm {e} ^ {x} (2 \cos x - x \sin x) \mathrm {d} x \\ = \mathrm {e} ^ {x} (x \sin x - \sin x - x \cos x) + 2 \int \mathrm {e} ^ {x} \cos x \mathrm {d} x - \int x \mathrm {e} ^ {x} \sin x \mathrm {d} x. \\ \end{array}
$$

于是， $\int x\mathrm{e}^{x}\sin x\mathrm{d}x = \frac{\mathrm{e}^{x}}{2} (x\sin x - \sin x - x\cos x) + \int \mathrm{e}^{x}\cos x\mathrm{d}x$

$$
= \frac {e ^ {x}}{2} (x \sin x - \sin x - x \cos x) + \frac {e ^ {x}}{2} (\sin x + \cos x) ^ {\cdot}) + C = \frac {e ^ {x}}{2} [ x (\sin x - \cos x) + \cos x ] + C.
$$

\*）利用 1828 题的结果.
