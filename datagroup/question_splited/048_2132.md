【2132】 $\int \sqrt{\frac{x}{1 - x\sqrt{x}}}\mathrm{d}x.$

提示 令 $\sqrt{1 - x\sqrt{x}} = t$

解设 $\sqrt{1 - x\sqrt{x}} = t$ ，则 $x = (1 - t^2)^{\frac{2}{3}}$ $\mathrm{d}x = -\frac{4}{3} t(1 - t^2)^{-\frac{1}{3}}\mathrm{d}t.$ 代人得

$$
\int \sqrt {\frac {x}{1 - x \sqrt {x}}} d x = - \frac {4}{3} \int d t = - \frac {4}{3} t + C = - \frac {4}{3} \sqrt {1 - x \sqrt {x}} + C (0 <   x <   1).
$$
