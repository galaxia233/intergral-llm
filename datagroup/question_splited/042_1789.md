【1789】 $\int \frac{\mathrm{d}x}{\sqrt{(x + a)(x + b)}}.$

解不妨设 $a < b$ 注意被积函数的存在域为 $x > - a$ 及 $x < - b$

(1) 当 $x > -a$ 时，设 $x + a = (b - a)\mathrm{sh}^2 t$ ，并限制 $t > 0$   
(2) 当 $x < -b$ 时，设 $x + b = (a - b)\mathrm{sh}^2 t$ ，并限制 $t > 0$

解 不妨设 $a < b$ . 被积函数的存在域为 $x > -a$ 及 $x < -b$ .

（1）当 $x > - a$ 时，可设 $x + a = (b - a)\mathrm{sh}^2 t$ ，并限制 $t > 0$

从而， $\sqrt{(x + a)(x + b)} = (b - a)\mathrm{sh}t\mathrm{ch}t,\qquad \mathrm{d}x = 2(b - a)\mathrm{sh}t\mathrm{ch}t\mathrm{d}t.$

代入得 $\int \frac{\mathrm{d}x}{\sqrt{(x + a)(x + b)}} = 2\int \mathrm{d}t = 2t + C_1.$

注意到 $\sqrt{x + a} +\sqrt{x + b} = \sqrt{b - a} (\mathrm{sh}t + \mathrm{ch}t) = \sqrt{b - a}\mathrm{e}^t$ ，就有 $t = \ln \frac{\sqrt{x + a} + \sqrt{x + b}}{\sqrt{b - a}}$ 最后得

$$
\int \frac {\mathrm {d} x}{\sqrt {(x + a) (x + b)}} = 2 \ln (\sqrt {x + a} + \sqrt {x + b}) + C.
$$

(2) 当 $x < -b$ 时，可设 $x + b = (a - b)\mathrm{sh}^2 t$ , 并限制 $t > 0$ .

从而， $\sqrt{(x + a)(x + b)} = (b - a)\mathrm{sh}t\mathrm{ch}t,\qquad \mathrm{d}x = -(b - a)2\mathrm{sh}t\mathrm{ch}t\mathrm{d}t.$

代入得 $\int \frac{\mathrm{d}x}{\sqrt{(x + a)(x + b)}} = -2\int \mathrm{d}t = -2t + C_1 = -2\ln (\sqrt{-(x + a)} +\sqrt{-(x + b)}) + C.$

总之， $\int \frac{\mathrm{d}x}{\sqrt{(x + a)(x + b)}} = \left\{ \begin{array}{ll}2\ln (\sqrt{x + a} +\sqrt{x + b}), & x + a > 0\\ -2\ln (\sqrt{-x - a} +\sqrt{-x - b}), & x + a <   0 \end{array} \right.$ 及 $x + b > 0$
