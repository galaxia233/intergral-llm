【2952】 $f(x) = \operatorname{sgn}(\cos x)$

解由于

$$
f (x + 2 \pi) = \operatorname {s g n} [ \cos (x + 2 \pi) ] = \operatorname {s g n} (\cos x) = f (x),
$$

故 $f(x)$ 是以 $2\pi$ 为周期的周期函数，又 $f(x)$ 为偶函数，从而， $b_{n} = 0$ ，且

$$
\begin{array}{l} a _ {0} = \frac {2}{\pi} \int_ {0} ^ {\pi} \operatorname {s g n} (\cos x) d x = \frac {2}{\pi} \left(\int_ {0} ^ {\frac {\pi}{2}} d x + \int_ {\frac {\pi}{2}} ^ {\pi} (- 1) d x\right) = 0, \\ a _ {n} = \frac {2}{\pi} \int_ {0} ^ {\pi} \operatorname {s g n} (\cos x) \cos n x d x = \frac {2}{\pi} \left(\int_ {0} ^ {\frac {\pi}{2}} \cos n x d x - \int_ {\frac {\pi}{2}} ^ {\pi} \cos n x d x\right) \\ \end{array}
$$

$$
\begin{array}{l} = \frac {2}{\pi} \left(\frac {1}{n} \sin \frac {n \pi}{2} + \frac {1}{n} \sin \frac {n \pi}{2}\right) = \frac {4}{n \pi} \sin \frac {n \pi}{2} \\ = \left\{ \begin{array}{l l} 0, & n = 2 k (k = 1, 2, 3, \dots), \\ (- 1) ^ {k} \frac {4}{(2 k + 1) \pi}, & n = 2 k + 1 (k = 0, 1, 2, \dots), \end{array} \right. \\ \end{array}
$$

故按展开定理， $f(x)$ 可展开为 $\frac{4}{\pi}\sum_{k = 0}^{\infty}\left[-1\right]^k\frac{\cos(2k + 1)x}{2k + 1} = \operatorname {sgn}(\cos x)\quad (-\pi < x < \pi).$

注意 此式在 $f(x)$ 的不连续点 $x = -\frac{\pi}{2}$ 和 $x = \frac{\pi}{2}$ 也成立，这是因为在这些点满足 $f(x) = \frac{1}{2} [f(x - 0) + f(x + 0)]$ 。于是，上述展式对一切 $-\infty < x < +\infty$ 皆成立。
