【1966】 $\int \frac{\mathrm{d}x}{x + \sqrt{x^2 + x + 1}}.$

提示 令 $\sqrt{x^2 + x + 1} = z - x$ .

解设 $\sqrt{x^2 + x + 1} = z - x$ ，则 $x = \frac{z^2 - 1}{1 + 2z},\mathrm{d}x = \frac{2(z^2 + z + 1)}{(1 + 2z)^2}\mathrm{d}z,\sqrt{x^2 + x + 1} = \frac{z^2 + z + 1}{1 + 2z}.$

代入得 $\int \frac{\mathrm{d}x}{x + \sqrt{x^2 + x + 1}} = \frac{1}{2}\int \frac{z^2 + z + 1}{z\left(z + \frac{1}{2}\right)^2}\mathrm{d}z = \frac{1}{2}\int \left[\frac{4}{z} -\frac{3}{z + \frac{1}{2}} -\frac{3}{2\left(z + \frac{1}{2}\right)^2}\right]\mathrm{d}z$

$$
= \frac {1}{2} \ln \frac {z ^ {4}}{\left| z + \frac {1}{2} \right| ^ {3}} + \frac {3}{4 \left(z + \frac {1}{2}\right)} + C _ {1} = \frac {1}{2} \ln \frac {z ^ {4}}{\left| 2 z + 1 \right| ^ {3}} + \frac {3}{2 (2 z + 1)} + C,
$$

其中 $z = x + \sqrt{x^2 + x + 1}$
