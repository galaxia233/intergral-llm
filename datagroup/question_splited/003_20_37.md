20.37 Verify the Mean-Value Theorem for integrals, for the function $f(x) = x^{2} + 5$ on $[0,3]$ .

$\frac{1}{b - a}\int_{a}^{b}f(x)dx = \frac{1}{3}\int_{0}^{3}(x^{2} + 5)dx = \frac{1}{3} (\frac{1}{3} x^{3} + 5x)\big]_{0}^{3} = \frac{1}{3} (9 + 15) = 8.$ But, $8 = x^{2} + 5$ when $x = \sqrt{3}$ and $0 < \sqrt{3} < 3$
