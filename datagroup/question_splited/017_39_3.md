39.3 Find the Taylor series for $\sin x$ about $\pi /4$

Let $f(x) = \sin x$ . Then $f(\pi / 4) = \sin (\pi / 4) = \sqrt{2} / 2$ , $f'(\pi / 4) = \cos (\pi / 4) = \sqrt{2} / 2$ , $f''(\pi / 4) = -\sin (\pi / 4) = -\sqrt{2} / 2$ , and, thereafter, this cycle of four values keeps repeating. Thus, the Taylor series for $\sin x$ about $\frac{\pi}{4}$ is $\frac{\sqrt{2}}{2} \left[1 + \frac{x - \pi / 4}{1!} - \frac{(x - \pi / 4)^2}{2!} - \frac{(x - \pi / 4)^3}{3!} + \frac{(x - \pi / 4)^4}{4!} + \frac{(x - \pi / 4)^5}{5!} - \frac{(x - \pi / 4)^6}{6!} - \frac{(x - \pi / 4)^7}{7!} + \dots \right]$ .
