【2048】 $\int \frac{\sin x\mathrm{d}x}{\sqrt{2} + \sin x + \cos x}.$

解此为2046题的特例，这里

$$
a _ {1} = 1, b _ {1} = 0, c _ {1} = 0, a = 1, b = 1, c = \sqrt {2}; A = \frac {1}{2}, B = - \frac {1}{2}, C = - \frac {1}{\sqrt {2}}.
$$

代入得 $\int \frac{\sin x\mathrm{d}x}{\sqrt{2} + \sin x + \cos x} = \frac{1}{2} x - \frac{1}{2}\ln |\sqrt{2} +\sin x + \cos x| - \frac{1}{\sqrt{2}}\int \frac{\mathrm{d}x}{\sqrt{2} + \sin x + \cos x}$

$$
= \frac {1}{2} x - \frac {1}{2} \ln | \sqrt {2} + \sin x + \cos x | - \frac {1}{\sqrt {2}} \int \frac {\mathrm {d} x}{\sqrt {2} + \sqrt {2} \cos \left(x - \frac {\pi}{4}\right)}
$$

$$
\begin{array}{l} = \frac {1}{2} x - \frac {1}{2} \ln | \sqrt {2} + \sin x + \cos x | - \frac {1}{2} \int \frac {\mathrm {d} x}{2 \cos^ {2} \left(\frac {x}{2} - \frac {\pi}{8}\right)} \\ = \frac {1}{2} x - \frac {1}{2} \ln | \sqrt {2} + \sin x + \cos x | - \frac {1}{2} \tan \left(\frac {x}{2} - \frac {\pi}{8}\right) + C. \\ \end{array}
$$
