20.27 $\int_{3}^{15}\sqrt[3]{x^2 - 9} x^3 dx.$

Let $u = x^{2} - 9, x^{2} = u + 9, du = 2x dx$ . Then $\int_{3}^{15}\sqrt[3]{x^2 - 9} x^3 dx = \frac{1}{2}\int_{0}^{216}u^{1 / 3}(u + 9)du = \frac{1}{2}\int_{0}^{216}(u^{4 / 3} + 9u^{1 / 3})du = \frac{1}{2} (\frac{3}{7} u^{7 / 3} + 9\cdot \frac{3}{4} u^{4 / 3})]_0^{216} = \frac{3}{56} u^{4 / 3}(4u + 63)]_0^{216} = \frac{3}{56}\{(6)^4 [4(216) + 63] - 0\} = \frac{349,522}{7}$ .
