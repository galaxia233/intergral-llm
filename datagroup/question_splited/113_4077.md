【4077】 $\iiint_{V} \frac{\mathrm{d}x \, \mathrm{d}y \, \mathrm{d}z}{(1 + x + y + z)^3}$ ，其中 $V$ 是曲面 $x + y + z = 1, x = 0, y = 0, z = 0$ 所围的区域.

解 $\iiint_{V} \frac{\mathrm{d}x \, \mathrm{d}y \, \mathrm{d}z}{(1 + x + y + z)^3} = \int_0^1 \mathrm{d}x \int_0^1 x \, \mathrm{d}y \int_0^{1 - x - y} \frac{\mathrm{d}z}{(1 + x + y + z)^3}$

$$
\begin{array}{l} = \int_ {0} ^ {1} d x \int_ {0} ^ {1 - x} \left[ - \frac {1}{2 (1 + x + y + z) ^ {2}} \right] \Bigg | _ {0} ^ {1 - x - y} d y = \int_ {0} ^ {1} d x \int_ {0} ^ {1 - x} \left[ - \frac {1}{8} + \frac {1}{2 (1 + x + y) ^ {2}} \right] d y \\ = \int_ {0} ^ {1} \left[ - \frac {1}{8} y - \frac {1}{2 (1 + x + y)} \right] \Bigg | _ {0} ^ {1} d x = \int_ {0} ^ {1} \left[ - \frac {3}{8} + \frac {x}{8} + \frac {1}{2 (1 + x)} \right] d x \\ = \left[ - \frac {3}{8} x + \frac {1}{1 6} x ^ {2} + \frac {1}{2} \ln (1 + x) \right] \Bigg | _ {0} ^ {1} = \frac {1}{2} \ln 2 - \frac {5}{1 6}. \\ \end{array}
$$
