39.32 Find $f^{(100)}(0)$ if $f(x) = e^{x^2}$ .

$$
e ^ {x} = \sum_ {n = 0} ^ {\infty} \frac {1}{n !} x ^ {n}. \quad \text {H e n c e}, \quad f (x) = \sum_ {n = 0} ^ {\infty} \frac {1}{n !} x ^ {2 n}. \quad \frac {f ^ {(1 0 0)} (0)}{1 0 0 !} = a _ {1 0 0} = \frac {1}{(5 0) !}. \quad \text {H e n c e}, \quad f ^ {(1 0 0)} (0) = \frac {1 0 0 !}{5 0 !}.
$$
