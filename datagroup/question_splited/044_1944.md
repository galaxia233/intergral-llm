【1944】 $\int \frac{x^{10}}{\sqrt{1 + x^2}}\mathrm{d}x.$

解设 $\int \frac{x^{10}}{\sqrt{1 + x^2}}\mathrm{d}x = (ax^9 +bx^8 +cx^7 +dx^6 +ex^5 +fx^4 +gx^3 +hx^2 +lx + m)\sqrt{1 + x^2} +\lambda \int \frac{\mathrm{d}x}{\sqrt{1 + x^2}},$ 从而有

$$
\begin{array}{l} x ^ {1 0} \equiv (9 a x ^ {8} + 8 b x ^ {7} + 7 c x ^ {6} + 6 d x ^ {5} + 5 e x ^ {4} + 4 f x ^ {3} + 3 g x ^ {2} + 2 h x + l) (1 + x ^ {2}) \\ + x \left(a x ^ {9} + b x ^ {8} + c x ^ {7} + d x ^ {6} + e x ^ {5} + f x ^ {4} + g x ^ {3} + h x ^ {2} + l x + m\right) + \lambda . \\ \end{array}
$$

比较等式两端 $x$ 的同次幂系数，求得

$$
a = \frac {1}{1 0}, b = 0, c = - \frac {9}{8 0}, d = 0, e = \frac {2 1}{1 6 0}, f = 0, g = - \frac {2 1}{1 2 8}, h = 0, l = \frac {6 3}{2 5 6}, m = 0, \lambda = - \frac {6 3}{2 5 6}.
$$

于是， $\int \frac{x^{10}}{\sqrt{1 + x^2}}\mathrm{d}x = \left(\frac{63}{256} x - \frac{21}{128} x^3 +\frac{21}{160} x^5 -\frac{9}{80} x^7 +\frac{1}{10} x^9\right)\sqrt{1 + x^2} -\frac{63}{256}\ln (x + \sqrt{1 + x^2}) + C.$
