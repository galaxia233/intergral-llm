【2518】若 $10\mathrm{N}$ 的力能使弹簧伸长 $1\mathrm{cm}$ ，现在要使这弹簧伸长 $10\mathrm{cm}$ ，问需要作多少功？

提示 利用胡克定律

解 由胡克定律知，弹性恢复力 $F$ 与伸长量 $\pmb{x}$ 成正比，即 $F = kx.$ 由条件知： $k = 10$ 因而， $F = 10x$

现将 $10\mathrm{cm}n$ 等分，每份上恢复力的大小近似地看作是不变的，并取右端点来作和，即得功 $\pmb{W}$ 的近似值为

$$
W \approx \sum_ {i = 1 } ^ { n } \frac { 1 0}{n} i \frac { 1 0 ^ {2}}{n}.
$$

显然， $n$ 愈大愈近似.于是，所要求的功为

$$
W = \lim _ {n \rightarrow \infty} \sum_ {i = 1 } ^ { n } \frac { 1 0}{n} i \frac { 1 0 ^ {2}}{n} = \lim _ {n \rightarrow \infty} 5 0 0 \frac {n + 1}{n} = 5 0 0 (N \cdot c m) = 5 (N \cdot m) = 5 J.
$$
