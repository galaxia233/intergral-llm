【2032】 $\int \frac{\sin x\cos x}{\sin x + \cos x}\mathrm{d}x.$

解 $\int \frac{\sin x\cos x}{\sin x + \cos x}\mathrm{d}x = \int \frac{\sin^2\left(x + \frac{\pi}{4}\right) - \frac{1}{2}}{\sqrt{2}\sin\left(x + \frac{\pi}{4}\right)}\mathrm{d}x = \frac{1}{\sqrt{2}}\int \sin \left(x + \frac{\pi}{4}\right)\mathrm{d}x - \frac{1}{2\sqrt{2}}\int \frac{\mathrm{d}x}{\sin\left(x + \frac{\pi}{4}\right)}$

$$
\begin{array}{l} = - \frac {1}{\sqrt {2}} \cos \left(x + \frac {\pi}{4}\right) - \frac {1}{2 \sqrt {2}} \ln \left| \tan \left(\frac {x}{2} + \frac {\pi}{8}\right) \right| + C \\ = \frac {1}{2} (\sin x - \cos x) - \frac {1}{2 \sqrt {2}} \ln \left| \tan \left(\frac {x}{2} + \frac {\pi}{8}\right) \right| + C. \\ \end{array}
