(2) $\int_0^1\int_0^1\dots \int_0^1 (x_1 + x_2 + \dots +x_n)^2\mathrm{d}x_1\mathrm{d}x_2\dots \mathrm{d}x_n$

$$
\begin{array}{l} = \int_ {0} ^ {1} d x _ {1} \int_ {0} ^ {1} d x _ {2} \dots \int_ {0} ^ {1} \left[ \left(x _ {1} ^ {2} + x _ {2} ^ {2} + \dots + x _ {n} ^ {2}\right) + 2 \left(x _ {1} x _ {2} + x _ {1} x _ {3} + \dots + x _ {1} x _ {n} + x _ {2} x _ {3} + \dots + x _ {2} x _ {n} + x _ {3} x _ {4} \right. \right. \\ \left. + \dots + x _ {3} x _ {n} + \dots + x _ {n - 1} x _ {n}\right) ] d x _ {n} \\ = \frac {n}{3} ^ {*}) + 2 \int_ {0} ^ {1} d x _ {1} \int_ {0} ^ {1} d x _ {2} \dots \int_ {0} ^ {1} \left[ \left(x _ {1} x _ {2} + \dots x _ {1} x _ {n}\right) + \left(x _ {2} x _ {3} + \dots + x _ {2} x _ {n}\right) + \dots + x _ {n - 1} x _ {n} \right] d x _ {n} \\ = \frac {n}{3} + 2 \left(\frac {n - 1}{4} + \frac {n - 2}{4} + \dots + \frac {1}{4}\right) = \frac {n (3 n + 1)}{1 2}. \\ \end{array}
$$

*) 利用本题 (1) 的结果.
