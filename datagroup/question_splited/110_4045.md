【4045】求曲面 $x^{2} + y^{2} = a^{2}$ 被平面 $x + z = 0, x - z = 0 (x > 0, y > 0)$ 所截部分的面积.

解 因为 $\sqrt{1 + \left(\frac{\partial y}{\partial x}\right)^2 + \left(\frac{\partial y}{\partial z}\right)^2} = \sqrt{1 + \left(\frac{x}{y}\right)^2} = \frac{a}{\sqrt{a^2 - x^2}}$

于是，所求的面积为 $S = \iint_{\Omega} \frac{a}{\sqrt{a^2 - x^2}} \mathrm{d}x \mathrm{~d}z = \int_{0}^{a} \mathrm{~d}x \int_{x}^{x'} \frac{a}{\sqrt{a^2 - x^2}} \mathrm{~d}z = \int_{0}^{a} \frac{2ax}{\sqrt{a^2 - x^2}} \mathrm{~d}x = 2a^2.$
