【4290】 $\mathrm{du} = (x^{2} - 2yz)\mathrm{dx} + (y^{2} - 2xz)\mathrm{dy} + (z^{2} - 2xy)\mathrm{dz}$

解 $\mathrm{du} = (x^{2}\mathrm{dx} + y^{2}\mathrm{dy} + z^{2}\mathrm{dz}) - 2(yz\mathrm{dx} + xz\mathrm{dy} + xy\mathrm{dz}) = \mathrm{d}\left(\frac{x^{3} + y^{3} + z^{3}}{3} -2xyz\right).$

于是， $u = \frac{1}{3} (x^3 +y^3 +z^3) - 2xyz + C.$
