【4001】求椭圆 $(a_{1}x + b_{1}y + c_{1})^{2} + (a_{2}x + b_{2}y + c_{2})^{2} = 1$ （其中 $\delta = a_1b_2 - a_2b_1\neq 0)$ 的面积.

提示 作变换 $a_1x + b_1y + c_1 = u$ ， $a_2x + b_2y + c_2 = v$ ，则椭圆所围区域变为 $u^2 + v^2 \leqslant 1$ ，且有 $|I| = \frac{1}{|\delta|}$

解 作变换 $a_1x + b_1y + c_1 = u, a_2x + b_2y + c_2 = v$ ，则椭圆所围区域变为 $u^2 + v^2 \leqslant 1$ ，且有 $|I| = \frac{1}{|\delta|}$ 。

于是，所求的面积为 $S = \frac{1}{|\delta|}\iint_{u^2 +v^2\leqslant 1}du\mathrm{d}v = \frac{\pi}{|\delta|}.$
