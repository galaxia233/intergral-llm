29.41 Find $\int \frac{\sqrt{a^2 - x^2}}{x} dx.$

Refer to Fig. 29-10. Let $x = a \sin \theta$ , $dx = a \cos \theta d\theta$ . Then $\int \frac{\sqrt{a^2 - x^2}}{x} dx = \int \frac{a \cos \theta \cdot a \cos \theta d\theta}{a \sin \theta} = a \int \frac{\cos^2 \theta}{\sin \theta} d\theta = a \int \frac {1 - \sin^2 \theta}{\sin \theta} d\theta = a (\int \csc \theta d\theta - \int \sin \theta d\theta) = a (\ln |\csc \theta - \cot \theta| + \cos \theta) + C = a \left( \ln \left| \frac{a}{x} - \frac{\sqrt{a^2 - x^2}}{x} \right| + \frac{\sqrt{a^2 - x^2}}{a} \right) + C = a \ln \left| \frac{a - \sqrt{a^2 - x^2}}{x} \right| + \sqrt{a^2 - x^2} + C.$
