【1896】 $\int \frac{x^2 + 3x - 2}{(x - 1)(x^2 + x + 1)^2}\mathrm{d}x.$

解 $Q = (x - 1)(x^{2} + x + 1)^{2}, Q_{1} = x^{2} + x + 1, Q_{2} = (x - 1)(x^{2} + x + 1).$

设 $\frac{x^2 + 3x - 2}{(x - 1)(x^2 + x + 1)^2} = \left(\frac{Ax + B}{x^2 + x + 1}\right)' + \frac{Cx^2 + Dx + E}{(x - 1)(x^2 + x + 1)}$ ，从而，

$$
x ^ {2} + 3 x - 2 \equiv A (x - 1) \left(x ^ {2} + x + 1\right) - (2 x + 1) (A x + B) (x - 1) + \left(C x ^ {2} + D x + E\right) \left(x ^ {2} + x + 1\right).
$$

比较等式两端 $x$ 的同次幂系数，得

$$
\begin{array}{l l} x ^ {4} & C = 0, \\ x ^ {3} & - A + C + D = 0, \\ x ^ {2} & A - 2 B + C + D + E = 1, \\ x ^ {1} & A + B + D + E = 3, \\ x ^ {0} & - A + B + E = - 2. \end{array}
$$

由此， $A = \frac{5}{3},B = \frac{2}{3},C = 0,D = \frac{5}{3},E = -1$ ，再将 $\frac{\frac{5}{3}x - 1}{(x - 1)(x^2 + x + 1)}$ 分解，可得

$$
\frac {\frac {5}{3} x - 1}{(x - 1) \left(x ^ {2} + x + 1\right)} = \frac {2}{9 (x - 1)} - \frac {2 x - 1 1}{9 \left(x ^ {2} + x + 1\right)}.
$$

于是， $\int \frac{x^2 + 3x - 2}{(x - 1)(x^2 + x + 1)^2}\mathrm{d}x = \frac{5x + 2}{3(x^2 + x + 1)} +\frac{2}{9}\int \frac{\mathrm{d}x}{x - 1} -\frac{1}{9}\int \frac{2x - 11}{x^2 + x + 1}\mathrm{d}x$

$$
\begin{array}{l} = \frac {5 x + 2}{3 \left(x ^ {2} + x + 1\right)} + \frac {2}{9} \ln | x - 1 | - \frac {1}{9} \int \frac {2 x + 1}{x ^ {2} + x + 1} d x + \frac {4}{3} \int \frac {d \left(x + \frac {1}{2}\right)}{\left(x + \frac {1}{2}\right) ^ {2} + \frac {3}{4}} \\ = \frac {5 x + 2}{3 \left(x ^ {2} + x + 1\right)} + \frac {1}{9} \ln \frac {(x - 1) ^ {2}}{x ^ {2} + x + 1} + \frac {8}{3 \sqrt {3}} \arctan \left(\frac {2 x + 1}{\sqrt {3}}\right) + C. \\ \end{array}
$$
