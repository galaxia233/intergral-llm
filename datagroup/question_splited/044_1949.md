【1949】+ $\int \frac{\mathrm{d}x}{(x - 1)^3\sqrt{x^2 + 3x + 1}}.$

解设 $x - 1 = \frac{1}{t}$ ，则 $\mathrm{d}x = -\frac{1}{t^2}\mathrm{d}t.$ 不妨设 $t > 0$ ，则有

$$
\sqrt {x ^ {2} + 3 x + 1} = \frac {\sqrt {5 t ^ {2} + 5 t + 1}}{t}.
$$

代入得 $\int \frac{\mathrm{d}x}{(x - 1)^3\sqrt{x^2 + 3x + 1}} = -\int \frac{t^2}{\sqrt{5t^2 + 5t + 1}}\mathrm{d}t = (at + b)\sqrt{5t^2 + 5t + 1} +\lambda \int \frac{\mathrm{d}t}{\sqrt{5t^2 + 5t + 1}},$

从而有 $-t^{2} \equiv a(1 + 5t + 5t^{2}) + \left(5t + \frac{5}{2}\right)(at + b) + \lambda.$

比较等式两端 $t$ 的同次幂系数，求得 $a = -\frac{1}{10}, b = \frac{3}{20}, \lambda = -\frac{11}{40}.$

于是， $\int \frac{\mathrm{d}x}{(x - 1)^3\sqrt{x^2 + 3x + 1}} = \left(-\frac{t}{10} +\frac{3}{20}\right)\sqrt{5t^2 + 5t + 1} -\frac{11}{40}\int \frac{\mathrm{d}t}{\sqrt{5t^2 + 5t + 1}}$

$$
\begin{array}{l} = \frac {3 - 2 t}{2 0} \sqrt {5 t ^ {2} + 5 t + 1} - \frac {1 1}{4 0 \sqrt {5}} \ln \left| t + \frac {1}{2} + \sqrt {t ^ {2} + t + \frac {1}{5}} \right| + C _ {1} \\ = \frac {3 x - 5}{2 0 (x - 1) ^ {2}} \sqrt {x ^ {2} + 3 x + 1} - \frac {1 1}{4 0 \sqrt {5}} \ln \left| \frac {\sqrt {5} (x + 1) + 2 \sqrt {x ^ {2} + 3 x + 1}}{x - 1} \right| + C. \\ \end{array}
