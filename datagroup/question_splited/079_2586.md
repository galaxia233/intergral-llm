【2586】 $\sum_{n = 1}^{\infty}\frac{n^{2}}{\left(2 + \frac{1}{n}\right)^{n}}.$

提示 利用柯西判别法及65题的结果.

解由于

$$
\lim  _ {n \rightarrow \infty} \sqrt [ n ]{a _ {n}} = \lim  _ {n \rightarrow \infty} \sqrt [ n ]{\frac {n ^ {2}}{\left(2 + \frac {1}{n}\right) ^ {n}}} = \lim  _ {n \rightarrow \infty} \frac {\sqrt [ n ]{n} \sqrt [ n ]{n}}{2 + \frac {1}{n}} = \frac {1}{2} <   1,
$$

故级数 $\sum_{n=1}^{\infty} \frac{n^{2}}{\left(2 + \frac{1}{n}\right)^{n}}$ 收敛.
