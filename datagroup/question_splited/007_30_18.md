30.18 $\int \frac{x^3 + 1}{x(x^2 + x + 1)^2} dx.$

$x^{2} + x + 1$ is irreducible $(b^{2} - 4ac = -3 < 0)$ . $\frac{x^3 + 1}{x(x^2 + x + 1)^2} = \frac{A}{x} + \frac{Bx + C}{x^2 + x + 1} + \frac{Dx + E}{(x^2 + x + 1)^2}$ . Then $x^3 + 1 = A(x^2 + x + 1)^2 + x(x^2 + x + 1)(Bx + C) + x(Dx + E)$ . Let $x = 0$ . Then $1 = 9A$ , $A = \frac{1}{9}$ . Equate coefficients of $x^4$ : $0 = A + B$ , $B = -A = -\frac{1}{9}$ . Equate coefficients of $x^3$ : $1 = 2A + C + B$ . Then $C = 1 - 2A - B = \frac{8}{9}$ . Equate coefficients of $x^2$ : $0 = 3A + B + C + D$ . Then $D = -3A - B - C = -\frac{10}{9}$ .

Equate coefficients of $x$ : $0 = 2A + C + E, \quad E = 2A - C = -\frac{10}{9}$ . Thus, $\frac{x^3 + 1}{x(x^2 + x + 1)^2} = \frac{1}{9}\frac{1}{x} +$

$$
\begin{array}{l} \frac {1}{9} \frac {- x + 8}{x ^ {2} + x + 1} - \frac {1 0}{9} \frac {x + 1}{\left(x ^ {2} + x + 1\right) ^ {2}}. \quad \text {T h e n} \quad \int \frac {x ^ {3} + 1}{x \left(x ^ {2} + x + 1\right) ^ {2}} d x = \frac {1}{9} \ln | x | + \frac {1}{9} \int \frac {8 - x}{x ^ {2} + x + 1} d x - \\ \frac {1 0}{9} \int \frac {(x + 1) d x}{(x ^ {2} + x + 1) ^ {2}}. \\ \end{array}
$$

$$
\begin{array}{l} \int \frac {8 - x}{x ^ {2} + x + 1} d x = \int \frac {(8 - x) d x}{(x + \frac {1}{2}) ^ {2} + \frac {3}{4}} = \int \frac {\left(- x - \frac {1}{2}\right) + \frac {1 7}{2}}{\left(x + \frac {1}{2}\right) ^ {2} + \frac {3}{4}} d x \\ = - \int \frac {x + \frac {1}{2}}{(x + \frac {1}{2}) ^ {2} + \frac {3}{4}} d x + \frac {1 7}{2} \int \frac {d x}{(x + \frac {1}{2}) ^ {2} + \frac {3}{4}} \\ = - \frac {1}{2} \ln \left[ \left(x + \frac {1}{2}\right) ^ {2} + \frac {3}{4}\right) \bigg ] + \frac {1 7}{2} \cdot \frac {2}{\sqrt {3}} \tan^ {- 1} \frac {x + \frac {1}{2}}{\sqrt {3} / 2} \\ = - \frac {1}{2} \ln (x ^ {2} + x + 1) + \frac {1 7}{\sqrt {3}} \tan^ {- 1} \frac {2 x + 1}{\sqrt {3}} \tag {I} \\ \end{array}
$$

$$
\begin{array}{l} \int \frac {(x + 1) d x}{(x ^ {2} + x + 1) ^ {2}} = \int \frac {(x + 1) d x}{[ (x + \frac {1}{2}) ^ {2} + \frac {3}{4} ] ^ {2}} = \int \frac {(x + \frac {1}{2}) + \frac {1}{2}}{[ (x + \frac {1}{2}) ^ {2} + \frac {3}{4} ] ^ {2}} d x \\ = \int \frac {\left(x + \frac {1}{2}\right) d x}{\left[ \left(x + \frac {1}{2}\right) ^ {2} + \frac {3}{4} \right] ^ {2}} + \frac {1}{2} \int \frac {d x}{\left[ \left(x + \frac {1}{2}\right) ^ {2} + \frac {3}{4} \right] ^ {2}} \\ = - \frac {1}{2} \frac {1}{\left(x + \frac {1}{2}\right) ^ {2} + \frac {3}{4}} + \frac {1}{2} \cdot \frac {1}{2 \left(\sqrt {3} / 2\right) ^ {3}} \left[ \tan^ {- 1} \frac {x + \frac {1}{2}}{\sqrt {3} / 2} + \frac {(\sqrt {3} / 2) x}{\left(x + \frac {1}{2}\right) ^ {2} + \frac {3}{4}} \right] \\ = - \frac {1}{2} \frac {1}{x ^ {2} + x + 1} + \frac {2}{3 \sqrt {3}} \tan^ {- 1} \frac {2 x + 1}{\sqrt {3}} + \frac {1}{3} \frac {x}{x ^ {2} + x + 1} \\ = \frac {1}{6} \frac {2 x - 3}{x ^ {2} + x + 1} + \frac {2}{3 \sqrt {3}} \tan^ {- 1} \frac {2 x + 1}{\sqrt {3}} \tag {II} \\ \end{array}
$$

Hence, the complete answer is $\frac{1}{9} \ln |x| + \frac{1}{9} \left[-\frac{1}{2} \ln (x^2 + x + 1) + \frac{17}{\sqrt{3}} \tan^{-1} \frac{2x + 1}{\sqrt{3}}\right] - \frac{10}{9} \left(\frac{1}{6} \frac{2x - 1}{x^2 + x + 1} + \frac{2}{3\sqrt{3}} \tan^{-1} \frac{2x + 1}{\sqrt{3}}\right)$ .
