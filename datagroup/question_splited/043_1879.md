【1879】 $\int \frac{x\mathrm{d}x}{(x - 1)^2(x^2 + 2x + 2)}.$

解设 $\frac{x}{(x - 1)^2(x^2 + 2x + 2)} = \frac{A}{x - 1} +\frac{B}{(x - 1)^2} +\frac{Cx + D}{x^2 + 2x + 2}$ ，通分后应有

$$
x \equiv A (x - 1) (x ^ {2} + 2 x + 2) + B (x ^ {2} + 2 x + 2) + (C x + D) (x - 1) ^ {2}.
$$

比较等式两端 $x$ 的同次幂系数，得

$$
\begin{array}{l l} x ^ {3} & A + C = 0, \\ x ^ {2} & A + B - 2 C + D = 0, \\ x ^ {1} & 2 B + C - 2 D = 1, \\ x ^ {0} & - 2 A + 2 B + D = 0. \end{array}
$$

由此 $A = \frac{1}{25}, B = \frac{1}{5}, C = -\frac{1}{25}, D = -\frac{8}{25}$ . 于是，

$$
\begin{array}{l} \int \frac {x \mathrm {d} x}{(x - 1) ^ {2} (x ^ {2} + 2 x + 2)} = \int \left[ \frac {1}{2 5 (x - 1)} + \frac {1}{5 (x - 1) ^ {2}} - \frac {x + 8}{2 5 (x ^ {2} + 2 x + 2)} \right] \mathrm {d} x \\ = \frac {1}{2 5} \ln | x - 1 | - \frac {1}{5 (x - 1)} - \frac {1}{5 0} \int \frac {2 x + 2}{x ^ {2} + 2 x + 2} d x - \frac {7}{2 5} \int \frac {d (x + 1)}{(x + 1) ^ {2} + 1} \\ = \frac {1}{2 5} \ln | x - 1 | - \frac {1}{5 (x - 1)} - \frac {1}{5 0} \ln (x ^ {2} + 2 x + 2) - \frac {7}{2 5} \arctan (x + 1) + C \\ = \frac {1}{5 0} \ln \frac {(x - 1) ^ {2}}{x ^ {2} + 2 x + 2} - \frac {1}{5 (x - 1)} - \frac {7}{2 5} \arctan (x + 1) + C. \\ \end{array}
$$
