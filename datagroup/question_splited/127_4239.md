【4239】 $\int_{C} z \mathrm{~d}s$ ，其中 $C$ 为圆锥螺旋线 $x = t \cos t, y = t \sin t, z = t (0 \leqslant t \leqslant t_0)$

解 弧长的微分为 $\mathrm{ds} = \sqrt{(\cos t - t\sin t)^2 + (\sin t + t\cos t)^2 + 1}\mathrm{dt} = \sqrt{2 + t^2}\mathrm{dt}.$

于是， $\int_{C}z\mathrm{d}s = \int_{0}^{t_0}t\sqrt{2 + t^2}\mathrm{d}t = \frac{1}{3} [(2 + t_0^2)^{\frac{3}{2}} - 2^{\frac{3}{2}}].$
