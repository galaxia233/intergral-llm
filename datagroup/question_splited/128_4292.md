【4292】 $\mathrm{du} = \frac{(x + y - z)\mathrm{dx} + (x + y - z)\mathrm{dy} + (x + y + z)\mathrm{dz}}{x^2 + y^2 + z^2 + 2xy}$

解由于

$$
\begin{array}{l} (x + y - z) d x + (x + y - z) d y + (x + y + z) d z \\ = (x \mathrm {d} x + y \mathrm {d} y) + (y \mathrm {d} x + x \mathrm {d} y) + (x + y) \mathrm {d} z - z (\mathrm {d} x + \mathrm {d} y) + z \mathrm {d} z \\ = \frac {1}{2} d \left[ \left(x ^ {2} + y ^ {2} + 2 x y\right) + z ^ {2} \right] + (x + y) d z - z d (x + y), \\ \end{array}
$$

故 $\mathrm{d}u = \frac{1}{2}\frac{\mathrm{d}\left[(x + y)^2 + z^2\right]}{(x + y)^2 + z^2} +\frac{(x + y)\mathrm{d}z - z\mathrm{d}(x + y)}{(x + y)^2 + z^2} = \frac{1}{2}\mathrm{d}\ln [(x + y)^2 +z^2 ] + \mathrm{d}\Big(\arctan \frac{z}{x + y}\Big)$

$$
= d \left[ \ln \sqrt {(x + y) ^ {2} + z ^ {2}} + \arctan \frac {z}{x + y} \right].
$$

于是， $u = \ln \sqrt{(x + y)^2 + z^2} +\arctan \frac{z}{x + y} +C.$
