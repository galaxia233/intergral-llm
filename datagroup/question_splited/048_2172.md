【2172】 $\int \varphi (x)\mathrm{d}x$ ，其中 $\varphi (x)$ 为数 $x$ 至其最接近的整数之距离.

提示仿 2170 题，可求满足 $F(0) = 0$ 的原函数 $F(x)$

解 显然 $\varphi(x)$ 在 $(- \infty, +\infty)$ 上连续，故其原函数在 $(- \infty, +\infty)$ 上连续可微。今求满足 $F(0) = 0$ 的原函数。由于

$$
\varphi (x) = \left\{ \begin{array}{l l} x - n, & n \leqslant x <   n + \frac {1}{2}, \\ - x + n + 1, & n + \frac {1}{2} \leqslant x <   n + 1, \end{array} \right.
$$

故 $F(x) = \left\{ \begin{array}{ll} \frac{x^2}{2} - nx + C_n, & n \leqslant x < n + \frac{1}{2}, \\ -\frac{x^2}{2} + (n + 1)x + C_n', & n + \frac{1}{2} \leqslant x < n + 1, \end{array} \right.$

其中 $C_n, C_n'$ 是两个常数。由 $\lim_{x \to \left(n + \frac{1}{2}\right) - 0} F(x) = F\left(n + \frac{1}{2}\right)$ 得， $C_n' = C_n - \left(n + \frac{1}{2}\right)^2$ 故

$$
F (x) = \left\{ \begin{array}{l l} \frac {x ^ {2}}{2} - n x + C _ {n}, & n \leqslant x <   n + \frac {1}{2}, \\ - \frac {x ^ {2}}{2} + (n + 1) x - \left(n + \frac {1}{2}\right) ^ {2} + C _ {n}, & n + \frac {1}{2} \leqslant x <   n + 1. \end{array} \right.
$$

由 $\lim_{x\to (n + 1) - 0}F(x) = F(n + 1)$ 得递推公式 $C_{n + 1} = C_n + n + \frac{3}{4}$

显然 $0 = F(0) = C_0$ 由此得 $C_n = \frac{1}{4} n(2n + 1)$ . 于是，

$$
F (x) = \left\{ \begin{array}{l} \frac {x ^ {2}}{2} - n x + \frac {1}{4} n (2 n + 1) = \frac {x}{4} + \frac {1}{4} \left(x - n - \frac {1}{2}\right) \left[ 1 + 2 \left(\frac {1}{2} - x - n\right) \right], \\ \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad n \leqslant x <   n + \frac {1}{2}, \\ - \frac {x ^ {2}}{2} + (n + 1) x - \frac {1}{4} (2 n + 1) (n + 1) = \frac {x}{4} + \frac {1}{4} \left(x - n - \frac {1}{2}\right) \left[ 1 - 2 \left(x - n - \frac {1}{2}\right) \right], \\ \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad \qquad n + \frac {1}{2} \leqslant x <   n + 1. \end{array} \right.
$$

记 $(x) = x - [x]$ 表数 $x$ 去掉其整数部分 $[x]$ 后所剩下的零头部分，那么最后得

$$
F (x) = \frac {x}{4} + \frac {1}{4} \left[ (x) - \frac {1}{2} \right] \left\{1 - 2 \left| (x) - \frac {1}{2} \right| \right\} (- \infty <   x <   + \infty).
$$

故 $\int \varphi (x)\mathrm{d}x = \frac{x}{4} +\frac{1}{4}\left[(x) - \frac{1}{2}\right]\left\{1 - 2\left|(x) - \frac{1}{2}\right|\right\} +C(-\infty < x < + \infty).$
