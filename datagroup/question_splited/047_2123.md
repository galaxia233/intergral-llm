【2123】 $\int \frac{\mathrm{d}x}{\mathrm{sh}x + 2\mathrm{ch}x}$

解设 $\mathrm{th}\frac{x}{2} = t$ ，则 $\mathrm{sh}x = \frac{2t}{1 - t^2},\quad \mathrm{ch}x = \frac{1 + t^2}{1 - t^2},x = \ln \frac{1 + t}{1 - t},\quad \mathrm{d}x = \frac{2}{1 - t^2}\mathrm{d}t.$ 代人得

$$
\int \frac {\mathrm {d} x}{\operatorname {s h } x + 2 \operatorname {c h } x} = \int \frac {\mathrm {d} t}{t ^ {2} + t + 1} = \frac {2}{\sqrt {3}} \arctan \frac {2 t + 1}{\sqrt {3}} + C = \frac {2}{\sqrt {3}} \arctan \frac {1 + 2 \operatorname {t h } \frac {x}{2}}{\sqrt {3}} + C.
$$
