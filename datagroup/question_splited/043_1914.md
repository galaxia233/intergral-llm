【1914】 $\int \frac{\mathrm{d}x}{x(x^{10} + 1)^2}$

提示 两次使用 $1 = x^{10} + 1 - x^{10}$ , 就可将被积函数变形为 $\frac{1}{x} - \frac{x^9}{x^{10} + 1} - \frac{x^9}{(x^{10} + 1)^2}$ , 从而易获解。

解 由于

$$
\frac {1}{x \left(x ^ {1 0} + 1\right) ^ {2}} = \frac {x ^ {1 0} + 1 - x ^ {1 0}}{x \left(x ^ {1 0} + 1\right) ^ {2}} = \frac {1}{x \left(x ^ {1 0} + 1\right)} - \frac {x ^ {9}}{\left(x ^ {1 0} + 1\right) ^ {2}} = \frac {1}{x} - \frac {x ^ {9}}{x ^ {1 0} + 1} - \frac {x ^ {9}}{\left(x ^ {1 0} + 1\right) ^ {2}},
$$

所以， $\int \frac{\mathrm{d}x}{x(x^{10} + 1)^2} = \int \left[\frac{1}{x} -\frac{x^9}{x^{10} + 1} -\frac{x^9}{(x^{10} + 1)^2}\right]\mathrm{d}x = \ln |x| - \frac{1}{10}\int \frac{\mathrm{d}(x^{10} + 1)}{x^{10} + 1} -\frac{1}{10}\int \frac{\mathrm{d}(x^{10} + 1)}{(x^{10} + 1)^2}$

$$
= \ln | x | - \frac {1}{1 0} \ln (x ^ {1 0} + 1) + \frac {1}{1 0 (x ^ {1 0} + 1)} + C = \frac {1}{1 0} \ln \frac {x ^ {1 0}}{x ^ {1 0} + 1} + \frac {1}{1 0 (x ^ {1 0} + 1)} + C.
$$
