20.4 Prove the formula $1 + 2 + \dots + n = \frac{n(n + 1)}{2}$ .

Let $S = 1 + 2 + \dots + (n - 1) + n$ . Then we also can write $S = n + (n - 1) + \dots + 2 + 1$ . If we add these two equations column by column, we see that $2S$ is equal to the number $n + 1$ added to itself $n$ times. Thus, $2S = n(n + 1)$ , $S = n(n + 1) / 2$ .
