【2344】 $\int_{a}^{+\infty}\frac{x\ln x}{(1 + x^2)}\mathrm{d}x.$

解 我们有

$$
\begin{array}{l} \int \frac {x \ln x}{(1 + x ^ {2}) ^ {2}} d x = - \frac {1}{2} \int \ln x d \left(\frac {1}{1 + x ^ {2}}\right) = - \frac {\ln x}{2 (1 + x ^ {2})} + \frac {1}{2} \int \frac {d x}{x (1 + x ^ {2})} \\ = - \frac {\ln x}{2 \left(1 + x ^ {2}\right)} + \frac {1}{2} \int \left(\frac {1}{x} - \frac {x}{1 + x ^ {2}}\right) d x = - \frac {\ln x}{2 \left(1 + x ^ {2}\right)} + \frac {1}{4} \ln \frac {x ^ {2}}{1 + x ^ {2}} + C. \\ \end{array}
$$

由于 $\lim_{\substack{t\to +0\\ b\to +\infty}}\int_a^b\frac{x\ln x}{(1 + x^2)^t}\mathrm{d}x = \lim_{\substack{t\to +0\\ b\to +\infty}}\left[-\frac{\ln x}{2(1 + x^2)} +\frac{1}{4}\ln \frac{x^2}{1 + x^2}\right]\Bigg{|}_t^b.$

$$
\begin{array}{l} = \lim  _ {\substack {\epsilon \rightarrow + 0\\b \rightarrow + \infty}} \left[ - \frac {\ln b}{2 (1 + b ^ {2})} + \frac {\ln \varepsilon}{2 (1 + \varepsilon^ {2})} + \frac {1}{4} \ln \frac {b ^ {2}}{b ^ {2} + 1} - \frac {1}{4} \ln \frac {\varepsilon^ {2}}{\varepsilon^ {2} + 1} \right] \\ = \lim  _ { \varepsilon \rightarrow + 0 } \left[ - \frac {\varepsilon^ {2}}{2 \left(\varepsilon^ {2} + 1\right)} \ln \varepsilon + \frac {1}{4} \ln \left(\varepsilon^ {2} + 1\right)\right] = 0, \\ \end{array}
$$

所以， $\int_0^{+\infty}\frac{x\ln x}{(1 + x^2)^2}\mathrm{d}x = 0.$

注 $\varepsilon \rightarrow +0$ 与 $b \rightarrow +\infty$ 的极限过程是独立的，因此，可分别取极限.
