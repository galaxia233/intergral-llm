【2045】 $\int \frac{a_1\sin x + b_1\cos x}{(a\sin x + b\cos x)^2}\mathrm{d}x.$

提示仿2042题，并利用2041题的结果

解仿2042题，有

$$
\begin{array}{l} \int \frac {a _ {1} \sin x + b _ {1} \cos x}{(a \sin x + b \cos x) ^ {2}} d x = A \int \frac {a \sin x + b \cos x}{(a \sin x + b \cos x) ^ {2}} d x + B \int \frac {a \cos x - b \sin x}{(a \sin x + b \cos x) ^ {2}} d x \\ = A \int \frac {d x}{a \sin x + b \cos x} d x + B \int \frac {d (a \sin x + b \cos x)}{(a \sin x + b \cos x) ^ {2}} = \frac {A}{\sqrt {a ^ {2} + b ^ {2}}} \ln \left| \tan \left(\frac {x}{2} + \frac {\varphi}{2}\right) \right| ^ {*}) - \frac {B}{a \sin x + b \cos x} + C \\ = \frac {a a _ {1} + b b _ {1}}{\left(a ^ {2} + b ^ {2}\right) ^ {\frac {3}{2}}} \ln \left| \tan \left(\frac {x}{2} + \frac {\varphi}{2}\right) \right| - \frac {a b _ {1} - a _ {1} b}{\left(a ^ {2} + b ^ {2}\right) \left(a \sin x + b \cos x\right)} + C, \\ \end{array}
$$

式中 $A = \frac{aa_1 + bb_1}{a^2 + b^2},\quad B = \frac{ab_1 - a_1b}{a^2 + b^2},\cos \varphi = \frac{a}{\sqrt{a^2 + b^2}},\quad \sin \varphi = \frac{b}{\sqrt{a^2 + b^2}},\quad a^2 +b^2\neq 0$

（显然按题意 $a, b$ 不同时为零）.

\*) 利用2041题的结果.
