【1692】 $\int \frac{\mathrm{d}x}{\sqrt{1 + \mathrm{e}^{2x}}}$

提示 注意 $\frac{\mathrm{d}x}{\sqrt{1 + \mathrm{e}^{2x}}} = -\frac{\mathrm{d}(\mathrm{e}^{-x})}{\sqrt{1 + (\mathrm{e}^{-x})^2}}.$

解 $\int \frac{\mathrm{d}x}{\sqrt{1 + \mathrm{e}^{2x}}} = -\int \frac{\mathrm{d}(\mathrm{e}^{-x})}{\sqrt{1 + (\mathrm{e}^{-x})^2}} = -\ln (\mathrm{e}^{-x} + \sqrt{1 + \mathrm{e}^{-2x}}) + C.$
