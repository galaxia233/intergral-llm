【1962】 $\int \frac{x^2\mathrm{d}x}{(4 - 2x + x^2)\sqrt{2 + 2x - x^2}}.$

解 $\int \frac{x^2\mathrm{d}x}{(4 - 2x + x^2)\sqrt{2 + 2x - x^2}} = \int \frac{(x - 1)^2 + 2(x - 1) + 1}{[3 + (x - 1)^2]\sqrt{3 - (x - 1)^2}}\mathrm{d}x.$

设 $x - 1 = \sqrt{3}\sin t$ ，并限制 $-\frac{\pi}{2} < t < \frac{\pi}{2}$ 则 $\mathrm{dx} = \sqrt{3}\cos t\mathrm{dt},\sqrt{2 + 2x - x^2} = \sqrt{3}\cos t.$

代入得 $\int \frac{x^2\mathrm{d}x}{(4 - 2x + x^2)\sqrt{2 + 2x - x^2}} = \int \frac{1 + 2\sqrt{3}\sin t + 3\sin^2t}{3(1 + \sin^2t)}\mathrm{d}t$

$$
\begin{array}{l} = \int \mathrm {d} t + \frac {2}{\sqrt {3}} \int \frac {\sin t}{1 + \sin^ {2} t} \mathrm {d} t - \frac {2}{3} \int \frac {\mathrm {d} t}{1 + \sin^ {2} t} = t - \frac {2}{\sqrt {3}} \int \frac {\mathrm {d} (\cos t)}{2 - \cos^ {2} t} - \frac {2}{3} \int \frac {\mathrm {d} (\tan t)}{1 + 2 \tan^ {2} t} \\ = t - \frac {1}{\sqrt {6}} \ln \left| \frac {\sqrt {2} + \cos t}{\sqrt {2} - \cos t} \right| - \frac {\sqrt {2}}{3} \arctan (\sqrt {2} \tan t) + C \\ = \arcsin \frac {x - 1}{\sqrt {3}} - \frac {1}{\sqrt {6}} \ln \frac {\sqrt {6} + \sqrt {2 + 2 x - x ^ {2}}}{\sqrt {6} - \sqrt {2 + 2 x - x ^ {2}}} - \frac {\sqrt {2}}{3} \arctan \frac {(x - 1) \sqrt {2}}{\sqrt {2 + 2 x - x ^ {2}}} + C. \\ \end{array}
