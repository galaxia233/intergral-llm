【4275】 $\mathrm{d}z = \frac{\partial^{n + m + 1}u}{\partial x^{n + 1}\partial y^m}\mathrm{d}x + \frac{\partial^{n + m + 1}u}{\partial x^n\partial y^{m + 1}}\mathrm{d}y.$

提示 注意 $\mathrm{d}z = \mathrm{d}\left(\frac{\partial^{n + m}u}{\partial x^n\partial y^m}\right)$

解 因为 $\mathrm{d}z = \frac{\partial^{n + m + 1}u}{\partial x^{n + 1}\partial y^m}\mathrm{d}x + \frac{\partial^{n + m + 1}u}{\partial x^n\partial y^{m + 1}}\mathrm{d}y = \mathrm{d}\left(\frac{\partial^{n + m}u}{\partial x^n\partial y^m}\right).$

故有 $z = \frac{\partial^{n + m}u}{\partial x^n\partial y^m} +C.$
