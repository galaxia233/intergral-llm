【2157】 $\int \frac{x\ln(x + \sqrt{1 + x^2})}{(1 - x^2)^2}\mathrm{d}x.$

解 $\int \frac{x\ln(x + \sqrt{1 + x^2})}{(1 - x^2)^2}\mathrm{d}x = \frac{1}{2}\int \ln (x + \sqrt{1 + x^2})\mathrm{d}\left(\frac{1}{1 - x^2}\right)$

$$
= \frac {1}{2 \left(1 - x ^ {2}\right)} \ln \left(x + \sqrt {1 + x ^ {2}}\right) - \frac {1}{2} \int \frac {\mathrm {d} x}{\left(1 - x ^ {2}\right) \sqrt {x ^ {2} + 1}}.
$$

对于右端积分设 $x = \tan t$ ，并限制 $-\frac{\pi}{2} < t < \frac{\pi}{2}$ ，则 $\sqrt{1 + x^2} = \sec t$ ， $\mathrm{dx} = \sec^2 t\mathrm{dt}$ 代入得

$$
\begin{array}{l} \int \frac {d x}{(1 - x ^ {2}) \sqrt {1 + x ^ {2}}} = \int \frac {\sec t d t}{1 - \tan^ {2} t} = \int \frac {\cos t d t}{\cos^ {2} t - \sin^ {2} t} = \int \frac {d (\sin t)}{1 - 2 \sin^ {2} t} = \frac {1}{2 \sqrt {2}} \ln \left| \frac {1 + \sqrt {2} \sin t}{1 - \sqrt {2} \sin t} \right| + C \\ = \frac {1}{2 \sqrt {2}} \ln \left| \frac {\sqrt {1 + x ^ {2}} + x \sqrt {2}}{\sqrt {1 + x ^ {2}} - x \sqrt {2}} \right| + C, \\ \end{array}
$$

于是， $\int \frac{x\ln(x + \sqrt{1 + x^2})}{(1 + x^2)^2}\mathrm{d}x = \frac{\ln(x + \sqrt{1 + x^2})}{2(1 - x^2)} +\frac{1}{4\sqrt{2}}\ln \left|\frac{\sqrt{1 + x^2} - x\sqrt{2}}{\sqrt{1 + x^2} + x\sqrt{2}}\right| + C.$
