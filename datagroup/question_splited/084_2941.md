【2941】在区间 $(0, 2\pi)$ 内展开 $f(x) = \frac{\pi - x}{2}$ .

解由于

$$
a _ {0} = \frac {1}{\pi} \int_ {0} ^ {2 \pi} \frac {\pi - x}{2} d x = \frac {1}{2 \pi} \left(\pi x - \frac {x ^ {2}}{2}\right) \Bigg | _ {0} ^ {2 \pi} = 0,
$$

$$
a _ {n} = \frac {1}{\pi} \int_ {0} ^ {2 \pi} \frac {\pi - x}{2} \cos n x d x = \frac {\pi - x}{2 n \pi} \sin n x \left| _ {0} ^ {2 \pi} + \frac {1}{2 n \pi} \int_ {0} ^ {2 \pi} \sin n x d x = 0; \right.
$$

$$
b _ {n} = \frac {1}{\pi} \int_ {0} ^ {2 \pi} \frac {\pi - x}{2} \sin n x d x = - \frac {\pi - x}{2 n \pi} \cos n x \left| _ {0} ^ {2 \pi} - \frac {1}{2 n \pi} \int_ {0} ^ {2 \pi} \cos n x d x = \frac {1}{n}, \right.
$$

故按展开定理， $f(x)$ 可展开为 $\sum_{n = 1}^{\infty}\frac{\sin nx}{n} = \frac{\pi - x}{2}\quad (0 < x < 2\pi).$
