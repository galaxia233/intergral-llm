【2051】 $\int \frac{\sin^2x - 4\sin x\cos x + 3\cos^2x}{\sin x + \cos x}\mathrm{d}x.$

解此为2050题的特例，这里

$$
a _ {1} = 1, b _ {1} = - 2, c _ {1} = 3, a = 1, b = 1;
$$

$$
A = \frac {b c _ {1} - a _ {1} b + 2 a b _ {1}}{a ^ {2} + b ^ {2}} = \frac {3 - 1 - 4}{1 + 1} = - 1, B = \frac {a c _ {1} - a a _ {1} - 2 b b _ {1}}{a ^ {2} + b ^ {2}} = \frac {3 - 1 + 4}{1 + 1} = 3, C = \frac {a _ {1} b ^ {2} + a ^ {2} c _ {1} - 2 a b b _ {1}}{a ^ {2} + b ^ {2}} = \frac {1 + 3 + 4}{1 + 1} = 4.
$$

代入得 $\int \frac{\sin^2 x - 4\sin x\cos x + 3\cos^2 x}{\sin x + \cos x}\mathrm{d}x = -\sin x + 3\cos x + 4\int \frac{\mathrm{d}x}{\sin x + \cos x}$

$$
= - \sin x + 3 \cos x + \frac {4}{\sqrt {2}} \int \frac {\mathrm {d} x}{\sin \left(x + \frac {\pi}{4}\right)} = - \sin x + 3 \cos x + 2 \sqrt {2} \ln \left| \tan \left(\frac {x}{2} + \frac {\pi}{8}\right) \right| + C.
$$
