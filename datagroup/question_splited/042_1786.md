【1786】 $\int \sqrt{a^2 + x^2} \, \mathrm{d}x.$

提示 设 $x = \text{asht}$ , 并利用 1762 题的结果.

解 被积函数的存在域为 $-\infty < x < +\infty$ ，因此，可设 $x = a \sin t$ 。从而，

$$
\sqrt {a ^ {2} + x ^ {2}} = a c h t, \quad d x = a c h t d t.
$$

代入得 $\int \sqrt{a^2 + x^2} \, \mathrm{d}x = a^2 \int \mathrm{ch}^2 t \, \mathrm{d}t = a^2 \left( \frac{t}{2} + \frac{1}{4} \mathrm{sh} 2t \right)^{-3} + C_1.$

注意到 $x + \sqrt{a^2 + x^2} = a(\mathrm{sh}t + \mathrm{ch}t) = ae^t$ ，即 $t = \ln \frac{x + \sqrt{a^2 + x^2}}{a}$ 及 $\mathrm{sh}2t = 2\mathrm{sh}t\mathrm{ch}t = \frac{2x\sqrt{a^2 + x^2}}{a^2}$ 最后得

$$
\int \sqrt {a ^ {2} + x ^ {2}} d x = \frac {a ^ {2}}{2} \ln (x + \sqrt {a ^ {2} + x ^ {2}}) + \frac {x}{2} \sqrt {a ^ {2} + x ^ {2}} + C.
$$

\*) 利用 1762 题的结果.
