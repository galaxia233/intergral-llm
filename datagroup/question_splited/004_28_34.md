28.34 If $n$ is a positive integer, find $\int_0^{2n}x\sin nxdx$

Let $u = x, \quad dv = \sin nx \, dx, \quad du = dx, \quad v = -(1/n) \cos nx.$ Then $\int x \sin nx \, dx = -\frac{1}{n} x \cos nx + \frac{1}{n} \int \cos nx \, dx = -\frac{1}{n} x \cos nx + \frac{1}{n^2} \sin nx.$ Hence, $\int_{0}^{2\pi} x \sin nx \, dx = \frac{1}{n^2} (\sin nx - nx \cos nx) \bigg|_{0}^{2\pi} = \frac{1}{n^2} [(0 - 2\pi n)(0 - 2\pi n) + (0 - 2\pi n)(0 - 2\pi n)] = -\frac{2\pi}{n}.$
