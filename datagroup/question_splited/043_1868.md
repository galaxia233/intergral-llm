【1868】 $\int \frac{x^{10}}{x^2 + x - 2}\mathrm{d}x.$

解 $\frac{x^{10}}{x^2 + x - 2} = x^8 - x^7 + 3x^6 - 5x^5 + 11x^4 - 21x^3 + 43x^2 - 85x + 171 + \frac{-341x + 342}{x^2 + x - 2},$

设 $\frac{-341x + 342}{x^2 + x - 2} = \frac{A}{x + 2} +\frac{B}{x - 1}$ ，通分后应有 $-341x + 342\equiv A(x - 1) + B(x + 2)$ .在这恒等式中，

令 $x = -2$ ，得 $1024 = -3A$ ， $A = -\frac{1024}{3}$ ；令 $x = 1$ ，得 $1 = 3B$ ， $B = \frac{1}{3}$

于是， $\int \frac{x^{10}}{x^2 + x - 2}\mathrm{d}x = \int \left[x^8 -x^7 +3x^6 -5x^5 +11x^4 -21x^3 +43x^2 -85x + 171 - \frac{1024}{3(x + 2)} +\frac{1}{3(x - 1)}\right]\mathrm{d}x$
