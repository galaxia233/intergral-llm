20.84 If an object moves along a line with velocity $v = \sin t - \cos t$ from time $t = 0$ to $t = \pi / 2$ , find the distance traveled.

The distance is $s = \int_{0}^{\pi / 2} |v| dt$ . Now, for $0 < t < \pi / 2$ , $\sin t - \cos t \geq 0$ when and only when $\sin t \geq \cos t$ , that is, if and only if $\tan t \geq 1$ , which is equivalent to $\pi / 4 \leq t < \pi / 2$ . Hence, $s = \int_{0}^{\pi / 4} (\cos t - \sin t) dt + \int_{\pi / 4}^{\pi / 2} (\sin t - \cos t) dt = (\sin t + \cos t)_{0}^{\pi / 4} + (-\cos t - \sin t)_{\pi / 4}^{\pi / 2} = \left[ \left( \frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2} \right) - (0 + 1) \right] + \left[ (-0 - 1) - \left( -\frac{\sqrt{2}}{2} - \frac{\sqrt{2}}{2} \right) \right] = 2(\sqrt{2} - 1)$ .
