【2050】 证明：

$$
\int \frac {a _ {1} \sin^ {2} x + 2 b _ {1} \sin x \cos x + c _ {1} \cos^ {2} x}{a \sin x + b \cos x} d x = A \sin x + B \cos x + C \int \frac {d x}{a \sin x + b \cos x},
$$

式中 $A, B, C$ 为常数.

提示 首先， $a$ 及 $b$ 不可能同时为零.其次，令

$$
a _ {1} \sin^ {2} x + 2 b _ {1} \sin x \cos x + c _ {1} \cos^ {2} x \equiv A \cos x (a \sin x + b \cos x) - B \sin x (a \sin x + b \cos x) + C,
$$

比较等式两端同类项的系数，可得 $A, B, C$

$$
A = \frac {b c _ {1} - a _ {1} b + 2 a b _ {1}}{a ^ {2} + b ^ {2}}, B = \frac {a c _ {1} - a a _ {1} - 2 b b _ {1}}{a ^ {2} + b ^ {2}}, C = \frac {a _ {1} b ^ {2} + a ^ {2} c _ {1} - 2 a b b _ {1}}{a ^ {2} + b ^ {2}} (a ^ {2} + b ^ {2} \neq 0),
$$

命题即易获证.

证 按题意 $a, b$ 不同时为零. 设

$$
a _ {1} \sin^ {2} x + 2 b _ {1} \sin x \cos x + c _ {1} \cos^ {2} x \equiv A \cos x (a \sin x + b \cos x) - B \sin x (a \sin x + b \cos x) + C,
$$

比较等式两端同类项的系数，则有

$$
a A - b B = 2 b _ {1}, \quad C - a B = a _ {1}, \quad C + b A = c _ {1},
$$

从而， $A = \frac{bc_1 - a_1b + 2ab_1}{a^2 + b^2},\quad B = \frac{ac_1 - aa_1 - 2bb_1}{a^2 + b^2},C = \frac{a_1b^2 + a^2c_1 - 2abb_1}{a^2 + b^2}.$

代入得

$$
\begin{array}{l} \int \frac {a _ {1} \sin^ {2} x + 2 b _ {1} \sin x \cos x + c _ {1} \cos^ {2} x}{a \sin x + b \cos x} d x = A \int \cos x d x - B \int \sin x d x + C \int \frac {d x}{a \sin x + b \cos x} \\ = A \sin x + B \cos x + C \int \frac {d x}{a \sin x + b \cos x}. \\ \end{array}
$$
