【1975】 $\int \frac{\sqrt{x(x + 1)}}{\sqrt{x} + \sqrt{x + 1}}\mathrm{d}x.$

解 $\int \frac{\sqrt{x(x + 1)}}{\sqrt{x} + \sqrt{x + 1}}\mathrm{d}x = \int \frac{\sqrt{x(x + 1)}(\sqrt{x + 1} - \sqrt{x})}{(x + 1) - x}\mathrm{d}x = \int [(x + 1)\sqrt{x} - x\sqrt{x + 1}]\mathrm{d}x.$

$$
= \int \left[ x ^ {\frac {3}{2}} + x ^ {\frac {1}{2}} - (x + 1) ^ {\frac {3}{2}} + (x + 1) ^ {\frac {1}{2}} \right] d x = \frac {2}{3} \left[ (x + 1) ^ {\frac {3}{2}} + x ^ {\frac {3}{2}} \right] - \frac {2}{5} \left[ (x + 1) ^ {\frac {5}{2}} - x ^ {\frac {5}{2}} \right] + C.
$$
