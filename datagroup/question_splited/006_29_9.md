29.9 $\int \sec^5 x dx.$

Use Problem 28.39:

$$
\begin{array}{l} \int \sec^ {5} x d x = \frac {\tan x \sec^ {3} x}{4} + \frac {3}{4} \int \sec^ {3} x d x = \frac {\tan x \sec^ {3} x}{4} + \frac {3}{4} \left(\frac {\tan x \sec x}{2} + \frac {1}{3} \int \sec x d x\right) \\ = \frac {\tan x \sec^ {3} x}{4} + \frac {3}{8} \tan x \sec x + \frac {1}{4} \ln | \sec x + \tan x | + C \\ \end{array}
$$
