【2989】 $\sum_{n = 1}^{\infty}\frac{n}{(n + 1)(n + 2)(n + 3)}.$

提示 注意 $\frac{1}{n + 1} + \frac{1}{n + 2} - \frac{2}{n + 3} = \frac{3n + 5}{(n + 1)(n + 2)(n + 3)}$ ，并利用2987题的结果.

解 由于 $\frac{1}{n + 1} +\frac{1}{n + 2} -\frac{2}{n + 3} = \frac{3n + 5}{(n + 1)(n + 2)(n + 3)}$ 故

$$
\begin{array}{l} \sum_ {n = 1} ^ {\infty} \frac {n}{(n + 1) (n + 2) (n + 3)} = \frac {1}{3} \sum_ {n = 1} ^ {\infty} \left(\frac {1}{n + 1} + \frac {1}{n + 2} - \frac {2}{n + 3}\right) - \frac {5}{3} \sum_ {n = 1} ^ {\infty} \frac {1}{(n + 1) (n + 2) (n + 3)} \\ = \frac {1}{3} \left(\frac {1}{2} + \frac {2}{3}\right) - \frac {5}{3} \left(\frac {1}{4} - \frac {1}{1 \cdot 2 \cdot 3}\right) ^ {*}) = \frac {1}{4}. \\ \end{array}
$$

\*) 利用2987题的结果.
