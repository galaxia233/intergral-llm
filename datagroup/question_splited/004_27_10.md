27.10 $\sec^{-1}\sqrt{2}$

By definition, the value of $\sec^{-1}x$ is either an angle in the first quadrant (for positive arguments) or an angle in the third quadrant (for negative arguments). In this case, the value is in the first quadrant, so $\sec^{-1}\sqrt{2} = \cos^{-1}1 / \sqrt{2} = \pi /4$ .
