【2317】 确定哪个积分较大：
(3) $\int_{0}^{\pi} \mathrm{e}^{-x^{2}} \cos^{2} x \, \mathrm{d}x$ 或 $\int_{0}^{2\pi} \mathrm{e}^{-x^{2}} \cos^{2} x \, \mathrm{d}x.$

解 (3) $\int_{0}^{2\pi} \mathrm{e}^{-x^2} \cos^2 x \, \mathrm{d}x = \int_{0}^{\pi} \mathrm{e}^{-(\pi + x)^2} \cos^2 x \, \mathrm{d}x \leqslant \int_{0}^{\pi} \mathrm{e}^{-x^2} \cos^2 x \, \mathrm{d}x.$
