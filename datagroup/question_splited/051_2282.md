【2282】 $I_{n} = \int_{0}^{\frac{\pi}{2}}\cos^{n}x\mathrm{d}x.$

提示 令 $\frac{\pi}{2} - x = t$ ，并利用 2281 题的结果.

解设 $\frac{\pi}{2} - x = t$ ，则 $\mathrm{d}x = -\mathrm{d}t$ ，且 $\cos x = \cos \left(\frac{\pi}{2} - t\right) = \sin t.$

代入得 $I_{n} = \int_{0}^{\frac{\pi}{2}}\sin^{n}t\mathrm{d}t.$

因此，与 2281 题的结果相同
