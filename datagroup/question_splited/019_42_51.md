42.52 Verify that $f(x, y) = \ln (x^2 + y^2)$ satisfies Laplace's equation, $f_{xx} + f_{yy} = 0$ .

$f_{x} = \frac{2x}{x^{2} + y^{2}},\quad f_{xx} = \frac{(x^{2} + y^{2})(2) - 2x(2x)}{(x^{2} + y^{2})^{2}} = \frac{2(y^{2} - x^{2})}{(x^{2} + y^{2})^{2}}.\quad f_{y} = \frac{2y}{x^{2} + y^{2}},\quad f_{yy} = \frac{(x^{2} + y^{2})(2) - 2y(2y)}{(x^{2} + y^{2})^{2}} = \frac{2(x^{2} - y^{2})}{(x^{2} + y^{2})^{2}}.$ $f_{xx} + f_{yy} = 0.$
