30.2 $\int \frac{x dx}{(x + 2)(x + 3)}.$

$\frac{x}{(x + 2)(x + 3)} = \frac{A}{x + 2} + \frac{B}{x + 3}$ . Then $x = A(x + 3) + B(x + 2)$ . Let $x = -3$ . Then $-3 = -B$ , $B = 3$ . Let $x = -2$ . Then $-2 = A$ . So $\frac{x}{(x + 2)(x + 3)} = -\frac{2}{x + 2} + \frac{3}{x + 3}$ . Hence, $\int \frac{x dx}{(x + 2)(x + 3)} = -2 \ln |x + 2| + 3 \ln |x + 3| + C = \ln |(x + 3)^3 / (x + 2)^2| + C$ .
