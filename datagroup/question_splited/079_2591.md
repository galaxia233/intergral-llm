【2591】证明：若 $\lim_{n\to \infty}\frac{a_{n + 1}}{a_n} = q(a_n > 0)$ ，则 $a_{n} = o(q_{1}^{n})$ ，其中 $q_{1} > q$

证 由于 $\lim_{n\to \infty}\frac{a_{n + 1}}{a_n} = q$ ，故利用141题的结果，即得 $\lim_{n\to \infty}\sqrt[n]{a_n} = q.$

令 $\epsilon = \frac{1}{2} (q_1 - q) > 0$ ，则由上式知存在 $n_0$ ，使当 $n\geqslant n_0$ 时，有 $\left|\sqrt[n]{a_n} -q\right| <   \epsilon$ ，从而有

$$
\sqrt [ n ]{a _ {n}} <   q + \varepsilon = \lambda q _ {1} \quad (n \geqslant n _ {0}),
$$

其中 $\lambda = \frac{q_1 + q}{2q_1} < 1$ 利用 $\lambda^n = o(1)$ ，即证得 $a_{n} = \lambda^{n}q_{1}^{n} = o(q_{1}^{n})$
