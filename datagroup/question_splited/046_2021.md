【2021】 $\int \frac{\mathrm{d}x}{\cos(x + a)\cos(x + b)}.$

解 $\int \frac{\mathrm{d}x}{\cos(x + a)\cos(x + b)} = \frac{1}{\sin(a - b)}\int \frac{\sin[(x + a) - (x + b)]\mathrm{d}x}{\cos(x + a)\cos(x + b)}$

$$
= \frac {1}{\sin (a - b)} \int \left[ \frac {\sin (x + a)}{\cos (x + a)} - \frac {\sin (x + b)}{\cos (x + b)} \right] d x = \frac {1}{\sin (a - b)} \ln \left| \frac {\cos (x + b)}{\cos (x + a)} \right| + C,
$$

其中 $\sin (a - b)\neq 0^{*}$

\*) 当 $a - b = 2k\pi (k = 0, \pm 1, \pm 2, \dots)$ 时，是更简单的积分，2019题及2020题与本题类似，解法从略.
