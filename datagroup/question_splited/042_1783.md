【1783】 $\int x\sqrt{\frac{x}{2a - x}}\mathrm{d}x.$

解题思路 注意被积函数的存在域为 $0 \leqslant x < 2a$ ，设 $x = 2a\sin^2 t$ ，并限制 $0 \leqslant t < \frac{\pi}{2}$ 。利用 1749 题的结果，

解被积函数的存在域为 $0\leqslant x <   2a$ ，因此，可设 $x = 2a\sin^2 t$ ，并限制 $0\leqslant t <   \frac{\pi}{2}$ .从而，

$$
x \sqrt {\frac {x}{2 a - x}} = \frac {2 a \sin^ {3} t}{\cos t}, \quad d x = 4 a \sin t \cos t d t.
$$

代入得 $\int x\sqrt{\frac{x}{2a - x}} = 8a^2\int \sin^4 t\mathrm{d}t = 8a^2\left(\frac{3}{8} t - \frac{1}{4}\sin 2t + \frac{1}{32}\sin 4t\right)^{*}) + C.$

注意到 $\sin 2t = 2\sin t\cos t = 2\sqrt{\frac{x}{2a}}\sqrt{1 - \frac{x}{2a}} = \frac{1}{a}\sqrt{x(2a - x)}$

及 $\sin 4t = 2\sin 2t\cos 2t = 4\sin t\cos t(1 - 2\sin^2 t) = \frac{2}{a^2} (a - x)\sqrt{x(2a - x)},$

最后得 $\int x\sqrt{\frac{x}{2a - x}}\mathrm{d}x = 3a^2\arcsin \sqrt{\frac{x}{2a}} -2a^2\frac{1}{a}\sqrt{x(2a - x)} +\frac{1}{4} a^2\frac{2}{a^2} (a - x)\sqrt{x(2a - x)} +C$

$$
= 3 a ^ {2} \arcsin \sqrt {\frac {x}{2 a}} - \frac {3 a + x}{2} \sqrt {x (2 a - x)} + C.
$$

\*) 利用 1749 题的结果.
