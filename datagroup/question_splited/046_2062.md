【2062】 $\int \frac{\sin x\mathrm{d}x}{\sqrt{2 + \sin 2x}}.$

解 由于 $2 + \sin 2x = 1 + (\sin x + \cos x)^{2} = 3 - (\sin x - \cos x)^{2}$

于是，

$$
\begin{array}{l} \int \frac {\sin x d x}{\sqrt {2 + \sin 2 x}} = \int \frac {\cos x - (\cos x - \sin x)}{\sqrt {1 + (\sin x + \cos x) ^ {2}}} d x = \int \frac {\cos x}{\sqrt {3 - (\sin x - \cos x) ^ {2}}} d x - \ln (\sin x + \cos x + \sqrt {2 + \sin 2 x}) \\ = - \int \frac {\sin x \mathrm {d} x}{\sqrt {2 + \sin 2 x}} + \int \frac {\mathrm {d} (\sin x - \cos x)}{\sqrt {3 - (\sin x - \cos x) ^ {2}}} - \ln (\sin x + \cos x + \sqrt {2 + \sin 2 x}). \\ \end{array}
$$

因而，

$$
\begin{array}{l} \int \frac {\sin x \mathrm {d} x}{\sqrt {2 + \sin 2 x}} = \frac {1}{2} \int \frac {\mathrm {d} (\sin x - \cos x)}{\sqrt {3 - (\sin x - \cos x) ^ {2}}} - \frac {1}{2} \ln (\sin x + \cos x + \sqrt {2 + \sin 2 x}) \\ = \frac {1}{2} \arcsin \left(\frac {\sin x - \cos x}{\sqrt {3}}\right) - \frac {1}{2} \ln (\sin x + \cos x + \sqrt {2 + \sin 2 x}) + C. \\ \end{array}
$$
