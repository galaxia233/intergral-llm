【1940】 $\int \frac{\sqrt{x^2 + 2x + 2}}{x}\mathrm{d}x.$

提示 令 $\sqrt{x^2 + 2x + 2} = t - x$

解设 $\sqrt{x^2 + 2x + 2} = t - x$ ，则 $x = \frac{t^2 - 2}{2(t + 1)},\mathrm{d}x = \frac{t^2 + 2t + 2}{2(t + 1)^2}\mathrm{d}t,\sqrt{x^2 + 2x + 2} = \frac{t^2 + 2t + 2}{2(t + 1)}.$

代入得 $\int \frac{\sqrt{x^2 + 2x + 2}}{x}\mathrm{d}x = \frac{1}{2}\int \frac{(t^2 + 2t + 2)^2}{(t^2 - 2)(t + 1)^2}\mathrm{d}t$

$$
\begin{array}{l} = \frac {1}{2} \int \left[ 1 + \frac {2}{t + 1} - \frac {1}{(t + 1) ^ {2}} - \frac {2 \sqrt {2}}{t + \sqrt {2}} + \frac {2 \sqrt {2}}{t - \sqrt {2}} \right] d t \\ = \frac {t}{2} + \ln | t + 1 | + \frac {1}{2 (t + 1)} - \sqrt {2} \ln \left| \frac {t + \sqrt {2}}{t - \sqrt {2}} \right| + C _ {1} \\ = \sqrt {x ^ {2} + 2 x + 2} + \ln (x + 1 + \sqrt {x ^ {2} + 2 x + 2}) - \sqrt {2} \ln \left| \frac {x + 2 + \sqrt {2 (x ^ {2} + 2 x + 2)}}{x} \right| + C. \\ \end{array}
