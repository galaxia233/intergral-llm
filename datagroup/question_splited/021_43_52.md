43.52 Find the relative extrema of $f(x, y) = \frac{A}{x} + \frac{B}{y} + xy$ , where $A \neq 0$ and $B \neq 0$ .

$f_{x} = -\frac{A}{x^{2}} +y,f_{y} = -\frac{B}{y^{2}} +x.$ Set $f_{x} = f_{y} = 0$ . Then $y = A / x^2$ $x = B / y^{2}$ $yx^{2} = A$ ， $y(B / y^{2})^{2} = A$ $y^{3} = B^{2} / A$ $y = \sqrt[3]{B^2 / A}$ . Similarly, $x = \sqrt[3]{A^2 / B}$ . Thus, the only critical point is $(\sqrt[3]{A^2 / B},\sqrt[3]{B^2 / A})$ $f_{xy} = 1$ $f_{xx} = 2A / x^3,f_{yy} = 2B / y^3.$ Hence, $\Delta = 4AB / x^3 y^3 -1 = 4 - 1 > 0.$ In addition, $f_{xx} + f_{yy} = \frac{2A}{x^3} +\frac{2B}{y^3} = \frac{2B}{A}$ $+ \frac { 2 A } { B } = \frac { 2 ( A ^ { 2 } + B ^ { 2 } ) } { A B }$ Therefore, at the critical point, there is a relative maximum when $A$ and $B$ are of opposite sign and a relative minimum when $A$ and $B$ have the same sign.
