【1713】 $\int \frac{x^2 - 1}{x^4 + 1}\mathrm{d}x.$

提示 注意 $\frac{x^2 - 1}{x^4 + 1}\mathrm{d}x = \frac{1 - \frac{1}{x^2}}{x^2 + \frac{1}{x^2}}\mathrm{d}x = \frac{\mathrm{d}\left(x + \frac{1}{x}\right)}{\left(x + \frac{1}{x}\right)^2 - 2}.$

解 $\int \frac{x^2 - 1}{x^4 + 1}\mathrm{d}x = \int \frac{1 - \frac{1}{x^2}}{x^2 + \frac{1}{x^2}}\mathrm{d}x = \frac{\mathrm{d}\left(x + \frac{1}{x}\right)}{\left(x + \frac{1}{x}\right)^2 - 2} = \frac{1}{2\sqrt{2}}\ln \left(\frac{x^2 - x\sqrt{2} + 1}{x^2 + x\sqrt{2} + 1}\right) + C.$
