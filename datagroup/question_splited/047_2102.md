【2102】 $\int \ln^2 (x + \sqrt{1 + x^2})\mathrm{d}x.$

解 $\int \ln^2 (x + \sqrt{1 + x^2})\mathrm{d}x = x\ln^2 (x + \sqrt{1 + x^2}) - 2\int \frac{x}{\sqrt{1 + x^2}}\ln (x + \sqrt{1 + x^2})\mathrm{d}x$

$$
\begin{array}{l} = x \ln^ {2} (x + \sqrt {1 + x ^ {2}}) - 2 \int \ln (x + \sqrt {1 + x ^ {2}}) d (\sqrt {1 + x ^ {2}}) \\ = x \ln^ {2} (x + \sqrt {1 + x ^ {2}}) - 2 \sqrt {1 + x ^ {2}} \ln (x + \sqrt {1 + x ^ {2}}) + 2 \int d x \\ = x \ln^ {2} (x + \sqrt {1 + x ^ {2}}) - 2 \sqrt {1 + x ^ {2}} \ln (x + \sqrt {1 + x ^ {2}}) + 2 x + C. \\ \end{array}
