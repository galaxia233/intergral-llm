【1714】+ $\int \frac{x^{14} \mathrm{d}x}{(x^5 + 1)^4}$

提示 注意 $\frac{x^{14} \mathrm{d}x}{(x^5 + 1)^4} = \frac{x^{14} \mathrm{d}x}{x^{20}(1 + x^{-5})^4} = -\frac{1}{5} (1 + x^{-5})^{-4} \mathrm{d}(1 + x^{-5}).$

解 $\int \frac{x^{14} \mathrm{d}x}{(x^5 + 1)^4} = \int \frac{x^{14} \mathrm{d}x}{x^{20}(1 + x^{-5})^4} = -\frac{1}{5} \int (1 + x^{-5})^{-4} \mathrm{d}(1 + x^{-5}) = \frac{1}{15} (1 + x^{-5})^{-3} + C_1$
