【2040】 $\int \frac{\mathrm{d}x}{(\sin^2 x + 2\cos^2 x)^2}$

解 $\int \frac{\mathrm{d}x}{(\sin^2 x + 2\cos^2 x)^2} = \int \frac{\frac{1}{\cos^4 x} \mathrm{d}x}{(\tan^2 x + 2)^2} = \int \frac{\sec^2 x \mathrm{d}(\tan x)}{(\tan^2 x + 2)^2}$

$$
\begin{array}{l} = \int \frac {\tan^ {2} x}{(\tan^ {2} x + 2) ^ {2}} d (\tan x) + \int \frac {d (\tan x)}{(\tan^ {2} x + 2) ^ {2}} = \int \frac {(\tan^ {2} x + 2) - 2}{(\tan^ {2} x + 2) ^ {2}} d (\tan x) + \int \frac {d (\tan x)}{(\tan^ {2} x + 2) ^ {2}} \\ = \int \frac {d (\tan x)}{\tan^ {2} x + 2} - \int \frac {d (\tan x)}{(\tan^ {2} x + 2) ^ {2}} = \frac {1}{\sqrt {2}} \arctan \left(\frac {\tan x}{\sqrt {2}}\right) - \frac {\tan x}{4 (\tan^ {2} x + 2)} - \frac {1}{4 \sqrt {2}} \arctan \left(\frac {\tan x}{\sqrt {2}}\right) ^ {*}) + C \\ = \frac {3}{4 \sqrt {2}} \arctan \left(\frac {\tan x}{\sqrt {2}}\right) - \frac {\tan x}{4 (\tan^ {2} x + 2)} + C. \\ \end{array}
$$

\*) 利用1817题的结果.
