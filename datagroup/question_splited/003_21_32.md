21.32 Find the length of the arc of $y^3 = 8x^2$ from $x = 1$ to $x = 8$ .

$y = 2x^{2 / 3},\quad y' = \frac{4}{3} x^{-1 / 3},\quad (y')^2 = 16 / 9x^{2 / 3}.$ So, $1 + (y')^2 = (9x^{2 / 3} + 16) / 9x^{2 / 3}.$ Hence, $L = \frac{1}{3}\int_{1}^{8}\frac{\sqrt{9x^{2 / 3} + 16}}{x^{1 / 3}} dx = \frac{1}{18}\int_{1}^{8}(9x^{2 / 3} + 16)^{1 / 2}D_{x}(9x^{2 / 3} + 16)dx = \frac{1}{18}\cdot \frac{2}{3} (9x^{2 / 3} + 16)^{3 / 2}]_{1}^{8} = \frac{1}{27} (104\sqrt{13} -125).$
