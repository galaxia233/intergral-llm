【4222】 $\int_{C}y^{2}\mathrm{d}s$ ，其中 $C$ 为摆线 $x = a(t - \sin t),y = a(1 - \cos t)(0\leqslant t\leqslant 2\pi)$ 的一拱.

解 弧长的微分为 $\mathrm{ds} = \sqrt{(\mathrm{dx})^2 + (\mathrm{dy})^2} = \sqrt{a^2(1 - \cos t)^2 + a^2\sin^2t}\mathrm{dt} = 2a\sin \frac{t}{2}\mathrm{dt}.$

于是， $\int_{C}y^{2}\mathrm{d}s = 2a^{3}\int_{0}^{2\pi}\sin {\frac{t}{2}}(1 - \cos t)^{2}\mathrm{d}t = 8a^{3}\int_{0}^{2\pi}\sin^{5}\frac{t}{2}\mathrm{d}t = 32a^{3}\int_{0}^{\frac{\pi}{2}}\sin^{5}u\mathrm{d}u = \frac{256}{15} a^{3}.$
