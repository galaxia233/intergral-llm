【2182】把所给区间分为 $n$ 个相等的子区间，求下列函数 $f(x)$ 在相应区间上的下积分和 $\underline{S}_n$ 及上积分和 $\overline{S}_n$
(2） $f(x) = \sqrt{x}$ $(0\leqslant x\leqslant 1)$ 

解（2） $h = \frac{1}{n}, \quad m_i = \sqrt{\frac{i}{n}}, \quad M_i = \sqrt{\frac{i + 1}{n}} \quad (i = 0, 1, 2, \dots, n - 1).$

于是， $\underline{S_n} = \sum_{i = 0}^{n - 1}\frac{1}{n}\sqrt{\frac{i}{n}} = \frac{1}{n}\sum_{i = 0}^{n - 1}\sqrt{\frac{i}{n}};$ $\overline{S_n} = \sum_{i = 0}^{n - 1}\frac{1}{n}\sqrt{\frac{i + 1}{n}} = \frac{1}{n}\sum_{i = 1}^{n}\sqrt{\frac{i}{n}}.$
