【2955】 $f(x) = x - [x]$

提示 注意函数 $f(x)$ 的周期为1.

解因为

$$
f (x + 1) = x + 1 - [ x + 1 ] = x + 1 - [ x ] - 1 = x - [ x ] = f (x),
$$

故 $f(x)$ 是以1为周期的周期函数，而且，除 $x = k, k = 0, \pm 1, \pm 2, \dots$ 诸点外， $f(x)$ 都连续.由于

$$
\begin{array}{l} a _ {0} = \frac {1}{\frac {1}{2}} \int_ {0} ^ {1} \{x - [ x ] \} d x = 2 \int_ {0} ^ {1} x d x = 1, \\ a _ {n} = \frac {1}{\frac {1}{2}} \int_ {0} ^ {1} \{x - [ x ] \} \cos 2 n \pi x d x = 2 \int_ {0} ^ {1} x \cos 2 n \pi x d x \\ = 2 \left[ \frac {x}{2 n \pi} \sin 2 n \pi x + \frac {1}{4 (n \pi) ^ {2}} \cos 2 n \pi x \right] \Bigg | _ {0} ^ {1} = 0, \\ b _ {n} = \frac {1}{\frac {1}{2}} \int_ {0} ^ {1} \{x - [ x ] \} \sin 2 n \pi x d x = 2 \int_ {0} ^ {1} x \sin 2 n \pi x d x \\ \end{array}
$$

$$
= 2 \left[ - \frac {x}{2 n \pi} \cos 2 n \pi x + \frac {1}{4 (n \pi) ^ {2}} \sin 2 n \pi x \right] \Bigg | _ {0} ^ {1} = - \frac {1}{n \pi},
$$

故按展开定理， $f(x)$ 可展开为 $\frac{1}{2} -\frac{1}{\pi}\sum_{n = 1}^{\infty}\frac{\sin 2n\pi x}{n} = x - [x]\quad (x\neq k,k = 0,\pm 1,\pm 2,\dots).$
