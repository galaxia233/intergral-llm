46.5 Solve $\frac{dy}{dx} = ay(1 - by)$

Separate the variables. $\int \frac{dy}{y(1 - by)} = \int a dx, \int \left(\frac{1}{y} + \frac{b}{1 - by}\right) dy = \int a dx, \ln |y| - \ln |1 - by| = ax + C_1,$ $\ln \left|\frac{y}{1 - by}\right| = ax + C_1, \left|\frac{y}{1 - by}\right| = e^{ax + C_1}$ . Taking reciprocals, we get $\left|\frac{1 - by}{y}\right| = e^{-ax - C_1} = e^{-C_1} \cdot e^{-ax} = Ce^{-ax}$ . Hence, $\left|\frac{1}{y} - b\right| = Ce^{-ax}$ . If we allow $C$ to be negative (which means allowing $C_1$ to be complex), $\frac{1}{y} - b = Ce^{-ax}, \frac{1}{y} = b + Ce^{-ax}, y = \frac{1}{b + Ce^{-ax}}$ .
