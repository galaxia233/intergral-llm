【2099】 $\int x^{3}\ln^{3}x\mathrm{d}x.$

解 $\int x^{3}\ln^{3}x\mathrm{d}x = \frac{1}{4}\int \ln^{3}x\mathrm{d}(x^{4})$

$$
\begin{array}{l} = \frac {1}{4} x ^ {4} \ln^ {3} x - \frac {3}{4} \int x ^ {3} \ln^ {2} x d x = \frac {1}{4} x ^ {4} \ln^ {3} x - \frac {3}{1 6} \int \ln^ {2} x d (x ^ {4}) \\ = \frac {1}{4} x ^ {4} \ln^ {3} x - \frac {3}{1 6} x ^ {4} \ln^ {2} x + \frac {3}{8} \int x ^ {3} \ln x d x = \frac {1}{4} x ^ {4} \ln^ {3} x - \frac {3}{1 6} x ^ {4} \ln^ {2} x + \frac {3}{3 2} \int \ln x d (x ^ {4}) \\ = \frac {1}{4} x ^ {4} \ln^ {3} x - \frac {3}{1 6} x ^ {4} \ln^ {2} x + \frac {3}{3 2} x ^ {4} \ln x - \frac {3}{3 2} \int x ^ {3} d x = \frac {1}{4} x ^ {4} \left(\ln^ {3} x - \frac {3}{4} \ln^ {2} x + \frac {3}{8} \ln x - \frac {3}{3 2}\right) + C. \\ \end{array}
$$
