【1707】 $\int \frac{\mathrm{sh}x\mathrm{ch}x}{\sqrt{\mathrm{sh}^4x + \mathrm{ch}^4x}}\mathrm{d}x.$

提示 注意 $\mathrm{sh}^4 x + \mathrm{ch}^4 x = \frac{1}{2} (1 + \mathrm{ch}^2 2x)$

解 因为 $\mathrm{sh}^4 x + \mathrm{ch}^4 x = (\mathrm{sh}^2 x + \mathrm{ch}^2 x)^2 - 2\mathrm{sh}^2 x\mathrm{ch}^2 x = \mathrm{ch}^2 2x - \frac{1}{2}\mathrm{sh}^2 2x = \frac{1 + \mathrm{ch}^2 2x}{2},$

所以， $\int \frac{\mathrm{sh}x\mathrm{ch}x}{\sqrt{\mathrm{sh}^4x + \mathrm{ch}^4x}}\mathrm{d}x = \int \frac{\frac{1}{4}\mathrm{d}(\mathrm{ch}2x)}{\frac{1}{\sqrt{2}}\sqrt{1 + \mathrm{ch}^22x}} = \frac{1}{2\sqrt{2}}\ln (\mathrm{ch}2x + \sqrt{1 + \mathrm{ch}^22x}) + C_1$

$$
= \frac {1}{2 \sqrt {2}} \ln \left(\frac {\operatorname {c h} 2 x}{\sqrt {2}} + \sqrt {\operatorname {s h} ^ {4} x + \operatorname {c h} ^ {4} x}\right) + C.
$$
