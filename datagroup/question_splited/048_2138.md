【2138】 $\int \frac{(1 + x)\mathrm{d}x}{x + \sqrt{x + x^2}}.$

解 $\int \frac{(1 + x)\mathrm{d}x}{x + \sqrt{x + x^2}} = \int \frac{(1 + x)(x - \sqrt{x + x^2})}{(x + \sqrt{x + x^2})(x - \sqrt{x + x^2})}\mathrm{d}x$

$$
\begin{array}{l} = \int \frac {x + x ^ {2} - \sqrt {x + x ^ {2}} - x \sqrt {x + x ^ {2}}}{- x} d x = - x - \frac {1}{2} x ^ {2} + \int \frac {\sqrt {1 + x}}{\sqrt {x}} d x + \int \sqrt {x + x ^ {2}} d x \\ = - x - \frac {1}{2} x ^ {2} + 2 \int \sqrt {1 + (\sqrt {x}) ^ {2}} d (\sqrt {x}) + \int \sqrt {\left(x + \frac {1}{2}\right) ^ {2} - \left(\frac {1}{2}\right) ^ {2}} d \left(x + \frac {1}{2}\right) \\ = - x - \frac {1}{2} x ^ {2} + \sqrt {x} \sqrt {1 + x} + \ln (\sqrt {x} + \sqrt {1 + x}) + \frac {2 x + 1}{4} \sqrt {x + x ^ {2}} - \frac {1}{8} \ln \left(x + \frac {1}{2} + \sqrt {x + x ^ {2}}\right) + C _ {1} \\ = - x - \frac {1}{2} x ^ {2} + \frac {5 + 2 x}{4} \sqrt {x + x ^ {2}} + \frac {1}{2} \ln (2 x + 1 + 2 \sqrt {x + x ^ {2}}) - \frac {1}{8} \ln \left(x + \frac {1}{2} + \sqrt {x + x ^ {2}}\right) + C _ {1} \\ \end{array}
$$

$$
= - \frac {1}{2} (x + 1) ^ {2} + \frac {5 + 2 x}{4} \sqrt {x + x ^ {2}} + \frac {3}{8} \ln \left(x + \frac {1}{2} + \sqrt {x + x ^ {2}}\right) + C,
$$

其中设 $x > 0$ ，对于 $x < -1$ ，同样可获得上述结果，但要注意在对数中要加绝对值.
