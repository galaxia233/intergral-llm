【1806】 $\int \frac{\arcsin x}{x^2} \mathrm{d}x.$

提示 使用分部积分法后，令 $x = \sin t$ ，并利用 1703 题的结果.

解 $\int \frac{\arcsin x}{x^2} \mathrm{d}x = -\int \arcsin x \mathrm{d}\left(\frac{1}{x}\right) = -\frac{1}{x} \arcsin x + \int \frac{\mathrm{d}x}{x \sqrt{1 - x^2}}.$

对最后一个积分作代换 $x = \sin t$ ，得

$$
\begin{array}{l} \int \frac {\mathrm {d} x}{x \sqrt {1 - x ^ {2}}} = \int \frac {\cos t \mathrm {d} t}{\sin t \cos t} = \int \frac {\mathrm {d} t}{\sin t} = \ln \left| \tan \frac {t}{2} \right| ^ {*}) + C = \ln \left| \frac {\sin t}{1 + \cos t} \right| + C = - \ln \left| \frac {1 + \cos t}{\sin t} \right| + C \\ = - \ln \left| \frac {1 + \sqrt {1 - x ^ {2}}}{x} \right| + C, \\ \end{array}
$$

最后得 $\int \frac{\arcsin x}{x^2} \mathrm{d}x = -\frac{1}{x} \arcsin x - \ln \left| \frac{1 + \sqrt{1 - x^2}}{x} \right| + C.$

\*) 利用 1703 题的结果.
