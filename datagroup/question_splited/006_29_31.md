29.31 Find the arc length of the curve $y = \ln x$ from (1,0) to $(e,1)$ .

$y^\prime = \frac {1}{x},\quad 1 + (y^\prime)^2 = 1 + \frac {1}{x^2} = \frac {x^2 + 1}{x^2}.$ Then, using Problem 29.21, $L = \int_{1}^{e}\sqrt{1 + (y')^2} dx = \int_{1}^{e}\frac{\sqrt{x^2 + 1}}{x} dx$ $= \left(\ln \left|\frac{\sqrt{1 + x^2} - 1}{x}\right| + \sqrt{1 + x^2}\right)\Bigg|_1^e = \left(\ln \left|\frac{\sqrt{1 + e^2} - 1}{e}\right| + \sqrt{1 + e^2}\right) - [\ln (\sqrt{2} -1) + \sqrt{2}] = \sqrt{e^2 + 1} -(1 + \sqrt{2})+$ $\ln \frac{\sqrt{e^2 + 1} - 1}{\sqrt{2} - 1}.$
