【2276】 $\int_{0}^{2\pi}\frac{\mathrm{d}x}{\sin^4x + \cos^4x}.$

提示 注意 $\int_{0}^{2\pi}\frac{\mathrm{d}x}{\sin^4x + \cos^4x} = 8\int_{0}^{\frac{\pi}{4}}\frac{\mathrm{d}x}{\sin^4x + \cos^4x}$ 并利用 2035 题的结果.

解 $\int_{0}^{2\pi}\frac{\mathrm{d}x}{\sin^4x + \cos^4x} = 8\int_{0}^{\frac{\pi}{4}}\frac{\mathrm{d}x}{\sin^4x + \cos^4x} = \frac{1}{\sqrt{2}}\arctan \left(\frac{\tan 2x}{\sqrt{2}}\right)^{*})\Bigg|_{0}^{2\pi} = 2\pi \sqrt{2}.$

\*) 利用 2035 题的结果.
