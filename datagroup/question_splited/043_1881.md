【1881】 $\int \frac{\mathrm{d}x}{x^3 + 1}$

解设 $\frac{1}{x^3 + 1} = \frac{A}{x + 1} +\frac{Bx + C}{x^2 - x + 1}$ ，通分后应有 $1\equiv A(x^{2} - x + 1) + (Bx + C)(x + 1)$ .比较等式两端 $\pmb{x}$ 的同次幂系数，得

$$
\begin{array}{l l} x ^ {2} & A + B = 0, \\ x ^ {1} & - A + B + C = 0, \\ x ^ {0} & A + C = 1. \end{array}
$$

由此， $A = \frac{1}{3},B = -\frac{1}{3},C = \frac{2}{3}.$ 于是，

$$
\begin{array}{l} \int \frac {\mathrm {d} x}{x ^ {3} + 1} = \int \left[ \frac {1}{3 (x + 1)} - \frac {x - 2}{3 \left(x ^ {2} - x + 1\right)} \right] \mathrm {d} x = \frac {1}{3} \int \frac {\mathrm {d} x}{x + 1} - \frac {1}{6} \int \frac {2 x - 1}{x ^ {2} - x + 1} \mathrm {d} x + \frac {1}{2} \int \frac {\mathrm {d} \left(x - \frac {1}{2}\right)}{\left(x - \frac {1}{2}\right) ^ {2} + \frac {3}{4}} \\ = \frac {1}{6} \ln \frac {(x + 1) ^ {2}}{x ^ {2} - x + 1} + \frac {1}{\sqrt {3}} \arctan \frac {2 x - 1}{\sqrt {3}} + C. \\ \end{array}
$$
