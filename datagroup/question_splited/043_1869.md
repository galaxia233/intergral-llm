【1869】 $\int \frac{x^3 + 1}{x^3 - 5x^2 + 6x}\mathrm{d}x.$

解 $\frac{x^3 + 1}{x^3 - 5x^2 + 6x} = 1 + \frac{5x^2 - 6x + 1}{x^3 - 5x^2 + 6x} = 1 + \frac{5x^2 - 6x + 1}{x(x - 2)(x - 3)}$ ,

设 $\frac{5x^2 - 6x + 1}{x(x - 2)(x - 3)} = \frac{A}{x} +\frac{B}{x - 2} +\frac{C}{x - 3}$ ，通分后应有 $5x^{2} - 6x + 1\equiv A(x - 2)(x - 3) + Bx(x - 3) + Cx(x$ -2).在这恒等式中，
