20.32 Find the average value of $f(x) = \sqrt[3]{x}$ on [0, 1].

By definition, the average value of a function $f(x)$ on an interval $[a, b]$ is $\frac{1}{b - a} \int_{a}^{b} f(x) \, dx$ . Hence, we must compute $\int_{0}^{1} \sqrt[3]{x} \, dx = \int_{0}^{1} x^{1/3} \, dx = \left( \frac {3}{4} x ^ {4 / 3} \right) _ {0} ^ {1} = \left( 1 - 0 \right) = \frac {3}{4}$ .
