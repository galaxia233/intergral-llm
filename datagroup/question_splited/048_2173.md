【2173】 $\int [x]|\sin \pi x|\mathrm{d}x\quad (x\geqslant 0).$

解分别求出在区间[0,1), [1,2), [2,3), ..., [[x], x]上满足 $F(0) = 0$ 的原函数 $F(x)$ 的增量如下：

在[0,1)上， $\int 0\cdot \sin \pi x\mathrm{d}x = C_1$ ， $F(1) - F(0) = 0$

在[1,2)上， $-\int \sin \pi x\mathrm{d}x = \frac{1}{\pi}\cos \pi x + C_2$ ， $F(2) - F(1) = \frac{2}{\pi};$

在[2,3)上， $2\int \sin \pi x\mathrm{d}x = -\frac{2}{\pi}\cos \pi x + C_3,F(3) - F(2) = \frac{2\cdot 2}{\pi};$

在 $[ [x], x ]$ 上， $(-1)^{[x]}[x] \int \sin \pi x \mathrm{d} x = (-1)^{[x]}[x] \left(-\frac{1}{\pi}\right) \cos \pi x + C_{[x] + 1},$

$$
F (x) - F ([ x ]) = \frac {(- 1) ^ {[ x ]} [ x ]}{\pi} (\cos \pi [ x ] - \cos \pi x).
$$

从而，对于 $x \geqslant 0$ ，得到

$$
\begin{array}{l} \int [ x ] | \sin \pi x | d x = F (x) + C \\ = [ F (1) - F (0) ] + [ F (2) - F (1) ] + [ F (3) - F (2) ] + \dots + \frac {(- 1) ^ {[ x ]} [ x ]}{\pi} (\cos \pi [ x ] - \cos \pi x) + C \\ = \frac {2}{\pi} + \frac {2 \cdot 2}{\pi} + \dots + \frac {2 ([ x ] - 1)}{\pi} + \frac {(- 1) ^ {[ x ]} [ x ]}{\pi} (\cos \pi [ x ] - \cos \pi x) + C \\ = \frac {[ x ] ([ x ] - 1)}{\pi} + \frac {(- 1) ^ {[ x ]} [ x ] (- 1) ^ {[ x ]}}{\pi} - \frac {(- 1) ^ {[ x ]} [ x ] \cos \pi x}{\pi} + C \\ = \frac {[ x ]}{\pi} ([ x ] - (- 1) ^ {[ x ]} \cos \pi x) + C. \\ \end{array}
$$
