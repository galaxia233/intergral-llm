【4317】计算由曲线 $\left(\frac{x}{a}\right)^{2n+1} + \left(\frac{y}{b}\right)^{2n+1} = c\left(\frac{x}{a}\right)^n\left(\frac{y}{b}\right)^n (a > 0, b > 0, c > 0, n > 0)$ 所围的面积.

提示 仿4316题.

解 作代换 $y = \frac{b}{a} x t$ ，即得曲线的参数方程为

$$
x = \frac {a c t ^ {n}}{1 + t ^ {2 n + 1}}, \quad y = \frac {b c t ^ {n + 1}}{1 + t ^ {2 n + 1}} \quad (0 \leqslant t <   + \infty).
$$

易知 $x\mathrm{d}y - y\mathrm{d}x = \frac{abc^2t^{2n}}{(1 + t^{2n - 1})^2}\mathrm{d}t.$ 于是，面积为

$$
S = \frac {1}{2} \oint_ {c} x d y - y d x = \frac {a b c ^ {2}}{2} \int_ {0} ^ {+ \infty} \frac {t ^ {2 n}}{(1 + t ^ {2 n + 1}) ^ {2}} d t = - \frac {a b c ^ {2}}{2 (2 n + 1)} \cdot \frac {1}{1 + t ^ {2 n - 1}} \Bigg | _ {0} ^ {- \infty} = \frac {a b c ^ {2}}{2 (2 n + 1)}.
$$
