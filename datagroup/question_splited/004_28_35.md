28.35 Find a reduction formula for $\int \cos^n x dx$ for $n\geq 2$

Let $u = \cos^{n - 1}x, \quad du = \cos x dx, \quad du = -(n - 1)\cos^{n - 2}x\sin x dx, \quad v = \sin x.$ Then $\int \cos^n x dx = \sin x\cos^{n - 1}x + (n - 1)\int \cos^{n - 2}x\sin^2 x dx = \sin x\cos^{n - 1}x + (n - 1)\int \cos^{n - 2}x(1 - \cos^2 x)dx = \sin x\cos^{n - 1}x + (n - 1)\int \cos^{n - 2}x dx - (n - 1)\int \cos^n x dx.$ Solving for $\int \cos^n x dx$

$$
\int \cos^ {n} x d x = \frac {\sin x \cos^ {n - 1} x}{n} + \frac {n - 1}{n} \int \cos^ {n - 2} x d x.
