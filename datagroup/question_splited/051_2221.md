【2221】 $\lim_{n\to \infty}\left(\frac{n}{n^2 + 1^2} +\frac{n}{n^2 + 2^2} +\dots +\frac{n}{n^2 + n^2}\right).$

解 $\lim_{n\to \infty}\sum_{i = 1}^{n}\frac{n}{n^2 + i^2} = \lim_{n\to \infty}\sum_{i = 1}^{n}\left(\frac{1}{1 + \left(\frac{i}{n}\right)^2}\cdot \frac{1}{n}\right) = \int_0^1\frac{1}{1 + x^2}\mathrm{d}x = \frac{\pi}{4}.$
