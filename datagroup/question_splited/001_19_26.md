19.26 Evaluate $\int \frac{x}{\sqrt{x + 1}} dx$

Let $u = x + 1, \quad du = dx$ . Then $\int \frac{x}{\sqrt{x + 1}} dx = \int \frac{u - 1}{\sqrt{u}} du = \int (u^{1/2} - u^{-1/2}) du = \frac {2}{3} u^{3 / 2} - 2 u^{1 / 2} + C = \frac {2}{3} (\sqrt{u})^3 - 2 \sqrt{u} + C = \frac {2}{3} (\sqrt{x + 1})^3 - 2 \sqrt{x + 1} + C = \frac {2}{3} \sqrt{x + 1}(x - 2) + C.$
