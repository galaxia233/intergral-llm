【2155】 $\int \frac{x^4\arctan x}{1 + x^2}\mathrm{d}x.$

解 $\int \frac{x^4\arctan x}{1 + x^2}\mathrm{d}x = \int \left(x^2 -1 + \frac{1}{x^2 + 1}\right)\arctan x\mathrm{d}x.$

$$
\begin{array}{l} = \frac {1}{3} \int \arctan x d \left(x ^ {3}\right) - \int \arctan x d x + \int \arctan x d (\arctan x) \\ = \frac {1}{3} x ^ {3} \arctan x - \frac {1}{3} \int \frac {x ^ {3} d x}{1 + x ^ {2}} - x \arctan x + \int \frac {x d x}{1 + x ^ {2}} + \frac {1}{2} (\arctan x) ^ {2} \\ = \frac {1}{3} x ^ {3} \arctan x - \frac {1}{3} \int \left(x - \frac {x}{1 + x ^ {2}}\right) d x - x \arctan x + \frac {1}{2} \ln (1 + x ^ {2}) + \frac {1}{2} (\arctan x) ^ {2} \\ = \frac {1}{3} x ^ {3} \arctan x - \frac {1}{6} x ^ {2} + \frac {1}{6} \ln (1 + x ^ {2}) - x \arctan x + \frac {1}{2} \ln (1 + x ^ {2}) + \frac {1}{2} (\arctan x) ^ {2} + C \\ \end{array}
$$

$$
= - \frac {1}{6} x ^ {2} - \left(x - \frac {x ^ {3}}{3}\right) \arctan x + \frac {1}{2} (\arctan x) ^ {2} + \frac {2}{3} \ln (1 + x ^ {2}) + C.
$$
