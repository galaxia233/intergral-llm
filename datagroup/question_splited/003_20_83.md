20.83 Evaluate $\int_0^{\pi /12}\cos 5x\cdot \cos xdx.$

By the addition formula,

$$
\cos (n - 1) x = \cos (n x - x) = \cos n x \cos x + \sin n x \sin x
$$

$$
\cos (n + 1) x = \cos (n x + x) = \cos n x \cos x - \sin n x \sin x
$$

whence $\cos (n - 1)x + \cos (n + 1)x = 2\cos nx\cos x.$ In particular, $\cos 5x\cdot \cos x = \frac{1}{2} (\cos 6x + \cos 4x).$ Hence, $\int_0^{\pi /12}\cos 5x\cdot \cos xdx = \frac{1}{2}\int_0^{\pi /12}(\cos 6x + \cos 4x)dx = \frac{1}{2} (\frac{1}{6}\sin 6x + \frac{1}{4}\sin 4x)\big]_0^{\pi /12} = \frac{1}{2} (\frac{1}{6} +\frac{1}{4}\cdot \sqrt{3} /2) =$ $\frac{1}{48}(4 + 3\sqrt{3})$
