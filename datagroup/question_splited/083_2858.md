【2858】 $\frac{x}{1 + x - 2x^2}$

提示 将所给函数分解为部分分式.

解 $\frac{x}{1 + x - 2x^2} = \frac{1}{3}\left(\frac{1}{1 - x} -\frac{1}{1 + 2x}\right) = \frac{1}{3}\left[\sum_{n = 0}^{\infty}x^n -\sum_{n = 0}^{\infty}(-2)^n x^n\right] = \frac{1}{3}\sum_{n = 0}^{\infty}[1 - (-2)^n ]x^n$

$$
(| x | <   \frac {1}{2}).
$$
