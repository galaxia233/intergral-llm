46.2 Solve $\frac{dy}{dx} = \frac{5y}{7x}$

The variables are separable. $7\int \frac{1}{y} dy = 5\int \frac{1}{x} dx, 7\ln |y| = 5\ln |x| + C_1, \ln |y| = \frac{5}{7}\ln |x| + C_1, |y| = e^{5(\ln |x| + C_1) / 7} = e^{C_1} \cdot e^{5(\ln |x|) / 7}, |y| = C_2|x|^{5 / 7}$ , where $C_2 = e^{C_1} > 0$ . Note that any function of the form $y = Cx^{5 / 7}$ satisfies the given equation, where $C$ is an arbitrary constant (not necessarily positive).
