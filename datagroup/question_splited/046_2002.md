【2002】 $\int \frac{\mathrm{d}x}{\sin^3 x \cos^5 x}$ .

提示：多次利用等式 $1 = \sin^2 x + \cos^2 x$

解 $\int \frac{\mathrm{d}x}{\sin^3 x \cos^5 x} = \int \frac{\sin^2 x + \cos^2 x}{\sin^3 x \cos^5 x} \mathrm{d}x = \int \frac{\mathrm{d}x}{\sin x \cos^5 x} + \int \frac{\mathrm{d}x}{\sin^3 x \cos^3 x}$

$$
\begin{array}{l} = \int \frac {\sin^ {2} x + \cos^ {2} x}{\sin x \cos^ {5} x} d x + \int \frac {\sin^ {2} x + \cos^ {2} x}{\sin^ {3} x \cos^ {3} x} d x = \int \frac {\sin x}{\cos^ {5} x} d x + 2 \int \frac {d x}{\sin x \cos^ {3} x} + \int \frac {d x}{\sin^ {3} x \cos x} \\ = - \int \frac {\mathrm {d} (\cos x)}{\cos^ {5} x} + 2 \int \frac {\sin x}{\cos^ {3} x} \mathrm {d} x + 3 \int \frac {\mathrm {d} x}{\sin x \cos x} + \int \frac {\cos x}{\sin^ {3} x} \mathrm {d} x \\ = \frac {1}{4 \cos^ {4} x} - 2 \int \frac {\mathrm {d} (\cos x)}{\cos^ {3} x} + 3 \int \frac {\mathrm {d} (\tan x)}{\tan x} + \int \frac {\mathrm {d} (\sin x)}{\sin^ {3} x} = \frac {1}{4 \cos^ {4} x} + \frac {1}{\cos^ {2} x} + 3 \ln | \tan x | - \frac {1}{2 \sin^ {2} x} + C _ {1} \\ = \frac {1}{4} \tan^ {4} x + \frac {3}{2} \tan^ {2} x - \frac {1}{2} \cot^ {2} x + 3 \ln | \tan x | + C. \\ \end{array}
$$
