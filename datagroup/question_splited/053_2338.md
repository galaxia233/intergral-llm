【2338】 $\int_{2}^{+\infty}\frac{\mathrm{d}x}{x^2 + x - 2}.$

解 由于 $\lim_{b\to +\infty}\int_{2}^{b}\frac{\mathrm{d}x}{x^2 + x - 2} = \lim_{b\to +\infty}\left(\frac{1}{3}\ln \frac{x - 1}{x + 2}\right)\Big|_{2}^{b} = \frac{1}{3}\lim_{b\to +\infty}\left(\ln \frac{b - 1}{b + 2} +2\ln 2\right) = \frac{2}{3}\ln 2,$ 所以， $\int_{2}^{+\infty}\frac{\mathrm{d}x}{x^2 + x - 2} = \frac{2}{3}\ln 2.$
