【2041】把分母化为对数的形式，求积分 $\int \frac{\mathrm{d}x}{a\sin x + b\cos x}$

解 $\int \frac{\mathrm{d}x}{a\sin x + b\cos x} = \frac{1}{\sqrt{a^2 + b^2}}\int \frac{\mathrm{d}x}{\sin(x + \varphi)} = \frac{1}{\sqrt{a^2 + b^2}}\ln \left|\tan \left(\frac{x + \varphi}{2}\right)\right| + C,$

式中 $\cos \varphi = \frac{a}{\sqrt{a^2 + b^2}}, \sin \varphi = \frac{b}{\sqrt{a^2 + b^2}}, a^2 + b^2 \neq 0.$
