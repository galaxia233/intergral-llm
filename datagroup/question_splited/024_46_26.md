46.26 Determine whether the equation $(\cos y + y\cos x)dx + (\sin x - x\sin y)dy = 0$ is exact. If it is, solve the equation.

$\frac{\partial}{\partial y} (\cos y + y \cos x) = -\sin y + \cos x.$ $\frac{\partial}{\partial x} (\sin x - x \sin y) = \cos x - \sin y.$ Hence, the equation is exact. As in Problem 46.25, let $g(y) = \int \left\{\sin x - x \sin y - \frac{\partial}{\partial y} [\int (\cos y + y \cos x) dx]\right\} dy = \int \left[\sin x - x \sin y - \frac{\partial}{\partial y} (x \cos y + y \sin x)\right] dy = \int [\sin x - x \sin y - (-x \sin y + \sin x)] dy = \int 0 dy = 0.$ Set $f(x, y) = \int (\cos y + y \cos x) dx + 0 = x \cos y + y \sin x.$ Hence, the solution is $x \cos y + y \sin x = C$
