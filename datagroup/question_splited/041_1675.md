【1675】 $\int x^{2}\sqrt[3]{1 + x^{3}}\mathrm{d}x.$   
提示 注意 $x^{2}\sqrt[3]{1 + x^{3}}\mathrm{d}x = \frac{1}{3} (1 + x^{3})^{\frac{1}{3}}\mathrm{d}(1 + x^{3}).$   
解 $\int x^{2}\sqrt[3]{1 + x^{3}}\mathrm{d}x = \frac{1}{3}\int (1 + x^{3})^{\frac{1}{3}}\mathrm{d}(1 + x^{3}) = \frac{1}{4} (1 + x^{3})^{\frac{4}{3}} + C.$
