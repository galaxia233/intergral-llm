【2744】(1) 应当取级数 $\sum_{n=1}^{\infty} \frac{\sin nx}{n(n+1)}$ 的多少项方可使部分和 $S_n(x)$ 当 $-\infty < x < +\infty$ 时与级数的和之差小于 $\varepsilon$ ？对于下列 $\varepsilon$ 值，给出具体的计算结果：(1) $\epsilon = 0.1$

解 易证此级数收敛，记其和为

$$
S (x) = \sum_ {n = 1} ^ {\infty} \frac {\sin n x}{n (n + 1)}.
$$

如果取 $n$ 值，其部分和为 $S_{n}(x) = \sum_{k = 1}^{n}\frac{\sin kx}{k(k + 1)}.$ 要使其误差 $\Delta_n(x) = |S(x) - S_n(x)|$ 小于 $\varepsilon$ 问项数 $n$ 为若干？可用下列估计法：

$$
\begin{array}{l} \Delta_ {n} (x) = \left| \sum_ {k = n + 1} ^ {\infty} \frac {\sin k x}{k (k + 1)} \right| = \left| \lim  _ {N \rightarrow \infty} \sum_ {k = n + 1} ^ {N} \frac {\sin k x}{k (k + 1)} \right| \leqslant \lim  _ {N \rightarrow \infty} \sum_ {k = n + 1} ^ {N} \frac {| \sin k x |}{k (k + 1)} \leqslant \lim  _ {N \rightarrow \infty} \sum_ {k = n + 1} ^ {N} \frac {1}{k (k + 1)} \\ = \lim  _ {N \rightarrow \infty} \sum_ {k = n + 1} ^ {N} \left(\frac {1}{k} - \frac {1}{k + 1}\right) = \lim  _ {N \rightarrow \infty} \left(\frac {1}{n + 1} - \frac {1}{N + 1}\right) = \frac {1}{n + 1}. \\ \end{array}
$$

若令 $\frac{1}{n + 1} < \varepsilon$ ，也即当 $n > \frac{1}{\varepsilon} - 1$ 时就有 $\Delta_n(x) < \varepsilon$ 。记 $N_0 = \left[\frac{1}{\varepsilon}\right]$ ，则当

$$
n > \left[ \frac {1}{\varepsilon} \right] + \left\{\frac {1}{\varepsilon} \right\} - 1 = N _ {0} - (1 - \left\{\frac {1}{\varepsilon} \right\})
$$

时，即有 $\Delta_{n}(x) < \varepsilon$ ，其中 $\left\{\frac{1}{\varepsilon}\right\}$ 表示 $\frac{1}{\varepsilon}$ 的零头部分。也可取

$$
n = N _ {0}, N _ {0} + 1, N _ {0} + 2, \dots
$$

均有 $\Delta_{n}(x) < \epsilon$ 。所取的项数 $N_{0}$ 与 $\varepsilon$ 的关系，按题设数值，可有
