31.4 The same arc as in Problem 31.3, but about the $y$ -axis.

Use $S = 2\pi \int_{0}^{1/4} x \sqrt{1 + \left(\frac{dx}{dy}\right)^2} dy$ . Since $x = \sqrt{y}$ , $\frac{dx}{dy} = \frac{1}{2\sqrt{y}}$ , $1 + \left(\frac{dx}{dy}\right)^2 = 1 + \frac{1}{4y} = \frac{4y + 1}{4y}$ . So $S = 2\pi \int_{0}^{1/4} \sqrt{y} \frac{\sqrt{4y + 1}}{2\sqrt{y}} dy = \pi \int_{0}^{1/4} \sqrt{4y + 1} dy = \pi \frac{1}{4} \left(\frac{2}{3}\right) (4y + 1)^{3/2}]_0^{1/4} = \frac{\pi}{6} (2^{3/2} - 1) = \frac{\pi}{6} (2\sqrt{2} - 1)$ .
