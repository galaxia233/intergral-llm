【4370】$\oint_{C} (y + z) \mathrm{d}x + (z + x) \mathrm{d}y + (x + y) \mathrm{d}z$ ，式中 $C$ 为椭圆周 $x = a\sin^2 t, y = 2a\sin t \cos t, z = a\cos^2 t$ $(0 \leqslant t \leqslant \pi)$ ，并且以参数 $t$ 增大的方向为积分时的正方向。

解 $\oint_{c} (y + z) \mathrm{d}x + (z + x) \mathrm{d}y + (x + y) \mathrm{d}z = \iint_{S} 0 \mathrm{~d}y \mathrm{~d}z + 0 \mathrm{~d}z \mathrm{~d}x + 0 \mathrm{~d}x \mathrm{~d}y = 0.$
