【1784】 $\int \frac{\mathrm{d}x}{\sqrt{(x - a)(b - x)}}.$

解题思路 不妨设 $a < b$ . 注意被积函数的存在域为 $a < x < b$ , 设 $x - a = (b - a)\sin^2 t$ , 并限制 $0 < t < \frac{\pi}{2}$ .

解不妨设 $a < b$ 被积函数的存在域为 $a < x < b$ ，因此，可设 $x - a = (b - a)\sin^2 t$ ，并限制 $0 < t < \frac{\pi}{2}$ 从而，

$$
\sqrt {(x - a) (b - x)} = (b - a) \sin t \cos t, \quad d x = 2 (b - a) \sin t \cos t d t.
$$

代入得 $\int \frac{\mathrm{d}x}{\sqrt{(x - a)(b - x)}} = 2\int \mathrm{d}t = 2t + C = 2\arcsin \sqrt{\frac{x - a}{b - a}} +C.$
