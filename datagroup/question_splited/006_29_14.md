29.14 $\int \sin \pi x\cos 3\pi xdx.$

Use the formula $\sin Ax\cos Bx = \frac{1}{2} [\sin (A + B)x + \sin (A - B)x]$ . Then $\int \sin \pi x\cos 3\pi x dx = \frac{1}{2}\int [\sin 4\pi x + \sin (-2\pi x)]dx = \frac{1}{2}\int (\sin 4\pi x - \sin 2\pi x)dx = \frac{1}{2}\left(-\frac{\cos 4\pi x}{4\pi} +\frac{\cos 2\pi x}{2\pi}\right) + C = \frac{1}{8\pi} (2\cos 2\pi x -$ $\cos 4\pi x) + C.$
