【2552】 $\sum_{n = 1}^{\infty}(\sqrt{n + 2} -2\sqrt{n + 1} +\sqrt{n}).$

提示 $S_{n} = 1 - \sqrt{2} +\frac{1}{\sqrt{n + 2} + \sqrt{n + 1}}.$

解 由于

$$
\begin{array}{l} S _ {n} = (\sqrt {3} - 2 \sqrt {2} + 1) + (\sqrt {4} - 2 \sqrt {3} + \sqrt {2}) + (\sqrt {5} - 2 \sqrt {4} + \sqrt {3}) + (\sqrt {6} - 2 \sqrt {5} + \sqrt {4}) \\ + \dots + (\sqrt {n + 2} - 2 \sqrt {n + 1} + \sqrt {n}) \\ = 1 - \sqrt {2} + \sqrt {n + 2} - \sqrt {n + 1} = 1 - \sqrt {2} + \frac {1}{\sqrt {n + 2} + \sqrt {n + 1}}, \\ \end{array}
$$

故得 $S = \lim_{n\to \infty}S_n = 1 - \sqrt{2}$
