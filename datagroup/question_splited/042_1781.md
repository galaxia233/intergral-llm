【1781】 $\int \frac{\mathrm{d}x}{(x^2 + a^2)^{\frac{3}{2}}}$

解 被积函数的存在域为 $-\infty < x < +\infty$ ，因此，可设 $x = a\tan t$ ，并限制 $-\frac{\pi}{2} < t < \frac{\pi}{2}$ 。从而，

$$
\left(x ^ {2} + a ^ {2}\right) ^ {\frac {3}{2}} = a ^ {3} \sec^ {3} t, \quad \mathrm {d} x = a \sec^ {2} t d t.
$$

代人得 $\int \frac{\mathrm{d}x}{(x^2 + a^2)^{\frac{3}{2}}} = \frac{1}{a^2}\int \cos t\mathrm{d}t = \frac{1}{a^2}\sin t + C = \frac{1}{a^2}\cdot \frac{\tan t}{\sqrt{1 + \tan^2t}} +C = \frac{x}{a^2\sqrt{a^2 + x^2}} +C.$
