44.39 (a) Evaluate the integral $I = \int_{0}^{3/2} \int_{\sqrt{3}x}^{\sqrt{9 - x^2}} 2xy \, dy \, dx$ using (a) rectangular coordinates

$\begin{array}{r}I(a)I = \int_0^{3 / 2}xy^2\big]\sqrt[9]{3} x^2 dx = \int_0^{3 / 2}\left[x(9 - x^2) - 3x^3\right]dx = \int_0^{3 / 2}\left(9x - 4x^3\right)dx = \left(\frac{9}{2} x^2 -x^4\right)\big]_0^{3 / 2} = \frac{9}{4}\left(\frac{9}{2} -\frac{9}{4}\right) = \\ \binom {9}{4}^2 = \frac{81}{16}. \end{array}$
