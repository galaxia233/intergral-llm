【2156】 $\int \frac{x\mathrm{arccot}x}{(1 + x^2)^2}\mathrm{d}x.$

解 $\int \frac{x\operatorname{arccot}x}{(1 + x^2)^2}\mathrm{d}x = -\frac{1}{2}\int \operatorname {arccot}x\mathrm{d}\left(\frac{1}{1 + x^2}\right) = -\frac{\operatorname{arccot}x}{2(1 + x^2)} -\frac{1}{2}\int \frac{\mathrm{d}x}{(1 + x^2)^2}$

$$
= - \frac {\operatorname {a r c c o t} x}{2 \left(1 + x ^ {2}\right)} - \frac {1}{2} \left[ \frac {x}{2 \left(x ^ {2} + 1\right)} - \frac {1}{2} \operatorname {a r c c o t} x \right] ^ {*}) + C = - \frac {1 - x ^ {2}}{4 \left(1 + x ^ {2}\right)} \operatorname {a r c c o t} x - \frac {x}{4 \left(1 + x ^ {2}\right)} + C.
$$

\*) 利用 1921 题的递推公式.
