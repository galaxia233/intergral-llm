【2135】 $\int \frac{\mathrm{d}x}{x\sqrt{1 + x^3 + x^6}}.$

解 $\int \frac{\mathrm{d}x}{x\sqrt{1 + x^3 + x^6}} = \int \frac{\mathrm{d}x}{x^4\sqrt{x^{-6} + x^{-3} + 1}} = -\frac{1}{3}\int \frac{\mathrm{d}\left(x^{-3} + \frac{1}{2}\right)}{\sqrt{\left(x^{-3} + \frac{1}{2}\right)^2 + \frac{3}{4}}}$

$$
= - \frac {1}{3} \ln \left| x ^ {- 3} + \frac {1}{2} + \sqrt {x ^ {- 6} + x ^ {- 3} + 1} \right| + C _ {1} = - \frac {1}{3} \ln \left| \frac {2 + x ^ {3} + 2 \sqrt {x ^ {6} + x ^ {3} + 1}}{x ^ {3}} \right| + C.
$$

注 以上实际已设 $x > 0$ . 对于 $x < 0$ , 利用 1856 题的方法可得同一结果.
