29.21 $\int \frac{\sqrt{1 + x^2}}{x} dx.$

Since $\sqrt{1 + x^2}$ is present, we let $x = \tan \theta$ , $dx = \sec^2 \theta d\theta$ . Then (Fig. 29-3), $\sqrt{1 + x^2} = \sec \theta$ . So $\int \frac{\sqrt{1 + x^2}}{x} dx = \int \frac{\sec \theta}{\tan \theta} \sec^2 \theta d\theta = \int \frac{\sec \theta}{\tan \theta} (1 + \tan^2 \theta) d\theta = \int (\csc \theta = \sec \theta \tan \theta) d\theta = \ln |\csc \theta - \cot \theta| + \sec \theta + C = \ln \left|\frac{\sqrt{1 + x^2} - 1}{x}\right| + \sqrt{1 + x^2} + C.$
