【1772】 $\int \frac{\sin x\cos^3x}{1 + \cos^2x}\mathrm{d}x.$

提示 令 $\cos^2 x = t$

解设 $\cos^2 x = t$ ，则 $\sin x\cos x\mathrm{d}x = -\frac{1}{2}\mathrm{d}t,$

故得 $\int \frac{\sin x\cos^3x}{1 + \cos^2x}\mathrm{d}x = -\frac{1}{2}\int \frac{t}{1 + t}\mathrm{d}t = -\frac{1}{2}\int \left(1 - \frac{1}{1 + t}\right)\mathrm{d}t = -\frac{1}{2} t + \frac{1}{2}\ln (1 + t) + C$

$$
= - \frac {1}{2} \cos^ {2} x + \frac {1}{2} \ln (1 + \cos^ {2} x) + C.
$$
