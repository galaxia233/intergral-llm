【2340】 $\int_{0}^{+\infty}\frac{\mathrm{d}x}{1 + x^3}.$

提示 从定义出发，并利用1881题的结果.

解 由于 $\lim_{b\to +\infty}\int_{0}^{b}\frac{\mathrm{d}x}{1 + x^3} = \lim_{b\to +\infty}\left[\frac{1}{6}\ln \frac{(x + 1)^2}{x^2 - x + 1} +\frac{1}{\sqrt{3}}\arctan \frac{2x - 1}{\sqrt{3}}\right]^{*)}\Bigg|_{0}^{b} = \frac{2\pi}{3\sqrt{3}},$ 所以， $\int_0^{+\infty}\frac{\mathrm{d}x}{1 + x^3} = \frac{2\pi}{3\sqrt{3}}.$

*) 利用1881题的结果.
