【4143】求以下列曲面 (参变量是正的) 为界的均匀物体对于坐标平面的转动惯量：
$\frac{x}{a} +\frac{y}{b} +\frac{z}{c} = 1,x = 0,y = 0,z = 0.$

解 $I_{xy} = \int_{a}^{a}\mathrm{d}x\int_{a}^{b(1 - \frac{x}{a})}\mathrm{d}y\int_{a}^{1(1 - \frac{x}{a} +\frac{y}{b})}z^2\mathrm{d}z = \frac{c^3}{3}\int_{a}^{a}\mathrm{d}x\int_{a}^{b(1 - \frac{x}{a})}\left(1 - \frac{x}{a} -\frac{y}{b}\right)^3\mathrm{d}y$

$$
= - \frac {b c ^ {3}}{1 2} \int_ {0} ^ {u} \left(1 - \frac {x}{a} - \frac {y}{b}\right) ^ {4} \left| _ {0} ^ {b (1 - \frac {x}{a})} \right. \mathrm {d} x = \frac {b c ^ {3}}{1 2} \int_ {0} ^ {u} \left(1 - \frac {x}{a}\right) ^ {4} \mathrm {d} x = \frac {a b c ^ {3}}{6 0}.
$$

利用对称性可得 $I_{yz} = \frac{a^3bc}{60}, \quad I_{zx} = \frac{ab^3c}{60}.$
