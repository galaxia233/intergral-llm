39.46 Find the Maclaurin series for $\ln (2 + x)$

$\ln (2 + x) = \ln [2(1 + x / 2)] = \ln 2 + \ln (1 + x / 2).$ But $\ln (1 + x) = \sum_{n = 1}^{\infty}\frac{(-1)^{n + 1}x^n}{n}.$ Hence, $\ln \left(1 + \frac{x}{2}\right) = \sum_{n = 1}^{\infty}\frac{(-1)^{n + 1}}{n^2}x^n.$ Thus, $\ln (2 + x) = \ln 2 + \sum_{n = 1}^{\infty}\frac{(-1)^{n + 1}}{n2^n}x^n.$
