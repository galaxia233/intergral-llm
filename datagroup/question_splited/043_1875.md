【1875】 $\int \frac{\mathrm{d}x}{x^5 + x^4 - 2x^3 - 2x^2 + x + 1}$

解 $\frac{1}{x^5 + x^4 - 2x^3 - 2x^2 + x + 1} = \frac{1}{(x - 1)^2(x + 1)^3} = \frac{A}{x - 1} +\frac{B}{(x - 1)^2} +\frac{C}{x + 1} +\frac{D}{(x + 1)^2} +\frac{E}{(x + 1)^3},$

通分后应有 $1 \equiv A(x - 1)(x + 1)^{3} + B(x + 1)^{3} + C(x - 1)^{2}(x + 1)^{2} + D(x - 1)^{2}(x + 1) + E(x - 1)^{2}$ . 在这恒等式中，

令 $x = 1$ ，得 $1 = 8B$ ， $B = \frac{1}{8}$ ；令 $x = -1$ ，得 $1 = 4E$ ， $E = \frac{1}{4}$ ；令 $x = 0$ ，得 $-A + B + C + D + E = 1$

令 $x = 2$ ，得 $27A + 27B + 9C + 3D + E = 1$ ；令 $x = -2$ ，得 $3A - B + 9C - 9D + 9E = 1$

由此， $A = -\frac{3}{16}$ ， $C = \frac{3}{16}$ ， $D = \frac{1}{4}$ 于是，

$$
\begin{array}{l} \int \frac {\mathrm {d} x}{x ^ {5} + x ^ {4} - 2 x ^ {3} - 2 x ^ {2} + x + 1} = \int \left[ - \frac {3}{1 6 (x - 1)} + \frac {1}{8 (x - 1) ^ {2}} + \frac {3}{1 6 (x + 1)} + \frac {1}{4 (x + 1) ^ {2}} + \frac {1}{4 (x + 1) ^ {3}} \right] \mathrm {d} x \\ = - \frac {3}{1 6} \ln | x - 1 | - \frac {1}{8 (x - 1)} + \frac {3}{1 6} \ln | x + 1 | - \frac {1}{4 (x + 1)} - \frac {1}{8 (x + 1) ^ {2}} + C \\ = \frac {3}{1 6} \ln \left| \frac {x + 1}{x - 1} \right| - \frac {3 x ^ {2} + 3 x - 2}{8 (x - 1) (x + 1) ^ {2}} + C. \\ \end{array}
$$
