【3105】根据欧拉的定义 $\Gamma$ 函数 $\Gamma(x)$ 由下面的公式来确定：

$$
\Gamma (x) = \lim  _ {n \rightarrow \infty} \frac {n ! n ^ {x}}{x (x + 1) \cdots (x + n)}.
$$

(3) 推出下面这个性质： $\Gamma (x + 1) = x\Gamma (x)$ ；

解（3）由于

$$
\frac {\Gamma (x + 1)}{\Gamma (x)} = \frac {\lim  _ {n \rightarrow \infty} \frac {n ! n ^ {x + 1}}{(x + 1) \cdots (x + n + 1)}}{\lim  _ {n \rightarrow \infty} \frac {n ! n ^ {x}}{x (x + 1) \cdots (x + n)}} = \lim  _ {n \rightarrow \infty} \frac {\frac {n ! n ^ {x + 1}}{(x + 1) \cdots (x + n + 1)}}{\frac {n ! n ^ {x}}{x (x + 1) \cdots (x + n)}} = \lim  _ {n \rightarrow \infty} \frac {n x}{x + n + 1} = x,
$$

故 $\Gamma (x + 1) = x\Gamma (x)$
