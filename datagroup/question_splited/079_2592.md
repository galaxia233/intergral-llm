【2592】证明：若 $\lim_{n\to \infty}\frac{a_{n + 1}}{a_n} = q < 1$ （ $a_{n} > 0$ )，则级数 $\sum_{n = 1}^{\infty}a_{n}$ 收敛.

逆命题不成立. 研究例子 $\frac{1}{2} + \frac{1}{3} + \frac{1}{2^2} + \frac{1}{3^2} + \frac{1}{2^3} + \frac{1}{3^3} + \dots$ .

证取 $0 < \varepsilon < 1 - q$ ，由于 $\lim_{n\to \infty}\frac{a_{n + 1}}{a_n} = q < 1$ ，故存在 $n_0$ ，使当 $n\geqslant n_0$ 时，有 $\frac{a_{n + 1}}{a_n} < q + \varepsilon = l < 1.$ 从而，

$$
0 <   a _ {n} \leqslant a _ {n _ {0}} l ^ {n - n _ {0}} \quad (n \geqslant n _ {0}).
$$

由于级数 $\sum_{n = n_0}^{\infty}l^{n - n_0}$ 收敛，故级数 $\sum_{n = n_0}^{\infty}a_n$ 收敛，从而，级数 $\sum_{n = 1}^{\infty}a_{n}$ 收敛.

反之不真，例如，级数 $\frac{1}{2} + \frac{1}{3} + \frac{1}{2^2} + \frac{1}{3^2} + \frac{1}{2^3} + \frac{1}{3^3} + \dots$ 显然是收敛的。但是，

$$
\frac {a _ {n + 1}}{a _ {n}} = \left\{ \begin{array}{l l} \left(\frac {2}{3}\right) ^ {m + 1}, & n = 2 m + 1, \\ \frac {1}{2} \left(\frac {3}{2}\right) ^ {m}, & n = 2 m, \end{array} \right.
$$

故有 $\lim_{n\to \infty}\frac{a_{n + 1}}{a_n} = +\infty .$
