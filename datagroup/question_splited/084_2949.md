【2949】在区间 $(a, a + 2l)$ 内展开 $f(x) = x$

解 由于

$$
\begin{array}{l} a _ {0} = \frac {1}{l} \int_ {a} ^ {a - 2 l} x d x = 2 (a + l), \\ a _ {n} = \frac {1}{l} \int_ {a} ^ {a + 2 l} x \cos \frac {n \pi x}{l} d x = \left. \frac {1}{n \pi} x \sin \frac {n \pi x}{l} \right| _ {a} ^ {a + 2 l} - \frac {1}{n \pi} \int_ {a} ^ {a + 2 l} \sin \frac {n \pi x}{l} d x = \frac {2 l}{n \pi} \sin \frac {n \pi a}{l}; \\ b _ {n} = \frac {1}{l} \int_ {a} ^ {a + 2 l} x \sin \frac {n \pi x}{l} d x = - \frac {1}{n \pi} x \cos \frac {n \pi x}{l} \left| _ {a} ^ {a + 2 l} + \frac {1}{n \pi} \int_ {a} ^ {a + 2 l} \cos \frac {n \pi x}{l} d x = - \frac {2 l}{n \pi} \cos \frac {n \pi a}{l}, \right. \\ \end{array}
$$

故按展开定理， $f(x)$ 可展开为

$$
a + l + \frac {2 l}{\pi} \sum_ {n = 1} ^ {\infty} \frac {1}{n} \left(\sin \frac {n \pi a}{l} \cos \frac {n \pi x}{l} - \cos \frac {n \pi a}{l} \sin \frac {n \pi x}{l}\right) = x \quad (a < x < a + 2 l).
$$
