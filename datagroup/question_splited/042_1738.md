【1738】 $\int \frac{x\mathrm{d}x}{x^4 + 3x^2 + 2}$

提示 注意 $\frac{x\mathrm{d}x}{x^4 + 3x^2 + 2} = \frac{1}{2}\cdot \frac{\mathrm{d}(x^2)}{(x^2 + 1)(x^2 + 2)}$ 并仿 1733 题的解法.

解 $\int \frac{x\mathrm{d}x}{x^4 + 3x^2 + 2} = \frac{1}{2}\int \frac{\mathrm{d}(x^2)}{(x^2 + 1)(x^2 + 2)} = \frac{1}{2}\int \left(\frac{1}{x^2 + 1} -\frac{1}{x^2 + 2}\right)\mathrm{d}(x^2) = \frac{1}{2}\ln \frac{x^2 + 1}{x^2 + 2} +C.$
