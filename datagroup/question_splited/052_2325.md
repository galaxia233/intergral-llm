【2325】 $\int_0^{100}\frac{\mathrm{e}^{-x}}{x + 100}\mathrm{d}x.$

解 $I = \int_{0}^{50}\frac{\mathrm{e}^{-x}}{x + 100}\mathrm{d}x + \int_{50}^{100}\frac{\mathrm{e}^{-x}}{x + 100}\mathrm{d}x = \frac{1}{100 + \xi_1}\int_{0}^{50}\mathrm{e}^{-x}\mathrm{d}x + \frac{1}{100 + \xi_2}\int_{50}^{100}\mathrm{e}^{-x}\mathrm{d}x$

$$
= \frac {1 - e ^ {- 5 0}}{1 0 0 + \xi_ {1}} + \frac {e ^ {- 5 0} - e ^ {- 1 0 0}}{1 0 0 + \xi_ {2}},
$$

其中 $0 \leqslant \xi_{1} \leqslant 50, 50 \leqslant \xi_{2} \leqslant 100$ . 显然

$$
\frac {1 - e ^ {- 5 0}}{1 0 0 + \xi_ {1}} + \frac {e ^ {- 5 0} - e ^ {- 1 0 0}}{1 0 0 + \xi_ {2}} \leqslant \frac {1 - e ^ {- 5 0}}{1 0 0 + \xi_ {1}} + \frac {e ^ {- 5 0} - e ^ {- 1 0 0}}{1 0 0 + \xi_ {1}} = \frac {1 - e ^ {- 1 0 0}}{1 0 0 + \xi_ {1}} <   \frac {1}{1 0 0},
$$

$$
\frac {1 - e ^ {- 5 0}}{1 0 0 + \xi_ {1}} + \frac {e ^ {- 5 0} - e ^ {- 1 0 0}}{1 0 0 + \xi_ {2}} > \frac {1 - e ^ {- 5 0}}{1 0 0 + \xi_ {1}} \geqslant \frac {1 - e ^ {- 5 0}}{1 5 0} > \frac {1}{2 0 0},
$$

故 $\frac{1}{200} < I < \frac{1}{100}$ ，即 $I = 0.01 - 0.005\theta, 0 < \theta < 1$ 。此外，按中值定理，可写

$$
I = \int_ {0} ^ {1 0 0} \frac {\mathrm {e} ^ {- x}}{x + 1 0 0} \mathrm {d} x = \frac {1}{\xi + 1 0 0} \int_ {0} ^ {1 0 0} \mathrm {e} ^ {- x} \mathrm {d} x = \frac {1}{\xi + 1 0 0} \left(1 - \frac {1}{\mathrm {e} ^ {1 0 0}}\right),
$$

其中 $0 \leqslant \xi \leqslant 100$ ，如果改写 $I$ 为 $I = 0.01 - 0.005\theta$ ，则有 $\theta = f(\xi) = \frac{2}{100 + \xi}\left(\xi + \frac{100}{e^{100}}\right)$ .

易见导数 $f^{\prime}(\xi) = \frac{200(1 - e^{-100})}{(100 + \xi)^{2}} > 0,$

$f(\xi)$ 单调上升，故在 $[0,100]$ 上有 $f(0) \leqslant f(\xi) \leqslant f(100)$ ，也即有 $\frac{2}{\mathrm{e}^{100}} \leqslant \theta \leqslant 1 + \frac{1}{\mathrm{e}^{100}}$

根据前面的估计 $0 < \theta < 1$ ，综合起来，便有 $\frac{2}{\mathrm{e}^{100}} \leqslant \theta < 1.$

这个结果比原来的估计又好了一些。如果更精确一些，采用些近似计算方法，还可进一步明确 $\theta$ 的数值范围。此处从略.
