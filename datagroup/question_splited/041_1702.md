【1702】 $\int \frac{\mathrm{d}x}{\sin^2 x + 2\cos^2 x}$

提示 注意 $\frac{\mathrm{d}x}{\sin^2 x + 2\cos^2 x} = \frac{\frac{1}{\cos^2 x} \mathrm{d}x}{\tan^2 x + 2} = \frac{\mathrm{d}(\tan x)}{(\sqrt{2})^2 + (\tan x)^2}$ .

解 $\int \frac{\mathrm{d}x}{\sin^2 x + 2\cos^2 x} = \int \frac{\frac{1}{\cos^2 x}}{\tan^2 x + 2} \mathrm{d}x = \int \frac{\mathrm{d}(\tan x)}{\tan^2 x + 2} = \frac{1}{\sqrt{2}} \arctan \left( \frac{\tan x}{\sqrt{2}} \right) + C.$
