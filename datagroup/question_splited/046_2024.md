【2024】 $\int \tan x \tan (x + a) \, \mathrm{d}x.$

解 $\int \tan x \tan (x + a) \, \mathrm{d}x = \int \frac{\sin x \sin (x + a)}{\cos x \cos (x + a)} \, \mathrm{d}x$

$$
\begin{array}{l} = \int \frac {\cos x \cos (x + a) + \sin x \sin (x + a) - \cos x \cos (x + a)}{\cos x \cos (x + a)} d x = \int \frac {\cos a - \cos x \cos (x + a)}{\cos x \cos (x + a)} d x \\ = - x + \cos a \int \frac {d x}{\cos (x + a) \cos x} = - x + \cot a \ln \left| \frac {\cos x}{\cos (x + a)} \right| ^ {\prime \prime} + C, \\ \end{array}
$$

其中 $\sin a\neq 0$
