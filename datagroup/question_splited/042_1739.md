【1739】 $\int \frac{\mathrm{d}x}{(x + a)^2(x + b)^2}\quad (a\neq b).$

提示 注意 $\frac{1}{(x + a)^2(x + b)^2} = \frac{1}{(a - b)^2}\left(\frac{1}{x + b} -\frac{1}{x + a}\right)^2 = \frac{1}{(a - b)^2}\left[\frac{1}{(x + a)^2} +\frac{1}{(x + b)^2} -\frac{2}{(x + a)(x + b)}\right].$

解 $\int \frac{\mathrm{d}x}{(x + a)^2(x + b)^2} = \frac{1}{(a - b)^2}\int \left(\frac{1}{x + a} -\frac{1}{x + b}\right)^2\mathrm{d}x$

$$
\begin{array}{l} = \frac {1}{(a - b) ^ {2}} \int \left[ \frac {1}{(x + a) ^ {2}} + \frac {1}{(x + b) ^ {2}} - \frac {2}{(x + a) (x + b)} \right] d x \\ = - \frac {1}{(a - b) ^ {2}} \left(\frac {1}{x + a} + \frac {1}{x + b}\right) - \frac {2}{(a - b) ^ {2}} \int \frac {\mathrm {d} x}{(x + a) (x + b)} \\ = - \frac {2 x + a + b}{(a - b) ^ {2} (x + a) (x + b)} + \frac {2}{(a - b) ^ {3}} \ln \left| \frac {x + a}{x + b} \right| + C. \\ \end{array}
$$
