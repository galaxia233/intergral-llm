【1732】 $\int x^3\sqrt[3]{1 + x^2}\mathrm{d}x.$

解 $\int x^{3}\sqrt[3]{1 + x^{2}}\mathrm{d}x = \frac{1}{2}\int [(x^{2} + 1) - 1](1 + x^{2})^{\frac{1}{3}}\mathrm{d}(1 + x^{2}) = \frac{1}{2}\int [(1 + x^{2})^{\frac{4}{3}} - (1 + x^{2})^{\frac{1}{3}}]\mathrm{d}(1 + x^{2})$

$$
= \frac {3}{1 4} (1 + x ^ {2}) ^ {\frac {7}{3}} - \frac {3}{8} (1 + x ^ {2}) ^ {\frac {4}{3}} + C = \frac {1 2 x ^ {2} - 9}{5 6} (1 + x ^ {2}) ^ {\frac {4}{3}} + C.
$$
