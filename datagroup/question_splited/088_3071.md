【3071】 $\prod_{n = n_0}^{\infty}\frac{n^2 + a_1n + b_1}{n^2 + an + b}$ ，其中当 $n\geqslant n_0$ 时 $n^2 +an + b > 0.$

解通项 $p_{n} = \frac{n^{2} + a_{1}n + b_{1}}{n^{2} + an + b} = 1 + \frac{(a_{1} - a)n + (b_{1} - b)}{n^{2} + an + b}$

$$
a _ {n} = \frac {(a _ {1} - a) n + (b _ {1} - b)}{n ^ {2} + a n + b}.
$$

当 $a_1 = a$ 时， $a_n \sim \frac{1}{n^2}$ 。由于 $\sum_{n=n_0}^{\infty} \frac{1}{n^2}$ 收敛，故原无穷乘积收敛。当 $a_1 \neq a$ 时，由于 $n^2 + an + b > 0$ ，且 $a_n \sim \frac{a_1 - a}{n}$ ，故 $\sum_{n=n_0}^{\infty} a_n$ 发散，从而，原无穷乘积也发散。
