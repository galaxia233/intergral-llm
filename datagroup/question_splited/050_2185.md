【2185】 $\int_{-1}^{2}x^{2}\mathrm{d}x.$

解题思路 由于 $f(x) = x^2$ 在 $[-1,2]$ 上连续，故所给的定积分存在，且它与分法无关，同时也与点 $\xi_i$ 的取法无关。本题将 $[-1,2]$ 等分，得子区间的长 $h = \frac{3}{n}$ ，并取点 $\xi_i = -1 + ih$ （ $i = 0,1,2,\dots,n-1$ ），这种和的极限就是所求的定积分。以下各题如无特殊情况，不再说明定积分的存在性，直接对区间分段并取点 $\xi_i$ ，作和求极限。

解 将区间 $[-1, 2]$ 等分，得 $h = \frac{3}{n}$ . 取 $\xi_i = -1 + ih$ $(i = 0, 1, \dots, n-1)$ .

作和 $S_{n} = \sum_{i = 0}^{n - 1}(-1 + ih)^{2}h = nh - 2h^{2}\sum_{i = 0}^{n - 1}i + h^{3}\sum_{i = 0}^{n - 1}i^{2} = 3 + \frac{9 - 9n}{2n^{2}}.$

于是， $\lim_{n\to \infty}S_n = 3.$

由于 $f(x) = x^{2}$ 在 $[-1, 2]$ 上连续，故积分 $\int_{-1}^{2} x^{2} \mathrm{d}x$ 是存在的，且它与分法无关，同时也与点的取法无关。因此，上述和的极限就是所求的积分值，即定积分

$$
\int_ {- 1} ^ {2} x ^ {2} \mathrm {d} x = 3.
$$
