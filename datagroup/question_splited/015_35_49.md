35.49 Find the largest value of $y$ on the cardioid $r = 2(1 + \cos \theta)$

$y = r\sin \theta = 2(1 + \cos \theta)\sin \theta.$ Hence, $dy / d\theta = 2[(1 + \cos \theta)\cos \theta -\sin^2\theta ]$ Setting $dy / d\theta = 0$ we find $(1 + \cos \theta)\cos \theta = \sin^2\theta ,\cos \theta +\cos^2\theta = 1 - \cos^2\theta ,2\cos^2\theta +\cos \theta -1 = 0,(2\cos \theta -1)(\cos \theta +1) =$ $0,\cos \theta = \frac{1}{2}$ or $\cos \theta = -1$ . Hence, we get the critical numbers $\pi ,\pi /3,5\pi /3$ . If we calculate $y$ for these values and for the $t$ endpoints O and $2\pi$ , the largest value $3\sqrt{3} /2$ is assumed when $\theta = \pi /3$
