【1819】 $\int \sqrt{x^2 + a} \, \mathrm{d}x.$

解 $\int \sqrt{x^2 + a} \, \mathrm{d}x = x \sqrt{x^2 + a} - \int \frac{x^2 \, \mathrm{d}x}{\sqrt{x^2 + a}} = x \sqrt{x^2 + a} - \int \sqrt{x^2 + a} \, \mathrm{d}x + a \int \frac{\mathrm{d}x}{\sqrt{x^2 + a}}.$

于是，得 $\int \sqrt{x^2 + a} \, \mathrm{d}x = \frac{x}{2} \sqrt{x^2 + a} + \frac{a}{2} \int \frac{\mathrm{d}x}{\sqrt{x^2 + a}} = \frac{x}{2} \sqrt{x^2 + a} + \frac{a}{2} \ln |x + \sqrt{x^2 + a}| + C.$
