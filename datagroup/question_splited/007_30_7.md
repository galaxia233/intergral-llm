30.7 $\int \frac{x dx}{x^4 - 13x^2 + 36}.$

$x^4 - 13x^2 + 36 = (x^2 - 9)(x^2 - 4) = (x - 3)(x + 3)(x + 2)(x - 2). \quad \text{Let} \quad \frac{x}{(x - 3)(x + 3)(x + 2)(x - 2)} = \frac{A}{x - 3} + \frac{B}{x + 3} + \frac{C}{x + 2} + \frac{D}{x - 2}.$ Then $x = A(x + 3)(x + 2)(x - 2) + B(x - 3)(x + 2)(x - 2) + C(x - 3)(x + 3)(x - 2) + D(x - 3)(x + 3)(x - 2)$ . Let $x = 3$ . Then $3 = 30A$ , $A = \frac{1}{10}$ . Let $x = -3$ . Then $-3 = -30B$ . $B = \frac{1}{10}$ . Let $x = -2$ . Then $-2 = 20C$ , $C = -\frac{1}{10}$ . Let $x = 2$ . Then $2 = -20D$ , $D = -\frac{1}{10}$ . So $\frac{x}{x^4 - 13x^2 + 36} = \frac{1}{10}\frac{1}{x - 3} + \frac{1}{10}\frac{1}{x + 3} - \frac{1}{10}\frac{1}{x + 2} - \frac{1}{10}\frac{1}{x - 2},$ and $\int_{\frac{x}{x^4 - 13x^2 + 36}}^{x} dx = \frac{1}{10}\ln |x - 3| + \frac{1}{10}\ln |x + 3| - \frac{1}{10}\ln |x + 2| - \frac{1}{10}\ln |x - 2| + C_1 = \frac{1}{10}\ln \left|\frac{(x - 3)(x + 3)}{(x + 2)(x - 2)}\right| + C_1.$
