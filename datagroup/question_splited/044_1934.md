【1934】 $\int \frac{\mathrm{d}x}{\sqrt[n]{(x - a)^{n + 1}(x - b)^{n - 1}}}$ （ $n$ 为正整数）.

解当 $a = b$ 时，显然被积函数为 $(x - a)^{-2}$ ，因此，所求的积分为 $-\frac{1}{x - a} + C$ ；当 $a \neq b$ 时，设 $\sqrt[n]{\frac{x - b}{x - a}} = t$

则 $x = a + \frac{a - b}{t^n - 1},\quad \mathrm{d}x = -\frac{n(a - b)t^{n - 1}}{(t^n - 1)^2}\mathrm{d}t,x - a = \frac{a - b}{t^n - 1},\quad x - b = \frac{(a - b)t^n}{t^n - 1}.$

代人得 $\int \frac{\mathrm{d}x}{\sqrt[n]{(x - a)^{n + 1}(x - b)^{n - 1}}} = -\frac{n}{a - b}\int \mathrm{d}t = -\frac{n}{a - b} t + C = -\frac{n}{a - b}\sqrt[n]{\frac{x - b}{x - a}} +C.$
