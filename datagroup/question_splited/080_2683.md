【2683】 $\sum_{n = 1}^{\infty}(-1)^{n}\frac{n - 1}{n + 1}\cdot \frac{1}{\sqrt[100]{n}}.$

解通项为

$$
a _ {n} = (- 1) ^ {n} \frac {n - 1}{n + 1} \cdot \frac {1}{\sqrt [ 1 0 0 ]{n}} = (- 1) ^ {n} \frac {1}{\sqrt [ 1 0 0 ]{n}} \left[ 1 + O \left(\frac {1}{n}\right) \right] = \frac {(- 1) ^ {n}}{\sqrt [ 1 0 0 ]{n}} + O \left(\frac {1}{n ^ {1 + \frac {1}{1 0 0}}}\right).
$$

显然 $\sum_{n=1}^{\infty} \frac{1}{n^{1+\frac{1}{100}}}$ 绝对收敛，而 $\sum_{n=1}^{\infty} \frac{(-1)^n}{\sqrt[n]{n}}$ 条件收敛，故原级数条件收敛.
