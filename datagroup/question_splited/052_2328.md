【2328】 $\int_{100\pi}^{200\pi}\frac{\sin x}{x}\mathrm{d}x.$

解设 $f(x) = \sin x, \varphi (x) = \frac{1}{x},$

则 $f(x)$ 及 $\varphi (x)$ 在 $[100\pi ,200\pi ]$ 上满足第二中值定理的条件，又 $\varphi (x) = \frac{1}{x}$ 单调下降且不为负，于是；

$$
\int_ {1 0 0 \pi} ^ {2 0 0 \pi} \frac {\sin x}{x} d x = \frac {1}{1 0 0 \pi} \int_ {1 0 0 \pi} ^ {\varepsilon} \sin x d x = \frac {1 - \cos \xi}{1 0 0 \pi} = \frac {\sin^ {2} \frac {\xi}{2}}{5 0 \pi} = \frac {\theta}{5 0 \pi},
$$

其中 $100\pi \leqslant \xi \leqslant 200\pi$ 及 $0 \leqslant \theta \leqslant 1$
