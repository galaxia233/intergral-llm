【1880】 $\int \frac{\mathrm{d}x}{x(1 + x)(1 + x + x^2)}.$

解设 $\frac{1}{x(1 + x)(1 + x + x^2)} = \frac{A}{x} +\frac{B}{x + 1} +\frac{Cx + D}{x^2 + x + 1}$ ，通分后应有

$$
1 \equiv A (x + 1) (1 + x + x ^ {2}) + B x (1 + x + x ^ {2}) + x (x + 1) (C x + D).
$$

比较等式两端 $x$ 的同次幂系数，得

$$
\begin{array}{l l} x ^ {3} & A + B + C = 0, \\ x ^ {2} & 2 A + B + C + D = 0, \\ x ^ {1} & 2 A + B + D = 0, \\ x ^ {0} & A = 1. \end{array}
$$

由此， $A = 1$ ， $B = -1$ ， $C = 0$ ， $D = -1$ .于是，

$$
\int \frac {d x}{x (1 + x) \left(1 + x + x ^ {2}\right)} = \int \left(\frac {1}{x} - \frac {1}{1 + x} - \frac {1}{1 + x + x ^ {2}}\right) d x = \ln \left| \frac {x}{1 + x} \right| - \frac {2}{\sqrt {3}} \arctan \frac {2 x + 1}{\sqrt {3}} + C.
$$

本题也可以不用待定系数法。事实上，

$$
\frac {1}{x (1 + x) \left(1 + x + x ^ {2}\right)} = \frac {1}{(x + x ^ {2}) \left(1 + x + x ^ {2}\right)} = \frac {1}{x + x ^ {2}} - \frac {1}{1 + x + x ^ {2}} = \frac {1}{x} - \frac {1}{1 + x} - \frac {1}{1 + x + x ^ {2}}.
$$
