【1877】 $\int \frac{\mathrm{d}x}{(x + 1)(x^2 + 1)}.$

解设 $\frac{1}{(x + 1)(x^2 + 1)} = \frac{A}{x + 1} +\frac{Bx + C}{x^2 + 1}$ ，通分后应有 $1\equiv A(x^{2} + 1) + (Bx + C)(x + 1)$ .比较等式两端 $x$ 的同次幂的系数，得

$$
\begin{array}{l} x ^ {2} \mid A + B = 0, \\ x ^ {1} \mid B + C = 0, \\ x ^ {0} \mid A + C = 1. \end{array}
$$

由此， $A = \frac{1}{2}$ ， $B = -\frac{1}{2}$ ， $C = \frac{1}{2}$ 于是，

$$
\begin{array}{l} \int \frac {\mathrm {d} x}{(x + 1) (x ^ {2} + 1)} = \int \left[ \frac {1}{2 (x + 1)} - \frac {x - 1}{2 (x ^ {2} + 1)} \right] \mathrm {d} x = \frac {1}{2} \ln | x + 1 | - \frac {1}{4} \ln (x ^ {2} + 1) + \frac {1}{2} \arctan x + C \\ = \frac {1}{4} \ln \frac {(x + 1) ^ {2}}{x ^ {2} + 1} + \frac {1}{2} \arctan x + C. \\ \end{array}
$$
