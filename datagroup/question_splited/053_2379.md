【2379】 $\int_{0}^{+\infty}\frac{\sqrt{x}\cos x}{x + 100}\mathrm{d}x.$

解 对于任意的 $A > 0$ ，由于 $\left|\int_{0}^{A}\cos x\mathrm{d}x\right| \leqslant 2$ ，且当 $x \to +\infty$ 时， $\frac{\sqrt{x}}{x + 100}$ 单调地趋于零，因此，积分 $\int_{0}^{+\infty}\frac{\sqrt{x}\cos x}{x + 100}\mathrm{d}x$ 收敛。但它不是绝对收敛的。

事实上，由于

$$
\frac {\sqrt {x} | \cos x |}{x + 1 0 0} \geqslant \frac {\sqrt {x} \cos^ {2} x}{x + 1 0 0} = \frac {1}{2} \left(\frac {\sqrt {x}}{x + 1 0 0} - \frac {\sqrt {x} \cos 2 x}{x + 1 0 0}\right),
$$

且 $\lim_{x\to +\infty}\left(x^{\frac{1}{2}}\frac{\sqrt{x}}{x + 100}\right) = 1$ ，故积分 $\int_0^{+\infty}\frac{\sqrt{x}}{x + 100}\mathrm{d}x$ 发散。仿照前半段证明，可知 $\int_0^{+\infty}\frac{\sqrt{x}\cos 2x}{x + 100}\mathrm{d}x$ 收敛。从而，积分 $\int_0^{+\infty}\frac{\sqrt{x}\cos^2x}{x + 100}\mathrm{d}x$ 发散.

于是，积分 $\int_{0}^{+\infty}\frac{\sqrt{x}|\cos x|}{x + 100}\mathrm{d}x$ 发散.
