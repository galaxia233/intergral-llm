【4100】令 $x + y + z = \xi, y + z = \xi \eta, z = \xi \eta \zeta$ ，计算狄利克雷积分

$$
\iiint_ {V} x ^ {p} y ^ {q} z ^ {r} (1 - x - y - z) ^ {s} \mathrm {d} x \mathrm {d} y \mathrm {d} z \quad (p > 0, q > 0, r > 0, s > 0),
$$

其中 $V$ 是平面 $x + y + z = 1, x = 0, y = 0, z = 0$ 所围的区域.

解 由假设知 $x = \xi (1 - \eta),\quad y = \xi \eta (1 - \zeta),\quad z = \xi \eta \zeta .$

在此变换下可求得 $|I| = \xi^2\eta$ ，并且积分域 $V$ 变为：

$$
0 <   \xi <   1, 0 <   \eta <   1, 0 <   \zeta <   1.
$$

于是，

$$
\begin{array}{l} \iiint_ {V} x ^ {p} y ^ {q} z ^ {r} (1 - x - y - z) ^ {3} \mathrm {d} x \mathrm {d} y \mathrm {d} z = \int_ {0} ^ {1} \xi^ {p - q + r + 2} (1 - \xi) ^ {3} \mathrm {d} \xi \int_ {0} ^ {1} \eta^ {q - r - 1} (1 - \eta) ^ {p} \mathrm {d} \eta \int_ {0} ^ {1} \zeta^ {r} (1 - \zeta) ^ {q} \mathrm {d} \zeta \\ = B (p + q + r + 3, s + 1) B (q + r + 2, p + 1) B (r + 1, q + 1) \\ = \frac {\Gamma (p + q + r + 3) \Gamma (s + 1) \Gamma (q + r + 2) \Gamma (p + 1) \Gamma (r + 1) \Gamma (q + 1)}{\Gamma (p + q + r + s + 4) \Gamma (p + q + r + 3) \Gamma (q + r + 2)} \\ = \frac {\Gamma (p + 1) \Gamma (q + 1) \Gamma (s + 1) \Gamma (r + 1)}{\Gamma (p + q + r + s + 4)}. \\ \end{array}
$$
