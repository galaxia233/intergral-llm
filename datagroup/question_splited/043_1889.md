【1889】 $\int \frac{x^2\mathrm{d}x}{x^4 + 3x^3 + \frac{9}{2}x^2 + 3x + 1}.$

解设 $\frac{x^2}{x^4 + 3x^3 + \frac{9}{2}x^2 + 3x + 1} = \frac{Ax + B}{x^2 + 2x + 2} +\frac{Cx + D}{x^2 + x + \frac{1}{2}}$ ，通分后应有

$$
x ^ {2} \equiv (A x + B) (x ^ {2} + x + \frac {1}{2}) + (C x + D) (x ^ {2} + 2 x + 2).
$$

比较等式两端 $x$ 的同次幂系数，得

$$
\begin{array}{l l} x ^ {3} & A + C = 0, \\ x ^ {2} & A + B + 2 C + D = 1, \\ x ^ {1} & \frac {A}{2} + B + 2 C + 2 D = 0, \\ x ^ {0} & \frac {B}{2} + 2 D = 0. \end{array}
$$

由此， $A = \frac{4}{5},B = \frac{12}{5},C = -\frac{4}{5},D = -\frac{3}{5}$ 于是，

$$
\begin{array}{l} \int \frac {x ^ {2} \mathrm {d} x}{x ^ {4} + 3 x ^ {3} + \frac {9}{2} x ^ {2} + 3 x + 1} = \int \left[ \frac {4 (x + 3)}{5 (x ^ {2} + 2 x + 2)} - \frac {4 x + 3}{5 (x ^ {2} + x + \frac {1}{2})} \right] \mathrm {d} x \\ = \frac {2}{5} \int \frac {(2 x + 2) d x}{x ^ {2} + 2 x + 2} + \frac {8}{5} \int \frac {d (x + 1)}{(x + 1) ^ {2} + 1} - \frac {2}{5} \int \frac {(2 x + 1) d x}{x ^ {2} + x + \frac {1}{2}} - \frac {1}{5} \int \frac {d \left(x + \frac {1}{2}\right)}{\left(x + \frac {1}{2}\right) ^ {2} + \frac {1}{4}} \\ = \frac {2}{5} \ln \frac {x ^ {2} + 2 x + 2}{x ^ {2} + x + \frac {1}{2}} + \frac {8}{5} \arctan (x + 1) - \frac {2}{5} \arctan (2 x + 1) + C. \\ \end{array}
$$
