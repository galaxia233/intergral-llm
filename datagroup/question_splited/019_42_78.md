42.78 If $f(x, y)$ is homogeneous of degree $n$ and has continuous second-order partial derivatives, prove $x^{2}f_{xx} + 2xyf_{xy} + y^{2}f_{yy} = n(n - 1)f$ .

By Problem 42.74, $f_{x}(tx,ty)(x) + f_{y}(tx,ty)(y) = nt^{n - 1}f(x,y)$ . Differentiate with respect to $t$ : $x[f_{xx}(tx,ty)(x) + f_{xy}(tx,ty)(y)] + y[f_{yx}(tx,ty)(x) + f_{yy}(tx,ty)(y)] = n(n - 1)t^{n - 2}f(x,y)$ . Now let $t = 1$ : $x(f_{xx}\cdot x + f_{xy}\cdot y) + y(f_{yx}\cdot x + f_{yy}\cdot y) = n(n - 1)f, \quad x^{2}f_{xx} + 2xyf_{xy} + y^{2}f_{yy} = n(n - 1)f, \quad \text{since} \quad f_{xy} = f_{yx}.$
