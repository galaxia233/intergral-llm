гҖҗ1871гҖ‘ $\int \frac{x\mathrm{d}x}{x^3 - 3x + 2}$

и§Ј $\frac{x}{x^3 - 3x + 2} = \frac{x}{(x - 1)^2(x - 2)} = \frac{A}{x - 1} +\frac{B}{(x - 1)^2} +\frac{C}{x + 2}$ пјҢйҖҡеҲҶеҗҺеә”жңү $x\equiv A(x - 1)(x + 2) + B(x + 2)$ $+C(x - 1)^2$ .еңЁиҝҷжҒ’зӯүејҸдёӯпјҢ

д»Ө $x = 1$ пјҢеҫ— $1 = 3B$ пјҢ $B = \frac{1}{3}$ пјӣд»Ө $x = -2$ пјҢеҫ— $-2 = 9C$ пјҢ $C = -\frac{2}{9}$

жҜ”иҫғ $x^{2}$ зҡ„зі»ж•°пјҢеҫ— $A + C = 0$ пјҢд»ҺиҖҢпјҢ $A = \frac{2}{9}$ .дәҺжҳҜпјҢ

$$
\int \frac {x \mathrm {d} x}{x ^ {3} - 3 x + 2} = \int \left[ \frac {2}{9 (x - 1)} + \frac {1}{3 (x - 1) ^ {2}} - \frac {2}{9 (x + 2)} \right] \mathrm {d} x = - \frac {1}{3 (x - 1)} + \frac {2}{9} \ln \left| \frac {x - 1}{x + 2} \right| + C.
$$
