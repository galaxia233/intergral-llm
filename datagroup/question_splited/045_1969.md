【1969】 $\int \frac{x - \sqrt{x^2 + 3x + 2}}{x + \sqrt{x^2 + 3x + 2}x}\mathrm{d}x.$

提示 令 $\sqrt{x^2 + 3x + 2} = z(x + 1)$

解设 $\sqrt{x^2 + 3x + 2} = z(x + 1)$ ，则 $x = \frac{2 - z^2}{z^2 - 1},\mathrm{d}x = -\frac{2z}{(z^2 - 1)^2}\mathrm{d}z,\quad \sqrt{x^2 + 3x + 2} = \frac{z}{z^2 - 1}.$

代入得 $\int \frac{x - \sqrt{x^2 + 3x + 2}}{x + \sqrt{x^2 + 3x + 2}}\mathrm{d}x = \int \frac{2z(2 - z - z^2)}{(z^2 - z - 2)(z^2 - 1)^2}\mathrm{d}z$

$$
\begin{array}{l} = \int \left[ - \frac {1 7}{1 0 8 (z + 1)} + \frac {5}{1 8 (z + 1) ^ {2}} + \frac {1}{3 (z + 1) ^ {3}} + \frac {3}{4 (z - 1)} - \frac {1 6}{2 7 (z - 2)} \right] d z \\ = - \frac {1 7}{1 0 8} \ln | z + 1 | - \frac {5}{1 8 (z + 1)} - \frac {1}{6 (z + 1) ^ {2}} + \frac {3}{4} \ln | z - 1 | - \frac {1 6}{2 7} \ln | z - 2 | + C, \\ \end{array}
$$

其中 $z = \frac{\sqrt{x^2 + 3x + 2}}{x + 1}.$
