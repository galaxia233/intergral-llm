30.24 Find $\int \frac{\cos x}{\sin x - 1} dx$

Using the same approach as in Problem 30.23, we have: $\int \frac{\cos x dx}{\sin x - 1} = 2\int \frac{z + 1}{(1 + z^2)(z - 1)} dz.$ Let $\frac{z + 1}{(1 + z^2)(z - 1)} = \frac{A}{z - 1} + \frac{Bz + C}{1 + z^2}$ . Then $z + 1 = A(1 + z^2) + (z - 1)(Bz + C)$ . Let $z = 1$ . Then $2 = 2A$ , $A = 1$ . Equate coefficients of $z^2$ : $0 = A + B$ , $B = -A = -1$ . Equate constant coefficients: $1 = A + C$ , $C = 1 - A = 0$ . Thus, $\frac{z + 1}{(1 + z^2)(z - 1)} = \frac{A}{z - 1} - \frac{z}{1 + z^2}$ , and $2\int \frac{z + 1}{(1 + z^2)(z - 1)} dz = 2[\ln |z - 1| - \frac{1}{2}\ln (1 + z^2)] + C_1 = \ln |z - 1|^2 - \ln (1 + z^2) + C_1 = \ln \frac{(z - 1)^2}{1 + z^2} + C_1 = \ln (1 - \sin x) + C_1$ .
