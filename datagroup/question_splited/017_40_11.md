40.11 When does an equation $x^{2} + y^{2} + z^{2} + Ax + By + Cz + D = 0$ represent a sphere?

1 Complete the squares: $\left(x + \frac{A}{2}\right)^2 +\left(y + \frac{B}{2}\right)^2 +\left(z + \frac{C}{2}\right)^2 = \frac{A^2}{4} +\frac{B^2}{4} +\frac{C^2}{4} -D.$ This is a sphere if and only if the right side is positive; that is, if and only if $A^2 +B^2 +C^2 -4D > 0$ In that case, the sphere has radius $\sqrt[1]{A^2 + B^2 + C^2 - 4D}$ and center $\left(-\frac{A}{2}, - \frac{B}{2}, - \frac{C}{2}\right).]$ When $A^2 +B^2 +C^2 -4D = 0,$ the graph is a single point $\left(-\frac{A}{2}, - \frac{B}{2}, - \frac{C}{2}\right)$ . When $A^2 +B^2 +C^2 -4D <   0,$ there are no points on the graph at all.
