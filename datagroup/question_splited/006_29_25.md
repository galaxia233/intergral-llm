29.25 $\int \frac{dx}{(x^2 + 9)^2}.$

Since $x^{2} + 9$ appears, let $x = 3\tan \theta$ , $dx = 3\sec^2\theta d\theta$ , and (Fig. 29-5) $x^{2} + 9 = 9\sec^2\theta$ . Then, using Problem 29.1, $\int \frac{dx}{(x^2 + 9)^2} = \int \frac{3\sec^2\theta d\theta}{(9\sec^2\theta)^2} = \frac {1}{27}\int \frac {d \theta}{\sec^2\theta} = \frac {1}{27}\int \cos^2\theta d\theta = \frac {1}{27}\cdot \frac {1}{2} (\theta + \sin \theta \cos \theta) + C = \frac {1}{54}\left(\tan^{-1}\frac {x}{3} +\frac {x}{\sqrt {x^2 + 9}}\frac {3}{\sqrt {x^2 + 9}}\right) + C = \frac {1}{54}\left(\tan^{-1}\frac {x}{3} +\frac {3x}{x^2 + 9}\right) + C.$
