【2519】直径 $20\mathrm{cm}$ ，长 $80\mathrm{cm}$ 的圆柱形汽缸充满压强为 $100\mathrm{N/cm^2}$ 的蒸汽。假定蒸气的温度保持不变，要使其体积减小一半，需要作多少功？

解 由玻意耳一马略特定律有 $p v = C$ ，其中 $\pmb{p}$ 表示气体的压强， $\pmb{\nu}$ 表示体积， $C$ 为常量。由条件知，常量

$$
C = 1 0 \cdot \pi \cdot 1 0 0 \cdot 8 0 = 8 0 0 0 \pi (N \cdot m).
$$

设初始时气体体积为 $v_{0}$ ，将区间 $\left[\frac{v_0}{2}, v_0\right]$ 分成 $n$ 个小区间，分点依次为

$$
\frac {v _ {0}}{2}, \frac {v _ {0}}{2} q, \frac {v _ {0}}{2} q ^ {2}, \dots , \frac {v _ {0}}{2} q ^ {i}, \dots , \frac {v _ {0}}{2} q ^ {n} = v _ {0},
$$

其中 $q = \sqrt[3]{\frac{v_0}{\frac{v_0}{2}}} = \sqrt[3]{2}$ 由于气体体积从 $\frac{v_0}{2} q^{i+1}$ 减小至 $\frac{v_0}{2} q^i$ 需要花费功的近似值为

$$
C \left(\frac {v _ {0}}{2} q ^ {i}\right) ^ {- 1} \left(\frac {v _ {0}}{2} q ^ {i + 1} - \frac {v _ {0}}{2} q ^ {i}\right),
$$

于是，所要求的功为

$$
W = \lim _ {n \rightarrow \infty} \sum_ {i = 0 } ^ { n } C \left(\frac {v _ {0}}{2} q ^ {i}\right) ^ {- 1} \left(\frac {v _ {0}}{2} q ^ {i + 1} - \frac {v _ {0}}{2} q ^ {i}\right) = \lim _ {n \rightarrow \infty} C n (\sqrt [ n ]{2} - 1) = C \ln 2 ^ {*}) = 8 0 0 0 \pi \ln 2 \approx 1 7 4 2 0 J.
$$

\*) 利用 541 题的结果.
