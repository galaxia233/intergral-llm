【1840】 $\int \frac{x + 1}{x^2 + x + 1}\mathrm{d}x.$

解 $\int \frac{x + 1}{x^2 + x + 1} \mathrm{d}x = \int \frac{\frac{1}{2}(2x + 1) + \frac{1}{2}}{x^2 + x + 1} \mathrm{d}x = \frac{1}{2} \int \frac{2x + 1}{x^2 + x + 1} \mathrm{d}x + \frac{1}{2} \int \frac{\mathrm{d}\left(x + \frac{1}{2}\right)}{\left(x + \frac{1}{2}\right)^2 + \left(\frac{\sqrt{3}}{2}\right)^2}$

$$
= \frac {1}{2} \ln (x ^ {2} + x + 1) + \frac {1}{\sqrt {3}} \arctan \left(\frac {2 x + 1}{\sqrt {3}}\right) + C.
