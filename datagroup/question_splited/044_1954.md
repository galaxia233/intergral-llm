【1954】 $\int \frac{\sqrt{x^2 + x + 1}}{(x + 1)^2}\mathrm{d}x.$

解 $\int \frac{\sqrt{x^2 + x + 1}}{(x + 1)^2} \mathrm{d}x = \int \frac{x^2 + x + 1}{(x + 1)^2} \cdot \frac{\mathrm{d}x}{\sqrt{x^2 + x + 1}} = \int \frac{(x + 1)^2 - (x + 1) + 1}{(x + 1)^2} \cdot \frac{\mathrm{d}x}{\sqrt{x^2 + x + 1}}$

$$
= \int \frac {\mathrm {d} x}{\sqrt {x ^ {2} + x + 1}} - \int \frac {\mathrm {d} x}{(x + 1) \sqrt {x ^ {2} + x + 1}} + \int \frac {\mathrm {d} x}{(x + 1) ^ {2} \sqrt {x ^ {2} + x + 1}} = I _ {1} - I _ {2} + I _ {3}.
$$

对于 $I_{1}$ ，显然有

对于 $I_{2}$ ，利用1938题的结果，即得

$$
I _ {2} = \int \frac {\mathrm {d} x}{(x + 1) \sqrt {x ^ {2} + x + 1}} = - \ln \left| \frac {1 - x + 2 \sqrt {x ^ {2} + x + 1}}{x + 1} \right| + C _ {2};
$$

对于 $I_{3}$ ，设 $x + 1 = \frac{1}{t}$ 则 $\mathrm{d}x = -\frac{1}{t^2}\mathrm{d}t.$ 不妨设 $t > 0$ ，则有 $\sqrt{x^2 + x + 1} = \frac{\sqrt{t^2 - t + 1}}{t}.$

代入 $I_{3}$ ，得

$$
\begin{array}{l} = - \sqrt {t ^ {2} - t + 1} - \frac {1}{2} \ln \left| t - \frac {1}{2} + \sqrt {t ^ {2} - t + 1} \right| + C _ {3} \\ = - \frac {\sqrt {x ^ {2} + x + 1}}{x + 1} - \frac {1}{2} \ln \left| \frac {1 - x + 2 \sqrt {x ^ {2} + x + 1}}{x + 1} \right| + C _ {4}. \\ \end{array}
$$

于是，最后得到

$$
\int \frac {\sqrt {x ^ {2} + x + 1}}{(x + 1) ^ {2}} d x = \ln \left(x + \frac {1}{2} + \sqrt {x ^ {2} + x + 1}\right) - \frac {\sqrt {x ^ {2} + x + 1}}{x + 1} + \frac {1}{2} \ln \left| \frac {1 - x + 2 \sqrt {x ^ {2} + x + 1}}{x + 1} \right| + C.
$$

如用下述解法更简单些：

$$
\begin{array}{l} \int \frac {\sqrt {x ^ {2} + x + 1}}{(x + 1) ^ {2}} d x = - \int \sqrt {x ^ {2} + x + 1} d \left(\frac {1}{x + 1}\right) = - \frac {\sqrt {x ^ {2} + x + 1}}{x + 1} + \int \frac {\left(x + \frac {1}{2}\right) d x}{(x + 1) \sqrt {x ^ {2} + x + 1}} \\ = - \frac {\sqrt {x ^ {2} + x + 1}}{x + 1} + \int \frac {\mathrm {d} x}{\sqrt {x ^ {2} + x + 1}} - \frac {1}{2} \int \frac {\mathrm {d} x}{(x + 1) \sqrt {x ^ {2} + x + 1}} \\ = - \frac {\sqrt {x ^ {2} + x + 1}}{x + 1} + \ln \left(x + \frac {1}{2} + \sqrt {x ^ {2} + x + 1}\right) + \frac {1}{2} \ln \left| \frac {1 - x + 2 \sqrt {x ^ {2} + x + 1}}{x + 1} \right| ^ {*}) + C. \\ \end{array}
$$

\*）利用1938题的结果.
