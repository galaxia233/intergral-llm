【2665】 $\sum_{n = 1}^{\infty}(-1)^{n}\left(\frac{2n + 100}{3n + 1}\right)^{n}.$

解 $a_{n} = (-1)^{n}\left(\frac{2n + 100}{3n + 1}\right)^{n}$ 由于

$$
\lim  _ {n \rightarrow \infty} \sqrt [ n ]{| a _ {n} |} = \lim  _ {n \rightarrow \infty} \frac {2 n + 1 0 0}{3 n + 1} = \frac {2}{3} <   1,
$$

故原级数绝对收敛，从而也是收敛的
