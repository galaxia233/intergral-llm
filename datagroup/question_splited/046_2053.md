【2053】 证明：若 $(a - c)^2 + b^2 \neq 0$ ，则

$$
\int \frac {a _ {1} \sin x + b _ {1} \cos x}{a \sin^ {2} x + 2 b \sin x \cos x + c \cos^ {2} x} d x = A \int \frac {d u _ {1}}{k _ {1} u _ {1} ^ {2} + \lambda_ {1}} + B \int \frac {d u _ {2}}{k _ {2} u _ {2} ^ {2} + \lambda_ {2}},
$$

式中 $A, B$ 为待定系数， $\lambda_1, \lambda_2$ 为方程 $\left| \begin{array}{cc} a - \lambda & b \\ b & c - \lambda \end{array} \right| = 0 \quad (\lambda_1 \neq \lambda_2)$

的根，而 $u_{i} = (a - \lambda_{i})\sin x + b\cos x, k_{i} = \frac{1}{a - \lambda_{i}} (i = 1,2).$

证记

$$
\begin{array}{l} a ^ {2} \sin^ {2} x + 2 b \sin x \cos x + c \cos^ {2} x = (a - \lambda_ {i}) \sin^ {2} x + 2 b \sin x \cos x + (c - \lambda_ {i}) \cos^ {2} x + \lambda_ {i} \\ = \frac {1}{a - \lambda_ {i}} \left[ (a - \lambda_ {i}) ^ {2} \sin^ {2} x + 2 b (a - \lambda_ {i}) \sin x \cos x + (c - \lambda_ {i}) (a - \lambda_ {i}) \cos^ {2} x \right] + \lambda_ {i}, \\ \end{array}
$$

其中 $\lambda_{i}(i = 1,2)$ 为方程 $\left| \begin{array}{cc}a - \lambda & b\\ b & c - \lambda \end{array} \right| = 0$ 的根.

由假定 $(a - c)^2 + b^2 \neq 0$ ，从而， $(a - c)^2 + 4b^2 \neq 0$ ，因此 $\lambda_1 \neq \lambda_2$ 。再设 $k_i = \frac{1}{a - \lambda_i} (i = 1, 2)$ 及 $u_i = (a - \lambda_i)$ 。 $\cdot \sin x + b\cos x$ 。由于 $(a - \lambda_i)(c - \lambda_i) - b^2 = 0$ ，即 $b^2 = (a - \lambda_i)(c - \lambda_i)$ 。于是，

$$
\begin{array}{l} a ^ {2} \sin^ {2} x + 2 b \sin x \cos x + c \cos^ {2} x = k _ {i} \left[ (a - \lambda_ {i}) ^ {2} \sin^ {2} x + 2 b (a - \lambda_ {i}) \sin x \cos x + b ^ {2} \cos^ {2} x \right] + \lambda_ {i} \\ = k _ {i} \left[ (a - \lambda_ {i}) \sin x + b \cos x \right] ^ {2} + \lambda_ {i} = k _ {i} u _ {i} ^ {2} + \lambda_ {i}. \tag {1} \\ \end{array}
$$

其次，设

$$
a _ {1} \sin x + b _ {1} \cos x = A \left[ (a - \lambda_ {1}) \cos x - b \sin x \right] + B \left[ (a - \lambda_ {2}) \cos x - b \sin x \right], \tag {2}
$$

比较等式两端同类项的系数，则有

$$
- b (A + B) = a _ {1}, \quad A (a - \lambda_ {1}) + B (a - \lambda_ {2}) = b _ {1},
$$

$$
A = - \frac {a _ {1} (\lambda_ {1} - \lambda_ {2}) + b b _ {1} + a _ {1} (a - \lambda_ {1})}{b (\lambda_ {1} - \lambda_ {2})}, \quad B = \frac {b b _ {1} + a _ {1} (a - \lambda_ {1}) ^ {*}}{b (\lambda_ {1} - \lambda_ {2})}.
$$

由（1）式及（2）式即得

$$
\begin{array}{l} \int \frac {a _ {1} \sin x + b _ {1} \cos x}{a ^ {2} \sin^ {2} x + 2 b \sin x \cos x + c \cos^ {2} x} d x \\ = A \int \frac {(a - \lambda_ {1}) \cos x - b \sin x}{k _ {1} [ (a - \lambda_ {1}) \sin x + b \cos x ] ^ {2} + \lambda_ {1}} d x + B \int \frac {(a - \lambda_ {2}) \cos x - b \sin x}{k _ {2} [ (a - \lambda_ {2}) \sin x + b \cos x ] ^ {2} + \lambda_ {2}} d x \\ = A \int \frac {\mathrm {d} u _ {1}}{k _ {1} u _ {1} ^ {2} + \lambda_ {1}} + B \int \frac {\mathrm {d} u _ {2}}{k _ {2} u _ {2} ^ {2} + \lambda_ {2}}. \\ \end{array}
$$

*）按题意， $b \neq 0$ 。因若 $b = 0$ ，则 $\lambda_1 = a, \lambda_2 = c$ ，从而， $k_1$ 无意义。不过，当 $b = 0$ 时，仍能化为所要求的类似形式。事实上，当 $b = 0$ 时， $a \neq c$ ，我们有

$$
\begin{array}{l} \int \frac {a _ {1} \sin x + b _ {1} \cos x}{a \sin^ {2} x + 2 b \sin x \cos x + c \cos^ {2} x} d x = \int \frac {a _ {1} \sin x + b _ {1} \cos x}{a \sin^ {2} x + c \cos^ {2} x} d x \\ = a _ {1} \int \frac {\sin x}{a \sin^ {2} x + c \cos^ {2} x} d x + b _ {1} \int \frac {\cos x}{a \sin^ {2} x + c \cos^ {2} x} d x = - a _ {1} \int \frac {d (\cos x)}{(c - a) \cos^ {2} x + a} + b _ {1} \int \frac {d (\sin x)}{(a - c) \sin^ {2} x + c} \\ = A \int \frac {\mathrm {d} u _ {1}}{k _ {1} u _ {1} ^ {2} + \lambda_ {1}} + B \int \frac {\mathrm {d} u _ {2}}{k _ {2} u _ {2} ^ {2} + \lambda_ {2}}, \\ \end{array}
$$

式中 $A = -a_{1}, B = b_{1}, k_{1} = c - a, k_{2} = a - c, u_{1} = \cos x, u_{2} = \sin x, \lambda_{1} = a, \lambda_{2} = c.$

本题也可用下法另证：命 $u_{i} = (a - \lambda_{i})\sin x + b\cos x$ ， $k_{i} = \frac{1}{a - \lambda_{i}} (i = 1,2)$ ，代入积分等式.然后两边求导，整理并比较系数，便可知 $\lambda_{i}$ 必为方程 $\left| \begin{array}{cc}a - \lambda & b\\ b & c - \lambda \end{array} \right| = 0$ 的根，相应可求出系数 $A,B$
