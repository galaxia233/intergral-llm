【2011】推出下列积分的递推公式

(1) $I_{n} = \int \sin^{n}x\mathrm{d}x;$ (2） $K_{n} = \int \cos^{n}x\mathrm{d}x\quad (n > 2).$

利用这些公式计算 $\int \sin^6 x\mathrm{d}x$ 和 $\int \cos^8 x\mathrm{d}x.$

提示 利用分部积分法，易得：

(1) $I_{n} = -\frac{\cos x \sin^{n-1} x}{n} + \frac{n-1}{n} I_{n-2}$ ;

(2) $K_{n} = \frac{\sin x\cos^{n - 1}x}{n} +\frac{n - 1}{n} K_{n - 2}$

解（1） $I_{n} = \int \sin^{n}x\mathrm{d}x = -\int \sin^{n - 1}x\mathrm{d}(\cos x) = -\cos x\sin^{n - 1}x + (n - 1)\int \cos^{2}x\sin^{n - 2}x\mathrm{d}x$

$$
= - \cos x \sin^ {n - 1} x + (n - 1) \int (1 - \sin^ {2} x) \sin^ {n - 2} x d x = - \cos x \sin^ {n - 1} x + (n - 1) I _ {n - 2} + (1 - n) I _ {n},
$$

于是， $I_{n} = -\frac{\cos x \sin^{n-1} x}{n} + \frac{n-1}{n} I_{n-2}$ ;

利用此公式及 $I_0 = \int \mathrm{d}x = x + C$ ，即得

$$
\begin{array}{l} I _ {6} = \int \sin^ {6} x \mathrm {d} x = - \frac {\cos x \sin^ {5} x}{6} + \frac {5}{6} I _ {4} = - \frac {\cos x \sin^ {5} x}{6} - \frac {5 \cos x \sin^ {3} x}{2 4} + \frac {5}{8} I _ {2} \\ = - \frac {\cos x \sin^ {5} x}{6} - \frac {5 \cos x \sin^ {3} x}{2 4} - \frac {5 \cos x \sin x}{1 6} + \frac {5}{1 6} x + C. \\ \end{array}
$$
