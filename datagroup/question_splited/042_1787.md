【1787】 $\int \frac{x^2}{\sqrt{a^2 + x^2}}\mathrm{d}x.$

提示 设 $x = \operatorname{asht}$ ，并利用 1761 题的结果，

解 与 1786 题相同，设 $x = a\mathrm{sh}t$ ，则 $\frac{x^2}{\sqrt{a^2 + x^2}} = \frac{a\mathrm{sh}^2t}{\mathrm{ch}t},\quad \mathrm{dx} = a\mathrm{ch}t\mathrm{dt}.$

代人得 $\int \frac{x^2}{\sqrt{a^2 + x^2}}\mathrm{d}x = a^2\int \mathrm{sh}^2 t\mathrm{d}t = a^2\left(\frac{1}{4}\mathrm{sh}2t - \frac{t}{2}\right)^{\prime \prime} + C_1 = \frac{x}{2}\sqrt{a^2 + x^2} -\frac{a^2}{2}\ln (x + \sqrt{a^2 + x^2}) + C.$

\*) 利用 1761 题的结果.
