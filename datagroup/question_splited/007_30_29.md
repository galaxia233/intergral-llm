30.29 $\int \frac{dx}{x\sqrt{1 + 3x}}.$   
Let $1 + 3x = z^2, \quad 3dx = 2zdz$ . Then $\int \frac{dx}{x\sqrt{1 + 3x}} = \frac{2}{3}\int \frac{zdz}{[(z^2 - 1) / 3] \cdot z} = 2\int \frac{dz}{(z - 1)(z + 1)} = \int \left(\frac{1}{z - 1} - \frac{1}{z + 1}\right) dz = \ln |z - 1| - \ln |z + 1| + C = \ln \left|\frac{z - 1}{z + 1}\right| + C = \ln \left|\frac{(z - 1)^2}{z^2 - 1}\right| + C = \ln \left|\frac{z^2 - 2z + 1}{3x}\right| + C = \ln \left|\frac{3x + 2 - 2\sqrt{1 + 3x}}{x}\right| + C.$
