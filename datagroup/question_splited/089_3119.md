【3119】若 $n$ 甚大，近似地计算 $C_{2n}^{n}$

解 $C_{2n}^{n} = \frac{2n(2n - 1)\cdots(2n - n + 1)}{n!} = \frac{(2n)!}{(n!)^{2}} = \frac{\sqrt{2\pi\cdot 2n}(2n)^{2n}\mathrm{e}^{-2n}\mathrm{e}^{\frac{\theta_{1}}{24n}}}{2\pi nn^{2n}\mathrm{e}^{-2n}\mathrm{e}^{\frac{\theta_{2}}{6n}}} = \frac{2^{2n}}{\sqrt{n\pi}}\mathrm{e}^{\frac{\theta}{6n}},$

其中 $|\theta| < 1$ $(0 < \theta_{1} < 1, 0 < \theta_{2} < 1)$ .
