【2295】 $\int_0^\pi \sin^{n - 1}x\cos (n + 1)x\mathrm{d}x.$

解 $\int_0^\pi \sin^{n - 1}x\cos (n + 1)x\mathrm{d}x = \int_0^\pi \sin^{n - 1}x(\cos nx\cos x - \sin nx\sin x)\mathrm{d}x$

$$
\begin{array}{l} = \int_ {0} ^ {\pi} \sin^ {n - 1} x \cos n x d (\sin x) - \int_ {0} ^ {\pi} \sin^ {n} x \sin n x d x \\ = \frac {\sin^ {n} x \cos n x}{n} \left| _ {0} ^ {x} - \frac {1}{n} \int_ {0} ^ {x} \sin^ {n} x d (\cos n x) - \int_ {0} ^ {x} \sin^ {n} x \sin n x d x = 0. \right. \\ \end{array}
$$
