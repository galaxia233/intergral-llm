【1878】 $\int \frac{\mathrm{d}x}{(x^2 - 4x + 4)(x^2 - 4x + 5)}.$

提示 本题若用待定系数法，较麻烦一些。可将被积函数变形为 $\frac{1}{(x - 2)^2} - \frac{1}{(x - 2)^2 + 1}$ 后，即易获解。

解 由于 $\frac{1}{(x^2 - 4x + 4)(x^2 - 4x + 5)} = \frac{(x^2 - 4x + 5) - (x^2 - 4x + 4)}{(x^2 - 4x + 4)(x^2 - 4x + 5)} = \frac{1}{(x - 2)^2} - \frac{1}{x^2 - 4x + 5},$

于是， $\int \frac{\mathrm{d}x}{(x^2 - 4x + 4)(x^2 - 4x + 5)} = \int \left[\frac{1}{(x - 2)^2} -\frac{1}{x^2 - 4x + 5}\right]\mathrm{d}x = -\frac{1}{x - 2} -\int \frac{\mathrm{d}(x - 2)}{(x - 2)^2 + 1}$

$$
= - \frac {1}{x - 2} - \arctan (x - 2) + C.
$$
