【2356】数 $M[f] = \lim_{x\to +\infty}\frac{1}{x}\int_{0}^{x}f(\xi)\mathrm{d}\xi$

称为函数 $f(x)$ 在区间 $(0, +\infty)$ 上的平均值。求下列函数在此区间上的平均值：

（3） $f(x) = \sqrt{x}\sin x.$

解（1）由于

$$
\int_ {0} ^ {x} \left[ \sin^ {2} \xi + \cos^ {2} (\xi \sqrt {2}) \right] d \xi = \int_ {0} ^ {x} \left[ \frac {1 - \cos 2 \xi}{2} + \frac {1 + \cos (2 \xi \sqrt {2})}{2} \right] d \xi = x - \frac {1}{4} \sin 2 x + \frac {1}{4 \sqrt {2}} \sin (2 x \sqrt {2}),
$$

所以，

$$
M [ f ] = \lim  _ {x \rightarrow + \infty} \frac {1}{x} \int_ {0} ^ {x} \left[ \sin^ {2} \xi + \cos^ {2} (\xi \sqrt {2}) \right] d \xi = \lim  _ {x \rightarrow + \infty} \left[ 1 + \frac {1}{4 x} \sin 2 x + \frac {1}{4 x \sqrt {2}} \sin (2 x \sqrt {2}) \right] = 1;
$$

（3）利用第二中值定理，得

$$
\int_ {0} ^ {x} \sqrt {\xi} \sin \xi \mathrm {d} \xi = \sqrt {x} \int_ {c} ^ {x} \sin \xi \mathrm {d} \xi = \sqrt {x} (\cos c - \cos x) (0 \leqslant c \leqslant x).
$$

于是， $M[f] = \lim_{x\to +\infty}\frac{1}{x}\int_{0}^{x}\sqrt{\xi}\sin \xi \mathrm{d}\xi = \lim_{x\to +\infty}\frac{\cos c - \cos x}{\sqrt{x}} = 0.$
