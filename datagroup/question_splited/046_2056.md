【2056】 $\int \frac{\sin x - 2\cos x}{1 + 4\sin x\cos x}\mathrm{d}x.$

解 本题也是2053题的特例，且

$$
\int \frac {\sin x - 2 \cos x}{1 + 4 \sin x \cos x} d x = \int \frac {\sin x - 2 \cos x}{\sin^ {2} x + 4 \sin x \cos x + \cos^ {2} x} d x,
$$

这里， $a_1 = 1, b_1 = -2, a = 1, b = 2, c = 1; \lambda_1 = 3, \lambda_2 = -1; k_1 = -\frac{1}{2}, k_2 = \frac{1}{2}; A = \frac{1}{4}, B = -\frac{3}{4}$

$$
u _ {1} = 2 (\cos x - \sin x), \quad u _ {2} = 2 (\cos x + \sin x).
$$

代入得 $\int \frac{\sin x - 2\cos x}{1 + 4\sin x\cos x}\mathrm{d}x = \frac{1}{4}\int \frac{2\mathrm{d}(\cos x - \sin x)}{-2(\cos x - \sin x)^2 + 3} -\frac{3}{4}\int \frac{2\mathrm{d}(\cos x + \sin x)}{2(\cos x + \sin x)^2 - 1}$

$$
= \frac {3}{4 \sqrt {2}} \ln \left| \frac {\sqrt {2} (\sin x + \cos x) + 1}{\sqrt {2} (\sin x + \cos x) - 1} \right| - \frac {1}{4 \sqrt {6}} \ln \left| \frac {\sqrt {3} + \sqrt {2} (\sin x - \cos x)}{\sqrt {3} - \sqrt {2} (\sin x - \cos x)} \right| + C.
$$
