【4291】 $\mathrm{d}u = \left(1 - \frac{1}{y} +\frac{y}{z}\right)\mathrm{d}x + \left(\frac{x}{z} +\frac{x}{y^2}\right)\mathrm{d}y - \frac{xy}{z^2}\mathrm{d}z.$

解 $\mathrm{d}u = \mathrm{d}x + \left(-\frac{1}{y}\mathrm{d}x + \frac{x}{y^2}\mathrm{d}y\right) + \frac{1}{z} (y\mathrm{d}x + x\mathrm{d}y) - \frac{xy}{z^2}\mathrm{d}z$

$$
= \mathrm {d} x + \left(- \frac {1}{y} \mathrm {d} x + x \mathrm {d} \left(- \frac {1}{y}\right)\right) + \frac {1}{z} \mathrm {d} (x y) + x y \mathrm {d} \left(\frac {1}{z}\right) = \mathrm {d} x + \mathrm {d} \left(- \frac {x}{y}\right) + \mathrm {d} \left(\frac {x y}{z}\right)
$$

$$
= \mathrm {d} \left(x - \frac {x}{y} + \frac {x y}{z}\right).
$$

于是， $u = x - \frac{x}{y} +\frac{xy}{z} +C.$
