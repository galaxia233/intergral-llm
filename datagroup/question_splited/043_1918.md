【1918】 $\int \frac{x^2 - 1}{x^4 + x^3 + x^2 + x + 1} dx.$

解 $\int \frac{x^2 - 1}{x^4 + x^3 + x^2 + x + 1} \mathrm{d}x = \int \frac{\left(1 - \frac{1}{x^2}\right) \mathrm{d}x}{\left(x^2 + \frac{1}{x^2}\right) + \left(x + \frac{1}{x}\right) + 1} = \int \frac{\mathrm{d}\left(x + \frac{1}{x}\right)}{\left(x + \frac{1}{x}\right)^2 + \left(x + \frac{1}{x}\right) - 1}$

$$
= \int \frac {\mathrm {d} \left(x + \frac {1}{x} + \frac {1}{2}\right)}{\left[ \left(x + \frac {1}{x}\right) + \frac {1}{2} \right] ^ {2} - \frac {5}{4}} = \frac {1}{\sqrt {5}} \ln \frac {x + \frac {1}{x} + \frac {1}{2} - \frac {\sqrt {5}}{2}}{x + \frac {1}{x} + \frac {1}{2} + \frac {\sqrt {5}}{2}} + C = \frac {1}{\sqrt {5}} \ln \frac {2 x ^ {2} + (1 - \sqrt {5}) x + 2}{2 x ^ {2} + (1 + \sqrt {5}) x + 2} + C.
$$
