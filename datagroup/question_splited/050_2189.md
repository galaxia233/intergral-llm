【2189】 $\int_{a}^{b}\frac{\mathrm{d}x}{x^2}\quad (0 < a < b).$

提示 将 $[a, b]$ 等分，设分点为 $x_0 = a < x_1 < x_2 < \dots < x_n = b$ 。并取 $\xi_i = \sqrt{x_i x_{i+1}} (i = 0, 1, 2, \dots, n-1)$ 。

解 将区间 $[a, b]n$ 等分，设分点为

$$
x _ {0} = a <   x _ {1} <   x _ {2} <   \dots <   x _ {n} = b.
$$

取 $\xi_{i} = \sqrt{x_{i}x_{i + 1}} (i = 1,2,\dots ,n - 1)$ ，显然 $\xi_{i}\in [x_{i},x_{i + 1}]$ .作和

$$
S _ {n} = \sum_ {i = 0} ^ {n - 1} \frac {1}{x _ {i} x _ {i + 1}} \left(x _ {i + 1} - x _ {i}\right) = \sum_ {i = 0} ^ {n - 1} \left(\frac {1}{x _ {i}} - \frac {1}{x _ {i + 1}}\right) = \frac {1}{a} - \frac {1}{b}.
$$

于是， $\lim_{n\to \infty}S_n = \frac{1}{a} -\frac{1}{b}.$

即 $\int_{a}^{b}\frac{\mathrm{d}x}{x^2} = \frac{1}{a} -\frac{1}{b}.$
