【2853】 $\sin^3 x$

解 $\sin^3 x = \frac{3}{4}\sin x - \frac{1}{4}\sin 3x = \frac{3}{4}\sum_{n = 0}^{\infty}(-1)^n\frac{x^{2n + 1}}{(2n + 1)!} -\frac{1}{4}\sum_{n = 0}^{\infty}(-1)^n\frac{3^{2n + 1}x^{2n + 1}}{(2n + 1)!}$

$$
= \frac {3}{4} \sum_ {n = 0} ^ {\infty} (- 1) ^ {n + 1} \frac {3 ^ {2 n} - 1}{(2 n + 1) !} x ^ {2 n + 1} \quad (| x | <   + \infty).
$$
