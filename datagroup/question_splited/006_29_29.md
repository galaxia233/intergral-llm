29.29 $\int \frac{dx}{(x^2 - 6x + 13)^2}.$

By completing the square, $x^{2} - 6x + 13 = (x - 3)^{2} + 4$ . Let $x - 3 = 2\tan \theta$ , $dx = 2\sec^2\theta d\theta$ , $x^{2} - 6x + 13 = 4\sec^2\theta$ (see Fig. 29-6). Then $\int \frac{dx}{(x^2 - 6x + 13)^2} = \int \frac{2\sec^2\theta d\theta}{(4\sec^2\theta)^2} = \frac {1}{8}\int \frac {d \theta}{\sec^2\theta} = \frac {1}{8}\int \cos^2\theta d\theta$ . By Problem 29.1, we have $\frac {1}{8}\cdot \frac {1}{2} (\theta +\sin \theta \cos \theta) + C = \frac {1}{16}\Big(\tan^{-1}\frac {x - 3}{2} +\frac {x - 3}{\sqrt{x^2 - 6x + 13}}\frac {2}{\sqrt{x^2 - 6x + 13}}\Big) + C = \frac {1}{16}\left[\tan^{-1}\frac {x - 3}{2} +\frac {2(x - 3)}{x^2 - 6x + 13}\right] + C.$
