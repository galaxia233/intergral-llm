【1932】 $\int \frac{\mathrm{d}x}{\sqrt[3]{(x + 1)^2(x - 1)^4}}.$

提示 令 $\sqrt[3]{\frac{x + 1}{x - 1}} = t.$

解设 $\sqrt[3]{\frac{x + 1}{x - 1}} = t$ ，则 $x = \frac{t^3 + 1}{t^3 - 1},\mathrm{d}x = -\frac{6t^2}{(t^3 - 1)^2}\mathrm{d}t.$ 代人得

$$
\int \frac {\mathrm {d} x}{\sqrt [ 3 ]{(x + 1) ^ {2} (x - 1) ^ {4}}} = - \frac {3}{2} \int \mathrm {d} t = - \frac {3}{2} t + C = - \frac {3}{2} \sqrt [ 3 ]{\frac {x + 1}{x - 1}} + C.
$$
