30.25 Find $\int \frac{\cos x dx}{\sin x - 1}$ by a suitable substitution.

Let $u = \sin x - 1, \quad du = \cos x dx$ . Then $\int_{\sin x - 1}^{\cos x dx} \, du = \int_{u}^{\frac{du}{u}} \, du = \ln |u| + C = \ln |\sin x - 1| + C = \ln (1 - \sin x) + C$ , since $\sin x \leq 1$ . This is the same result as in Problem 30.24.
