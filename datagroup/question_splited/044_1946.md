【1946】 $\int \frac{x^3 - 6x^2 + 11x - 6}{\sqrt{x^2 + 4x + 3}}\mathrm{d}x.$

解设 $\int \frac{x^3 - 6x^2 + 11x - 6}{\sqrt{x^2 + 4x + 3}}\mathrm{d}x = (ax^2 +bx + c)\sqrt{x^2 + 4x + 3} +\lambda \int \frac{\mathrm{d}x}{\sqrt{x^2 + 4x + 3}},$

从而有 $x^{3} - 6x^{2} + 11x - 6 \equiv (2ax + b)(x^{2} + 4x + 3) + (x + 2)(ax^{2} + bx + c) + \lambda.$

比较等式两端 $x$ 同次幂的系数，求得 $a = \frac{1}{3}, b = -\frac{14}{3}, c = 37, \lambda = -66.$

于是， $\int \frac{x^3 - 6x^2 + 11x - 6}{\sqrt{x^2 + 4x + 3}}\mathrm{d}x = \left(\frac{1}{3} x^2 -\frac{14}{3} x + 37\right)\sqrt{x^2 + 4x + 3} -66\ln \left|x + 2 + \sqrt{x^2 + 4x + 3}\right| + C.$
