【2011】(2) 推出下列积分的递推公式

(1) $I_{n} = \int \sin^{n}x\mathrm{d}x;$ (2） $K_{n} = \int \cos^{n}x\mathrm{d}x\quad (n > 2).$

利用这些公式计算 $\int \sin^6 x\mathrm{d}x$ 和 $\int \cos^8 x\mathrm{d}x.$

提示 利用分部积分法，易得：

(1) $I_{n} = -\frac{\cos x \sin^{n-1} x}{n} + \frac{n-1}{n} I_{n-2}$ ;

(2) $K_{n} = \frac{\sin x\cos^{n - 1}x}{n} +\frac{n - 1}{n} K_{n - 2}$

解 (2) $K_{n} = \int \cos^{n}x\mathrm{d}x = \int \cos^{n - 1}x\mathrm{d}(\sin x)$

$$
\begin{array}{l} = \sin x \cos^ {n - 1} x + (n - 1) \int \sin^ {2} x \cos^ {n - 2} x d x = \sin x \cos^ {n - 1} x + (n - 1) \int (1 - \cos^ {2} x) \cos^ {n - 2} x d x \\ = \sin x \cos^ {n - 1} x + (n - 1) K _ {n - 2} - (n - 1) K _ {n}, \\ \end{array}
$$

于是， $K_{n} = \frac{\sin x\cos^{n - 1}x}{n} +\frac{n - 1}{n} K_{n - 2};$

利用此公式及 $K_{0} = x + C$ ，即得

$$
\begin{array}{l} K _ {8} = \int \cos^ {8} x d x = \frac {1}{8} \sin x \cos^ {7} x + \frac {7}{8} K _ {6} = \dots \\ = \frac {1}{8} \sin x \cos^ {7} x + \frac {7}{4 8} \sin x \cos^ {5} x + \frac {3 5}{1 9 2} \sin x \cos^ {3} x + \frac {3 5}{1 2 8} \sin x \cos x + \frac {3 5}{1 2 8} x + C. \\ \end{array}
$$
