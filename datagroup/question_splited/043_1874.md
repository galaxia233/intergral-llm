【1874】 $\int \frac{\mathrm{d}x}{(x + 1)(x + 2)^2(x + 3)^3}$

提示为了求得待定系数，可同时使用代入法及比较等式两端 $x$ 的同次幂的系数，这样做，可减少确定待定系数的计算工作量.

解设 $\frac{1}{(x + 1)(x + 2)^2(x + 3)^3} = \frac{A}{x + 1} +\frac{B}{x + 2} +\frac{C}{(x + 2)^2} +\frac{D}{x + 3} +\frac{E}{(x + 3)^2} +\frac{F}{(x + 3)^3},$

通分后应有

$$
\begin{array}{l} 1 \equiv A (x + 2) ^ {2} (x + 3) ^ {3} + B (x + 1) (x + 2) (x + 3) ^ {3} + C (x + 1) (x + 3) ^ {3} + D (x + 1) (x + 2) ^ {2} (x + 3) ^ {2} + \\ E (x + 1) (x + 2) ^ {2} (x + 3) + F (x + 1) (x + 2) ^ {2}. \\ \end{array}
$$

在这恒等式中，

令 $x = -1$ ，得 $1 = 8A$ ， $A = \frac{1}{8}$ ；令 $x = -2$ ，得 $1 = -C$ ， $C = -1$ ；令 $x = -3$ ，得 $1 = -2F$ ， $F = -\frac{1}{2}$ 比较 $x^5, x^4$ 及 $x^3$ 的系数，得

$$
\begin{array}{l l} x ^ {5} & A + B + D = 0, \\ x ^ {4} & 1 3 A + 1 2 B + C + 1 1 D + E = 0, \\ x ^ {3} & 6 7 A + 5 6 B + 1 0 C + 4 7 D + 8 E + F = 0. \end{array}
$$

由此， $B = 2$ ， $D = -\frac{17}{8}$ ， $E = -\frac{5}{4}$ .于是，

$$
\begin{array}{l} \int \frac {\mathrm {d} x}{(x + 1) (x + 2) ^ {2} (x + 3) ^ {3}} = \int \left[ \frac {1}{8 (x + 1)} + \frac {2}{x + 2} - \frac {1}{(x + 2) ^ {2}} - \frac {1 7}{8 (x + 3)} - \frac {5}{4 (x + 3) ^ {2}} - \frac {1}{2 (x + 3) ^ {3}} \right] \mathrm {d} x \\ = \frac {1}{8} \ln | x + 1 | + 2 \ln | x + 2 | + \frac {1}{x + 2} - \frac {1 7}{8} \ln | x + 3 | + \frac {5}{4 (x + 3)} + \frac {1}{4 (x + 3) ^ {2}} + C \\ = \frac {1}{8} \ln \left| \frac {(x + 1) (x + 2) ^ {1 6}}{(x + 3) ^ {1 7}} \right| + \frac {9 x ^ {2} + 5 0 x + 6 8}{4 (x + 2) (x + 3) ^ {2}} + C. \\ \end{array}
$$
