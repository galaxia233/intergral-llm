【2105】 $\int x\arctan (x + 1)\mathrm{d}x.$

解 $\int x\arctan (x + 1)\mathrm{d}x = \frac{1}{2}\int \arctan (x + 1)\mathrm{d}(x^2) = \frac{1}{2} x^2\arctan (x + 1) - \frac{1}{2}\int \frac{x^2}{x^2 + 2x + 2}\mathrm{d}x$

$$
= \frac {1}{2} x ^ {2} \arctan (x + 1) - \frac {1}{2} \int \left(1 - \frac {2 x + 2}{x ^ {2} + 2 x + 2}\right) d x = \frac {1}{2} x ^ {2} \arctan (x + 1) - \frac {x}{2} + \frac {1}{2} \ln (x ^ {2} + 2 x + 2) + C.
$$
