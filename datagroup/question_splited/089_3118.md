【3118】推出乘积 $(2n - 1)!! = 1\cdot 3\cdot 5\dots (2n - 1)$ 的渐近公式.

解 $(2n - 1)!! = \frac{2n!}{2^n n!} = \frac{\sqrt{2\pi \cdot 2n}(2n)^{2n}\mathrm{e}^{-2n}\mathrm{e}^{\frac{\theta_1}{24n}}}{2^n\sqrt{2\pi n}n^n\mathrm{e}^{-n}\mathrm{e}^{\frac{\theta_2}{12n}}} = \sqrt{2}(2n)^n\mathrm{e}^{-n + \frac{\theta}{12n}},$

其中 $|\theta| < 1$ $(0 < \theta_{1} < 1, 0 < \theta_{2} < 1)$ .
