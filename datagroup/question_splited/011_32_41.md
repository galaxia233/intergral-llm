32.42 Investigate $\int_0^1\frac{dx}{1 - x^4}.$

$\int_0^1 dx / (1 - x^4) = \lim_{v\to 1^-}\int_0^v dx / (1 - x^4).$ For $0\leq x < 1, 1 - x^4 = (1 - x)(1 + x)(1 + x^2) < 4(1 - x)$ . Hence, $1 / (1 - x^4) > 1 / 4(1 - x).$ But, $\int_0^1 dx / 4(1 - x) = \lim_{v\to 1^-} - \frac{1}{4}\ln |1 - x|]_0^v = \lim_{v\to 1^-} - \frac{1}{4} (\ln |1 - v|) = +\infty.$ Thus, $\int_0^1 dx / (1 - x^4) = +\infty.$
