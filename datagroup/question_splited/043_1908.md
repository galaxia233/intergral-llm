【1908】 $\int \frac{x^4\mathrm{d}x}{(x^{10} - 10)^2}$

解 $\int \frac{x^4\mathrm{d}x}{(x^{10} - 10)^2} = \frac{1}{5}\int \frac{\mathrm{d}(x^5)}{[(x^5 - \sqrt{10})(x^5 + \sqrt{10})]^2} = \frac{1}{200}\int \frac{[(x^5 - \sqrt{10}) - (x^5 + \sqrt{10})]^2}{[(x^5 - \sqrt{10})(x^5 + \sqrt{10})]^2}\mathrm{d}(x^5)$

$$
\begin{array}{l} = \frac {1}{2 0 0} \int \left(\frac {1}{x ^ {5} - \sqrt {1 0}} - \frac {1}{x ^ {5} + \sqrt {1 0}}\right) ^ {2} d (x ^ {5}) = \frac {1}{2 0 0} \int \frac {d (x ^ {5} - \sqrt {1 0})}{(x ^ {5} - \sqrt {1 0}) ^ {2}} - \frac {1}{1 0 0} \int \frac {d (x ^ {5})}{(x ^ {5}) ^ {2} - 1 0} + \frac {1}{2 0 0} \int \frac {d (x ^ {5} + \sqrt {1 0})}{(x ^ {5} + \sqrt {1 0}) ^ {2}} \\ = - \frac {1}{2 0 0 (x ^ {5} - \sqrt {1 0})} - \frac {1}{2 0 0 \sqrt {1 0}} \ln \left| \frac {x ^ {5} - \sqrt {1 0}}{x ^ {5} + \sqrt {1 0}} \right| - \frac {1}{2 0 0 (x ^ {5} + \sqrt {1 0})} + C \\ = - \frac {1}{1 0 0} \left[ \frac {x ^ {5}}{x ^ {1 0} - 1 0} + \frac {1}{2 \sqrt {1 0}} \ln \left| \frac {x ^ {5} - \sqrt {1 0}}{x ^ {5} + \sqrt {1 0}} \right| \right] + C. \\ \end{array}
