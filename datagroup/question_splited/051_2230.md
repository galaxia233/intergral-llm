【2230】 $\lim_{n\to \infty}\left(\frac{2^{\frac{1}{n}}}{n + 1} +\frac{2^{\frac{2}{n}}}{n + \frac{1}{2}} +\dots +\frac{2^{\frac{n}{n}}}{n + \frac{1}{n}}\right).$

解 由于 $0 < \frac{1}{n} - \frac{1}{n + \frac{1}{k}} = \frac{1}{n(nk + 1)} < \frac{1}{n^2}$ ，故

$$
0 <   \frac {1}{n} \sum_ {k = 1} ^ {n} 2 ^ {\frac {k}{n}} - \sum_ {k = 1} ^ {n} \frac {2 ^ {\frac {k}{n}}}{n + \frac {1}{k}} <   \frac {1}{n ^ {2}} \sum_ {k = 1} ^ {n} 2 ^ {\frac {k}{n}} <   \frac {2}{n} \rightarrow 0 \quad (n \rightarrow \infty).
$$

于是， $\lim_{n\to \infty}\sum_{k = 1}^{n}\left(\frac{1}{n + \frac{1}{k}}\cdot 2^{\frac{k}{n}}\right) = \lim_{n\to \infty}\frac{1}{n}\sum_{k = 1}^{n}2^{\frac{k}{n}} = \int_{0}^{1}2^{x}\mathrm{d}x = \frac{1}{\ln 2}.$
