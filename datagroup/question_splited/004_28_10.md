28.10 $\int \sin^2 x dx.$

Let $u = \sin x, \, dv = \sin x \, dx, \, du = \cos x \, dx, \, v = -\cos x.$ Then $\int \sin^2 x \, dx = -\sin x \cos x + \int \cos^2 x \, dx = -\sin x \cos x + \int (1 - \sin^2 x) \, dx = -\sin x \cos x + x - \int \sin^2 x \, dx.$ So $2 \int \sin^2 x \, dx = x - \sin x \cos x + C,$ $\int \sin^2 x \, dx = \frac{1}{2} (x - \sin x \cos x) + C_1.$
