44.59 Use a triple integral to find the volume $V$ inside $x^{2} + y^{2} = 9$ , above $z = 0$ , and below $x + z = 4$ .

\(
V = \int_{-3}^{3}\int_{-\sqrt{9 - x^2}}^{\sqrt{9 - x^2}}\int_{0}^{4 - x}1dzdydx. \text{The bounds on } z \text{ correspond to the requirement that the solid is above } z = 0 \text{ and below } x + z = 4. \text{ The bounds on } y \text{ come from the equation } x^2 + y^2 = 9. \quad V = \int_{-3}^{3}\int_{-\sqrt{9 - x^2}}^{\sqrt{9 - x^2}}z\int_{0}^{4 - x}dydx = \int_{-3}^{3}\int_{-\sqrt{9 - x^2}}^{\sqrt{9 - x^2}}(4 - x)dydx = \int_{-3}^{3}(4 - x)y\int_{-\sqrt{9 - x^2}}^{\sqrt{9 - x^2}}dx = \int_{-3}^{\sqrt{3}}2(4 - x)\sqrt{9 - x^2} dx = 8\int_{-3}^{3}\sqrt{9 - x^2} dx + \int_{-3}^{3}(-2x)\sqrt{9 - x^2} dx. \text{Now, } \int_{-3}^{3}\sqrt{9 - x^2} dx \text{ gives the area of the upper half of the disk } x^2 + y^2 \leq 9 \text{ of radius } 3, \text{ namely, } \frac{1}{2}(9\pi) = \frac{9}{2}\pi. \text{ Also, } \int_{-3}^{3}(-2x)\sqrt{9 - x^2} dx = 0 \text{ (odd function).} \text{ Therefore, } V = 8(\frac{9}{2}\pi) = 36\pi.
\)
