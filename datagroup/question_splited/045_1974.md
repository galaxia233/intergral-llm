【1974】 $\int \frac{x + \sqrt{1 + x + x^2}}{1 + x + \sqrt{1 + x + x^2}}\mathrm{d}x.$

解 $\int \frac{x + \sqrt{1 + x + x^2}}{1 + x + \sqrt{1 + x + x^2}}\mathrm{d}x = \int \frac{(x + \sqrt{1 + x + x^2})(1 + x - \sqrt{1 + x + x^2})}{(1 + x)^2 - (1 + x + x^2)}\mathrm{d}x$

$$
= \int \frac {\sqrt {1 + x + x ^ {2}} - 1}{x} d x = \int \frac {\sqrt {1 + x + x ^ {2}}}{x} d x - \ln | x |.
$$

对于积分 $\int \frac{\sqrt{1 + x + x^2}}{x} \mathrm{d}x$ ，设 $x = \frac{1}{t}$ ，则 $\mathrm{d}x = -\frac{1}{t^2} \mathrm{d}t$ ， $\sqrt{1 + x + x^2} = \frac{\sqrt{t^2 + t + 1}}{|t|}$ 。

不妨设 $t > 0$ ，代入得

$$
\begin{array}{l} \int \frac {\sqrt {1 + x + x ^ {2}}}{x} d x = - \int \frac {\sqrt {t ^ {2} + t + 1}}{t ^ {2}} d t = \int \sqrt {t ^ {2} + t + 1} d \left(\frac {1}{t}\right) \\ = \frac {\sqrt {t ^ {2} + t + 1}}{t} - \frac {1}{2} \int \frac {2 t + 1}{t \sqrt {1 + t + t ^ {2}}} d t = \sqrt {x ^ {2} + x + 1} - \int \frac {d t}{\sqrt {1 + t + t ^ {2}}} - \frac {1}{2} \int \frac {d t}{t \sqrt {1 + t + t ^ {2}}} \\ = \sqrt {x ^ {2} + x + 1} - \ln \left(t + \frac {1}{2} + \sqrt {1 + t + t ^ {2}}\right) + \frac {1}{2} \int \frac {\mathrm {d} \left(\frac {1}{t}\right)}{\sqrt {\left(\frac {1}{t}\right) ^ {2} + \left(\frac {1}{t}\right) + 1}} \\ = \sqrt {x ^ {2} + x + 1} - \ln \frac {2 + x + 2 \sqrt {1 + x + x ^ {2}}}{2 x} + \frac {1}{2} \ln \left(\frac {1}{t} + \frac {1}{2} + \sqrt {\frac {1}{t ^ {2}} + \frac {1}{t} + 1}\right) + C _ {1} \\ = \sqrt {x ^ {2} + x + 1} - \ln \frac {2 + x + 2 \sqrt {1 + x + x ^ {2}}}{2 x} + \frac {1}{2} \ln \frac {2 x + 1 + 2 \sqrt {1 + x + x ^ {2}}}{2} + C _ {1} \\ = \sqrt {x ^ {2} + x + 1} + \frac {1}{2} \ln \frac {2 x + 1 + 2 \sqrt {1 + x + x ^ {2}}}{(2 + x + 2 \sqrt {1 + x + x ^ {2}}) ^ {2}} + \ln x + C. \\ \end{array}
$$

于是，当 $x > 0$ 时，最后得到

$$
\int \frac {x + \sqrt {1 + x + x ^ {2}}}{1 + x + \sqrt {1 + x + x ^ {2}}} d x = \sqrt {x ^ {2} + x + 1} + \frac {1}{2} \ln \frac {2 x + 1 + 2 \sqrt {1 + x + x ^ {2}}}{(2 + x + 2 \sqrt {1 + x + x ^ {2}}) ^ {2}} + C.
$$

当 $x < 0$ 时，可获得同样的结果
