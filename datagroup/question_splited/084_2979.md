【2979】设 $f(x + \pi) = f(x)$ ，则函数 $f(x)$ 在区间 $(-\pi, \pi)$ 内的傅里叶级数具有怎样的特性？

提示 与2978题解法类似， $a_{2n - 1} = b_{2n - 1} = 0 (n = 1,2,\dots)$

解 与2978题解法类似，我们可求得

$$
a _ {n} = \frac {1}{\pi} \int_ {0} ^ {\pi} [ (- 1) ^ {n} + 1 ] f (x) \cos n x d x \quad (n = 0, 1, 2, \dots).
$$

因此，有

$$
a _ {2 n - 1} = 0 \quad (n = 1, 2, 3, \dots).
$$
