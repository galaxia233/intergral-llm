【2077】 $\int x^{2}\mathrm{e}^{x}\cos x\mathrm{d}x.$

解 $\int x^{2} \mathrm {e} ^ {x} \cos x \mathrm {~d} x = \int x^{2} \cos x \mathrm {~d} (\mathrm{e}^{x}) = x^{2} \mathrm{e}^{x} \cos x - \int \mathrm{e}^{x} (2 x \cos x - x^{2} \sin x) \mathrm {~d} x$

$$
\begin{array}{l} = x ^ {2} \mathrm {e} ^ {x} \cos x - \int (2 x \cos x - x ^ {2} \sin x) \mathrm {d} (\mathrm {e} ^ {x}) \\ = x ^ {2} \mathrm {e} ^ {x} \cos x - \mathrm {e} ^ {x} (2 x \cos x - x ^ {2} \sin x) + \int \mathrm {e} ^ {x} (2 \cos x - 4 x \sin x - x ^ {2} \cos x) \mathrm {d} x \\ = \mathrm {e} ^ {x} [ x ^ {2} (\sin x + \cos x) - 2 x \cos x ] + 2 \int \mathrm {e} ^ {x} \cos x \mathrm {d} x - 4 \int x \mathrm {e} ^ {x} \sin x \mathrm {d} x - \int x ^ {2} \mathrm {e} ^ {x} \cos x \mathrm {d} x. \\ \end{array}
$$

于是， $\int x^{2}\mathrm{e}^{x}\cos x\mathrm{d}x = \frac{\mathrm{e}^{x}}{2} [x^{2}(\sin x + \cos x) - 2x\cos x] + \int \mathrm{e}^{x}\cos x\mathrm{d}x - 2\int x\mathrm{e}^{x}\sin x\mathrm{d}x$

$$
\begin{array}{l} = \frac {\mathrm {e} ^ {x}}{2} \left[ x ^ {2} (\sin x + \cos x) - 2 x \cos x \right] + \frac {\mathrm {e} ^ {x}}{2} (\sin x + \cos x) ^ {*}) - 2 \frac {\mathrm {e} ^ {x}}{2} \left[ x (\sin x - \cos x) + \cos x \right] ^ {* *} + C \\ = \frac {e ^ {x}}{2} \left[ x ^ {2} (\sin x + \cos x) - 2 x \sin x + (\sin x - \cos x) \right] + C. \\ \end{array}
$$

\*) 利用 1828 题的结果.

\*\*）利用 2076 题的结果.
