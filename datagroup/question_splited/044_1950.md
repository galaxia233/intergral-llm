【1950】 $\int \frac{\mathrm{d}x}{(x + 1)^3\sqrt{x^2 + 2x}}.$

解设 $x + 1 = \frac{1}{t}$ 则 $\mathrm{d}x = -\frac{1}{t^2}\mathrm{d}t.$ 先设 $t > 0$ ，则有 $\sqrt{x^2 + 2x} = \frac{\sqrt{1 - t^2}}{t}.$

代入得 $\int \frac{\mathrm{d}x}{(x + 1)^5\sqrt{x^2 + 2x}} = -\int \frac{t^4}{\sqrt{1 - t^2}}\mathrm{d}t = (at^3 +bt^2 +ct + e)\sqrt{1 - t^2} +\lambda \int \frac{\mathrm{d}t}{\sqrt{1 - t^2}}.$

从而有 $-t^4 \equiv (3at^2 + 2bt + c)(1 - t^2) - t(at^3 + bt^2 + ct + e) + \lambda.$

比较等式两端 $t$ 的同次幂系数，求得 $a = \frac{1}{4}, b = 0, c = \frac{3}{8}, e = 0, \lambda = -\frac{3}{8}.$

于是， $\int \frac{\mathrm{d}x}{(x + 1)^5\sqrt{x^2 + 2x}} = \left(\frac{1}{4} t^3 +\frac{3}{8} t\right)\sqrt{1 - t^2} -\frac{3}{8}\int \frac{\mathrm{d}t}{\sqrt{1 - t^2}}$

$$
= \frac {3 x ^ {2} + 6 x + 5}{8 (x + 1) ^ {4}} \sqrt {x ^ {2} + 2 x} - \frac {3}{8} \arcsin \frac {1}{x + 1} + C.
$$

再设 $t < 0$ ，则答案前一项不改变符号，但后一项要改变符号，因此，最后得到

$$
\int \frac {\mathrm {d} x}{(x + 1) ^ {5} \sqrt {x ^ {2} + 2 x}} = \frac {3 x ^ {2} + 6 x + 5}{8 (x + 1) ^ {4}} \sqrt {x ^ {2} + 2 x} - \frac {3}{8} \arcsin \frac {1}{| x + 1 |} + C,
$$

其中 $x > 0$ 或 $x < -2$ .
