【1835】 $\int \frac{x\mathrm{e}^x}{(x + 1)^2}\mathrm{d}x.$

解 $\int \frac{x\mathrm{e}^x}{(x + 1)^2}\mathrm{d}x = -\int x\mathrm{e}^x\mathrm{d}\left(\frac{1}{1 + x}\right) = -\frac{x}{1 + x}\mathrm{e}^x +\int \frac{1}{1 + x}\mathrm{e}^x (x + 1)\mathrm{d}x = -\frac{x}{1 + x}\mathrm{e}^x +\mathrm{e}^x +C$

$$
= \frac {e ^ {x}}{1 + x} + C.
$$

下列积分的求法需要把二次三项式化成标准形式，并利用下列公式：

I. $\int \frac{\mathrm{d}x}{a^2 + x^2} = \frac{1}{a}\arctan \frac{x}{a} + C (a \neq 0)$ ;

II. $\int \frac{\mathrm{d}x}{a^2 - x^2} = \frac{1}{2a}\ln \left|\frac{a + x}{a - x}\right| + C (a\neq 0)$

Ⅲ. $\int \frac{x\mathrm{d}x}{a^2\pm x^2} = \pm \frac{1}{2}\ln |a^2\pm x^2 | + C;$

N. $\int \frac{\mathrm{d}x}{\sqrt{a^2 - x^2}} = \arcsin \frac{x}{a} + C (a > 0)$

V. $\int \frac{\mathrm{d}x}{\sqrt{x^2 \pm a^2}} = \ln |x + \sqrt{x^2 \pm a^2}| + C;$

VI. $\int \frac{x\mathrm{d}x}{\sqrt{a^2\pm x^2}} = \pm \sqrt{a^2\pm x^2} +C;$

VII. $\int \sqrt{a^2 - x^2} \, \mathrm{d}x = \frac{x}{2} \sqrt{a^2 - x^2} + \frac{a^2}{2} \arcsin \frac{x}{a} + C \quad (a > 0)$ ;   
VIII. $\int \sqrt{x^2 \pm a^2} \, \mathrm{d}x = \frac{x}{2} \sqrt{x^2 \pm a^2} \pm \frac{a^2}{2} \ln |x + \sqrt{x^2 \pm a^2}| + C.$
